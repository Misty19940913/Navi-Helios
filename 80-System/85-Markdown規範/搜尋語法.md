# Obsidian Graph View 大量寫作查詢語法

## 使用場景

在寫作一篇長文時，一邊開文章視圖，一邊開「名詞筆記」（原子化概念卡）視圖。
透過 Graph View 的 local graph，快速抓出需要的概念段落。

---

## 語法參考

### 按段落查詢（常用）

```dataview
line:("關鍵字")
```

### 按區塊查詢

```dataview
section:("關鍵字")
```

---

## 範例：寫「習慣養成」文章時

在右側名詞筆記的 local graph view 搜尋：

```
line:("習慣")
```

→ 抓出所有提到「習慣」的原子筆記段落，點擊節點即可跳到對應的确切位置。

---

## 對應工具設定

1. **左右分割視窗**：左側文章，右側名詞筆記
2. **開啟 Local Graph View**：右鍵名詞筆記 → Open Local Graph View
3. **在 Graph View 搜尋框**輸入 `line:("你的關鍵字")`
4. 點擊節點即可跳到對應段落的確切位置
