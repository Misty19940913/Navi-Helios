## 檢視維度清單

### 1. 戰略層檢視（000-100）
- [ ] 檢視 000 系統層是否有閒置或過時項目
- [ ] 檢視 100 夢想/策略層筆記的相關性
- [ ] 確認長期目標是否仍然有效
- [ ] 識別需要更新的策略性 MOC

### 2. 戰術層檢視（200）
- [ ] 檢視 200 領域層筆記的完整性
- [ ] 評估領域研究的深度是否足夠
- [ ] 檢查戰術性 MOC 是否需要更新
- [ ] 識別閒置或未完成的領域筆記

### 3. 執行層檢視（300）
- [ ] 檢視 300 執行層筆記的流動率
- [ ] 確認任務筆記是否有明確產出
- [ ] 檢查待處理筆記是否及時清理
- [ ] 評估執行筆記的可用性

### 4. Vault 分層健康度
- [ ] 識別閒置的分層或文件夾
- [ ] 檢查是否有過多「待處理」筆記
- [ ] 評估 tags 使用效率
- [ ] 確認 description 欄位的利用率

### 5. 輸出驅動原則檢視
- [ ] 回顧「輸出驅動輸入」原則是否仍適用
- [ ] 檢視標籤搜尋的成功率
- [ ] 評估三層映射的有效性
