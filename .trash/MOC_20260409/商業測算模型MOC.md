## WACC 計算

### 公式
```
WACC = (E/V) × Re + (D/V) × Rd × (1-Tc)
```

### 參數說明
| 參數 | 意義 |
|------|------|
| E | 股權市場價值 |
| D | 債務市場價值 |
| V | 總資本 (E+D) |
| Re | 股權成本 |
| Rd | 債務成本 |
| Tc | 企業稅率 |

---

## 商業模式要素

| 要素 | 說明 |
|------|------|
| 價值主張 | 產品/服務解決什麼問題 |
| 客戶細分 | 目標客群是誰 |
| 渠道 | 如何觸達客戶 |
| 客戶關係 | 如何維護客戶 |
| 收益流 | 如何變現 |
| 成本結構 | 花費哪些資源 |
