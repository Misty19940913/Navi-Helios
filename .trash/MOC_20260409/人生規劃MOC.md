# 人生規劃 MOC

這是一個關於人生規劃主題的MOC（Map of Content），彙整了多份對話紀錄分析報告，旨在提供一個系統性的檢索途徑，便於快速查閱與個人目標設定、策略制定、執行與調整、以及相關工具應用等知識與建議。

## 涵蓋的人生規劃子主題

*   **人生規劃步驟優化**：目標設定、策略制定、執行與調整、工具應用。
*   **個人成長與發展**：人生規劃與自我成長的緊密連結。
*   **工具應用**：Obsidian 等工具在人生規劃中的輔助作用。

## 人生規劃核心框架（精華摘要）

### 一、原版三步驟 vs 升級版六步驟

| 原版 | 升級版 |
|------|--------|
| 1. 核心原則（三觀、底限、行為準則） | 1. 核心原則（OS） |
| 2. 五年人生戰略 | 2. 深度現實評估 |
| 3. 高效日常 | 3. 五年人生戰略 |
| | 4. 目標分解＋資源調度 |
| | 5. 高效日常 |
| | 6. 反脆弱成長迴路 |

### 二、升級版新增三大模組

#### 模組一：深度現實評估
- 能力盤點（技能矩陣、可變項 vs 不可變項）
- 環境盤點（支持力、阻力、資源流動）
- 行為盤點（說想要 vs 真正有行動的部分）
- 心智盤點（核心信念、恐懼、慣性迴路、逃避模式）

#### 模組二：系統化資源調度
- 時間投資組合（12週計畫、專注區間、棄守區域）
- 能量管理（時段與任務匹配）
- 金錢分配（現金流策略、學習預算）
- 人脈運營（利益交換、價值循環、弱連結啟動）

#### 模組三：反脆弱成長迴路
- 決策複盤（當下 → 心智 → 行為）
- 迭代節奏（每週、每月、每季檢討）
- 停損機制（轉向、簡化、放棄時機）
- 耐久度設計（防倦怠、防分心、防中途放棄）

---

### 三、核心價值觀 OS 完整系統

#### 3.1 理論基礎

**三大成熟模型：**

| 模型 | 作者 | 用途 |
|------|------|------|
| Barrett 七層價值觀 | Richard Barrett | 價值觀層級（生存→貢獻→整合） |
| VIA 24 種性格優勢 | VIA Institute | 天賦傾向與行為模式 |
| Schwartz 十大價值觀 | Shalom Schwartz | 價值動機衝突檢查 |

#### 3.2 完整執行流程（6大模組 × 22步驟）

**模組0：人生操作系統框架**
- 三層架構：三觀（價值觀系統）、底限（尊嚴線）、行為準則（Daily OS）

**模組1：基準校準（Step 1-4）**
- Barrett Values Test（價值層級）
- VIA Character Strengths（天賦傾向）
- Schwartz Basic Values（價值方向）
- 輸出：7-12組關鍵字、天賦模型、層級模型、價值向量圖

**模組2：真實驅動解構（Step 5-11）**
- 四象限驅動源：慾望(Desire)、恐懼(Fear)、痛點(Pain)、逃避(Avoidance)
- 極端壓力測試（底層行為斷言）
- 誘惑情境測試（原則句）
- 尊嚴線測試（3-5條行為底線）

**模組3：理想驅動解構（Step 12-14）**
- 理想自我敘述
- 上升路徑
- 與底線OS比對

**模組4：價值選拔（Step 15-17）**
- 從底線行為抽象化出初步價值（3-5個）
- 蒐集20個候選價值
- 三維評分（真實度×約束力×可行性）
- Pairwise兩兩比較 → 選出前5
- 極端情境二次驗證

**模組5：語義校準 × 心智模型整合（Step 18-20）**
- 價值語義精煉（命名深度化）
- Schwartz衝突檢查
- Barrett層級升級
- VIA對接（天賦→行為準則）

**模組6：行為OS實作（Step 21-22）**
- 每個核心價值產出3條可觀察行為
- 建立「人生OS」總表

### 四、三套價值觀模型比較

| 模型 | 層級/類別 | 特色 | 適用場景 |
|------|-----------|------|----------|
| **Barrett七層** | 生存→關係→自尊→轉化→內在凝聚→貢獻→全人整合 | 層級清晰，最適合人生規劃 | 價值觀對照與衝突分析 |
| **VIA 24種** | 智慧、勇氣、仁愛、公正、節制、超越 | 有研究支持，易於量化 | 天賦識別與行為設計 |
| **Schwartz十大** | 自主、刺激、享樂、成就、尊重、安全、遵從、傳統、善意、普世關懷 | 研究最普及，適合動機分析 | 行為動機分析與衝突檢查 |

---

### 五、價值觀候選清單（20項）

**生存與能量**：健康、穩定、自律、安全感

**關係與影響力**：真誠、信任、愛、貢獻、公平

**成就與突破**：使命感、創造、勇氣、生長、承擔

**內在一致與超越**：自由、智慧、意義、直覺、平靜

---

### 六、核心價值觀篩選流程

1. **準備**：20個候選價值寫成標籤
2. **篩選法1—快速分數法**：三維度評分（真實度/約束力/可行性），保留前10名
3. **篩選法2—兩兩比較法**：Pairwise比較，勝場數排序得前5
4. **尊嚴線檢驗**：套用極端情境，失守者剔除
5. **最終輸出**：3-5個初步核心

---

### 七、價值觀行為化模板

每個核心價值須包含：

| 欄位 | 內容 |
|------|------|
| 名稱 | 你的版本命名 |
| 定義 | 個人語境下的定義 |
| 行為準則 | 3-5條可執行行為 |
| 違反警訊 | 2條可觀察信號 |
| 衡量指標 | KPI |
| Schwartz標籤 | 所屬類別 |
| Barrett層級 | 現狀/目標 |
| VIA支持強項 | 配對的天賦 |
| 升級動作 | 2-4條升級行動 |
| 決策優先規則 | 衝突時的選擇 |
| 最小可行行動 | 每日/每週可執行動作 |

---

### 八、最終檢核標準

- [ ] 產出3-5個價值卡，每卡包含12個欄位
- [ ] 每個價值至少有3條可執行行為，其中1條有量化KPI
- [ ] 衝突矩陣完成，每對高衝突有優先決策規則
- [ ] Barrett層級對照表有每條升級行動（至少2條）
- [ ] VIA配對表能把每個核心連到可執行習慣或制度
