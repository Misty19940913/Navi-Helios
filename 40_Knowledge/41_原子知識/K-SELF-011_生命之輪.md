# K-SELF-011 生命之輪診斷

生命之輪是一套系統性地評估人生八個面向平衡度的工具，幫助個體發現失衡區域並優先投入資源修復。

## 核心問題

人們傾向於過度專注單一領域（通常是工作/金錢），而忽視其他領域，最終導致整體生活品質下降。

## 八大面向

1. 事業（Career）
2. 財務（Financial）
3. 健康（Health）
4. 家庭（Family）
5. 社交（Social）
6. 個人成長（Personal Growth）
7. 休閒（Leisure）
8. 環境（Environment）

## 應用方式

1. 對每個面向以 1-10 評分
2. 視覺化繪製生命之輪
3. 識別明顯落後的面向（低分區域）
4. 從願景出發，設定各面向的理想分數
5. 制定縮小差距的具體行動

## 與五大資產框架的對應

| 生命之輪面向 | 五大資產 |
|-------------|---------|
| 財務 | 金錢財富 |
| 健康 | 身體財富 |
| 社交、家庭 | 社會財富 |
| 個人成長 | 心理財富 |
| 休閒、環境 | 時間財富 |

生命之輪是五大資產框架的**診斷工具**，用於快速定位失衡領域。

## 使用時機

- 年度或季度回顧起點
- 感到生活失衡但無法明確指出問題時
- 作為人生終極劇本九宮格的評估基礎
