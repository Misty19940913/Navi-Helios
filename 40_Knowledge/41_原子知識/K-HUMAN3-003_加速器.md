# 加速器 (Glitches)

## 核心定義

加速器是高風險催化劑，能強制繞過常規路徑突破局限。但若缺乏基礎整合，將導致系統崩潰——1.0級使用加速器如同給幼兒電鋸，2.0級是高風險/高回報，2.5-3.0級才能計算風險。

## 加速器類型

| 加速器 | 作用象限 | 機制 |
|--------|----------|------|
| 迷幻藥 | 精神 | 強制神秘體驗 |
| 興奮劑 (PED) | 身體 | 加速身體發展 |
| 財務壓力 | 職業 | 設定截止日期強制行動 |
| 極端生活變化 | 多象限 | 關係結束、地理遷移、職業轉型 |
| AI | 全部 | 唯一同時跨越所有象限 |

## 風險等級

| 等級 | 加速器風險 |
|------|-----------|
| 1.0 + 加速器 | 死刑（缺乏基礎，無法區分有益和有害） |
| 2.0 + 加速器 | 高風險/高回報（有一定基礎但理解不完整） |
| 2.5-3.0 + 加速器 | 計算後風險（堅實基礎允許有意識的選擇） |

## 使用前提

1. 先最大化「自然」潛能
2. 至少5年以上發展基礎
3. 各象限基本整合
4. 有支持系統

> [!abstract] 核心洞見
> 加速器不是捷徑，是有前提的高風險工具。沒有基礎整合，加速度就是加速度崩潰。
