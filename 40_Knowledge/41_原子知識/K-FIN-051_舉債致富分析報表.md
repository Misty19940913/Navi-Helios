# 舉債致富分析報表

> [!abstract] 核心洞見 (Insight)
> 舉債致富分析報表

### 定義與機制 (What & How)
## 核心功能

- **績效追蹤**：記錄投資表現和收益情況
- **風險監控**：監控關鍵風險指標和預警信號
- **成本分析**：分析借貸成本與投資回報關係
- **決策支持**：提供數據支持投資決策

## 核心數據欄位

### 資金結構
- **投資總金額**：投入特定資產的總金額
- **自有資金**：個人投入的資金
- **借貸金額**：為該項投資借入的資金
- **槓桿比率**：投資總金額 / 自有資金

### 關鍵公式

| 指標 | 公式 |
|------|------|
| 槓桿比率 | =投資總金額/(投資總金額-借貸金額) |
| 淨收益 | =累計收益-累計利息支出 |
| ROI | =淨收益/投資總金額 |
| ROE | =淨收益/(投資總金額-借貸金額) |
| 利息覆蓋率 | =投資收益/利息支出 |
| 年化收益率 | =((期末價值/期初價值)^(12/月數))-1 |

## 績效指標

### 回報率分析
- **投資回報率（ROI）**：淨收益 / 投資總金額
- **股權回報率（ROE）**：淨收益 / 自有資金
- **槓桿效應**：ROE - ROI

### 風險指標
- **最大回撤**：投資期間最大損失幅度
- **波動率**：投資價值的波動程度
- **夏普比率**：風險調整後的超額報酬
- **VaR（風險價值）**：特定置信度下的最大可能損失

## 實務案例

### 房地產投資案例
- 投資標的：台北市住宅
- 投資總金額：2,000萬、自有資金：600萬、借貸：1,400萬
- 利率：2.5%、期限：20年
- ROI：3.25%、ROE：10.83%

### 股票投資案例
- 投資標的：台積電股票
- 投資總金額：100萬、自有資金：50萬、借貸：50萬
- 利率：4%、期限：1年
- ROI：8%、ROE：16%
