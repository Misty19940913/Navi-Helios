# K-SELF-006 個人商業模式圖

將企業「商業模式圖」（Business Model Canvas）應用於個人職涯與生活規劃的九宮格框架，幫助個人像經營企業般系統性地評估、設計並優化自身的人生獲利模式。

## 核心問題

傳統職涯規劃專注於「做了什麼」（關鍵活動），卻忽略了三個更根本的問題：
- **為誰** 提供價值？（目標客層）
- 提供什麼**獨特價值**？（價值主張）
- 如何**持續獲利**？（收益流與成本結構）

多數人對職涯感到迷茫並非能力不足，而是缺乏一套框架來系統性地回答上述問題。

## 九宮格要素

| 要素 | 核心問題 | 個人應用 |
|------|---------|---------|
| **價值主張** | 你為客戶解決什麼問題？ | 從「做了什麼」轉為「創造什麼結果」 |
| **目標客層** | 你要幫助誰？ | 雇主、客戶、同事、家人皆為客層 |
| **通路** | 別人如何知道你？ | 人脈、作品集、口碑、線上 presence |
| **顧客關係** | 你如何與他人互動？ | 協作模式、溝通風格、信任建立方式 |
| **關鍵資源** | 你是誰，擁有什麼？ | 技能、經驗、興趣、人脈、性格特質 |
| **關鍵活動** | 你做了什麼？ | 戰略性任務，區分高價值與低價值活動 |
| **關鍵合作夥伴** | 誰幫助了你？ | 導師、同事、專業網絡、家人 |
| **收益流** | 你得到什麼？ | 薪資、成就感、彈性時間、心理收益 |
| **成本結構** | 你付出什麼？ | 時間、精力、壓力、健康、學習成本 |

## 價值主張的思維轉變

> 翻譯師的價值不在於「翻譯文件」，而在於「幫助律師事務所打贏官司」。

從**活動導向**到**價值導向**的轉變，是提升個人市場價值與職涯滿足感的關鍵。

## 四階段實作方法

### 階段一：繪製現況模式圖
將現有職涯套入九宮格，客觀呈現現狀。

### 階段二：自我探索
- **生命歷程圖**：標注人生高點與低點，發現熱情與天賦所在
- **多重角色練習**：列出十種身份，提取核心價值觀
- **「前十大」與「五最愛」**：從技能列表中找出最擅長與最喜歡的交集

### 階段三：診斷痛點，修正模式
- 圈出造成不滿或壓力的要素
- 針對每要素提出診斷問題
- 小規模實驗替代方案

### 階段四：行動與持續迭代
- 透過兼職、對話或小規模項目驗證新想法
- 定期評估收益與成本，進行新一輪迭代

## 與系統化成長框架的整合

BMY 是「價值創造層」（第六系統）的核心工具。個體從認知重建（信念）到願景聚焦（夢想）到知識積累後，需透過商業模式思維將所學轉化為可持續的價值輸出。

```
認知（信念）→ 願景（夢想）→ 知識 → 流程 → 行動 → 價值創造（BMY）
```
