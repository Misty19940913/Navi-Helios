# 五大元素 (Five Elements)

## 核心定義

五大元素是財富流沙盤模擬人生的核心維度，玩家需要學習同時管理這五種資源。

## 元素定義

| 元素 | 說明 | 人生對應 | 管理重點 |
|------|------|----------|----------|
| 時間 | 有限資源，每5年敲鐘 | 人生階段 | 優先級配置 |
| 精力 | 耗盡導致猝死 | 健康管理 | 能量平衡 |
| 人脈 | 社會互動資源 | 關係網絡 | 質量勝數量 |
| 能力 | 技能與專業知識 | 競爭力 | 持續學習 |
| 金錢 | 現金流與資產 | 財務狀況 | 資產配置 |

## 元素之間的關係

```
時間 → 需要精力來運用
精力 → 來自健康與恢復
人脈 → 擴大能力變現
能力 → 創造金錢回報
金錢 → 購買時間和資源
```

## 在遊戲中的消耗

| 元素 | 減少行為 | 增加行為 |
|------|----------|----------|
| 時間 | 年齡增長 | 無法增加 |
| 精力 | 生孩子、逆流事件 | 休息、補充 |
| 人脈 | 缺乏維護 | 社交互動 |
| 能力 | 不學習 | 持續學習 |
| 金錢 | 支出、投資 | 收入、被動收入 |

## 與現實生活的對應

- 時間管理 → 優先級設置
- 精力管理 → 作息與健康
- 人脈管理 → 關係質量
- 能力管理 → 終身學習
- 財務管理 → 資產配置

> [!abstract] 核心洞見
> 五大元素構成一個封閉系統：任何一個元素的極度匱乏，都會制約其他元素的發揮空間。
