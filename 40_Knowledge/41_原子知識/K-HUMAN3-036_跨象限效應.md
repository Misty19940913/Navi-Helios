# 跨象限效應

## 核心定義

一個領域的發展會影響其他象限，稱為跨象限效應。這種效應有正向（綜效）和負向（阻礙）兩種方向。

## 正向螺旋（綜效模式）

| 模式 | 描述 |
|------|------|
| 身心靈職整合 | 能量提升 → 精神連結加深 → 心智清晰 → 職業資源增加 |
| 卓越迴圈 | 職業成功提供資源 → 身體與精神投資 → 更高認知能力 → 更好職業決策 |
| 整合波 | 將轉化智慧從一領域應用到另一領域 |

## 負向陷阱（阻礙模式）

| 模式 | 描述 |
|------|------|
| 貧窮陷阱 | 財務壓力 → 身心健康惡化 → 人際關係疏離 → 賺錢能力下降 |
| 忙碌至死 | 過度工作犧牲身體 → 耗竭導致效率下降 |
| 靈性逃避 | 空談精神而不實際解決物質問題 |
| 最佳化陷阱 | 過度分析導致癱瘓，無法行動 |

## 實際應用

- 職業困境：先解決身體精力或精神意義問題
- 關係問題：先解決經濟基礎或溝通能力
- 意義危機：先建立身體基礎和經濟保障
- 發展停滯：檢查是否在正確階段

> [!abstract] 核心洞見
> 單一象限的突破或崩潰會引發全系統連鎖反應。善用這個機制，讓正向級聯成為你的加速器。
