# 基質消耗 (Substrate Depletion)

## 核心定義

當複雜系統消耗其賴以生存的基礎（基質）的速度，超過這些基礎的再生速度時，就會產生系統性功能障礙。表現在個人發展中，是犧牲身體健康或精神意義來換取短期財務或職業增長，最終導致整個生命系統的枯竭。

## 運作原理

| 階段 | 機制 |
|------|------|
| 再生失衡 | 系統的提取率 > 再生率 |
| 級聯崩潰 | 基質耗盡後，建立在其上的高層建築將發生不可逆崩潰 |
| 元危機 | 與競爭性動態和指數級技術共同驅動文明風險 |

## 工作演進邏輯

```
工作（生存機制）→ 職業（發展路徑）→ 使命（無法割捨的工作）
     消耗型           積累型            再生型
```

當工作成為使命時，它會產生能量，而不是消耗能量。

## 行動指南

1. **基質審計**：定期檢查四個象限中，哪些活動正在「消耗基質」而無回饋
2. **動機轉化**：將行動驅動力從「避免痛苦（消耗性）」轉向「創造價值（再生性）」
3. **整合生活**：讓 Vocation 的產出能同時滋養其他三個象限

> [!abstract] 核心洞見
> 唯有將「消耗型工作」轉化為「再生型使命」，才能實現超級個體的永續能量循環。
