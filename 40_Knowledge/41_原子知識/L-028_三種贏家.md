# 三種贏家 (Three Winners)

## 核心定義

三種贏家是財富流沙盤順流層的三種勝利條件，代表不同的人生目標與意義實現方式。

## 贏家類型

| 類型 | 說明 | 達成條件 |
|------|------|----------|
| 夢想家 | 實現個人夢想 | 達成夢想目標 |
| 創富家 | 創造豐富財富 | 資產達標 |
| 慈善家 | 回饋社會 | 捐獻達標 |

## 三者關係

```
創富 → 支持 → 夢想實現
夢想 → 激勵 → 創造更多
創造 → 回饋 → 慈善循環
```

## 現實對應

| 類型 | 人生階段 | 核心行動 |
|------|----------|----------|
| 夢想家 | 自我實現 | 設定並達成夢想 |
| 創富家 | 財務自由 | 累積並管理財富 |
| 慈善家 | 回饋階段 | 貢獻並分享 |

## 反思問題

- 你想成為哪種贏家？
- 三種對你來說意味著什麼？
- 如何平衡三者？

> [!abstract] 核心洞見
> 三種贏家不是互斥的。真正的順流層贏家，是能同時平衡夢想、財富與回饋的人。
