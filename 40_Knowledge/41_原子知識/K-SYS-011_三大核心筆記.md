# 三大核心筆記與轉化邏輯

## 核心定義

卡片盒不是「資料夾」，而是一個有生命週期的流轉系統。理解三種核心筆記的角色與轉化邏輯，是建立「會思考」系統的關鍵。

## 三種筆記

| 筆記類型 | 使命 | 存放位置 | 壽命 |
|----------|------|----------|------|
| 閃念筆記（Fleeting） | 無壓力捕捉瞬間靈光 | 隨身筆記本/數位收件箱 | 24-48 小時 |
| 文獻筆記（Literature） | 記錄來源說了什麼 | 參考文獻管理系統（如 Zotero） | 長期 |
| 永久筆記（Permanent/Zettel） | 成為系統中長期的知識基石 | 卡片盒 | 永久 |

## 關鍵區分

**文獻筆記記錄的是「他人的想法」，永久筆記記錄的是「我的洞察」。**

## 轉化邏輯

```
閃念筆記 + 文獻筆記 → 永久筆記
流程：隨時捕捉 → 文獻提煉（24-48 小時內處理）→ 精煉入庫 → 斷捨離
```

## 行動指南

1. 靈感筆記：隨時捕捉，不惜一切代價保持「無摩擦」
2. 文獻筆記：使用自己的話重述，附上來源 URL/DOI
3. 永久筆記：強制要求自己尋找至少一個與現有筆記的連結
4. 每次轉化完成後果斷「丟棄」原始的閃念筆記，保持系統清潔

> [!abstract] 核心洞見
> 閃念筆記是燃料，文獻筆記是原料，永久筆記是引擎——沒有轉化過程，就只有庫存而沒有系統。
