# 沙盤推演流程 (Sandtable Process)

## 核心定義

沙盤推演是財富流沙盤的完整流程，模擬人生20-60歲的財務歷程，3-4小時涵蓋40年。

## 準備階段

| 步驟 | 內容 |
|------|------|
| 1 | 角色設定與財務初始狀態 |
| 2 | 選擇九仔與夢想 |
| 3 | 玩家自我介紹 |
| 4 | 擲骰決定起始玩家 |

## 遊戲階段

| 階段 | 內容 |
|------|------|
| 平流層 | 1顆骰子移動 |
| 逆流層 | 覺察卡+恩典 |
| 順流層 | 2顆骰子 |

## 特殊規則

1. 每5年敲鐘提醒
2. 結算日記錄財務
3. 精力低於0猝死
4. 報表錯誤罰款100萬

## 覆盤階段

| 步驟 | 內容 |
|------|------|
| 1 | 遊戲回顧 |
| 2 | 行為覺察 |
| 3 | 信念顯現 |
| 4 | 行動計劃 |

> 「推演越多越富有，覺察越多越自由」

> [!abstract] 核心洞見
> 沙盤的價值不在於遊戲本身，而在於覆盤。遊戲中每一個「為什麼我會這樣選擇」，都是改變現實模式的線索。
