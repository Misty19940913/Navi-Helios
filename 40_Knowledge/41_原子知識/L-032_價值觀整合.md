# 價值觀×財富流 (Values×WealthFlow)

## 核心定義

將價值觀探索與金錢信念覺察整合的系統。價值觀決定選擇，選擇決定財富。

## 對應關係

| 價值觀 | 金錢信念 | 財富流角色 |
|----------|------------|------------|
| 優先順序 | 花錢態度 | 支出管理 |
| 人生使命 | 賺錢目的 | 目標設定 |
| 核心價值 | 信念根源 | 覺察卡 |

## 探索維度

### 1. 金錢價值觀
- 問題：你為什麼想賺錢？
- 對應：金錢信念
- 遊戲：覺察卡問題

### 2. 時間價值觀
- 問題：你願意用什麼換取金錢？
- 對應：五大元素
- 遊戲：精力消耗

### 3. 關係價值觀
- 問題：金錢vs關係如何取捨？
- 對應：消費決策
- 遊戲：結婚生子格

### 4. 成就價值觀
- 問題：多少錢才夠？
- 對應：目標設定
- 遊戲：三種贏家

## 整合操作

1. 價值觀探索 → 金錢信念覺察
2. 信念顯現 → 行為模式
3. 遊戲體驗 → 低成本試錯
4. 現實應用 → 財務改善

> 「價值觀決定選擇，選擇決定財富」

> [!abstract] 核心洞見
> 多數人的金錢問題，表面是財務問題，深層是價值觀問題。低自信商數遊戲能讓你在玩中看見自己的金錢信念。
