# 覺察卡 (Awareness Card)

## 核心定義

覺察卡是財富流沙盤中的心理反思工具，幫助玩家覺察潛意識中的金錢信念與行為模式。

## 使用時機

1. 進入逆流層時
2. 停留在覺察格
3. 教練引導時
4. 盤後覆盤

## 卡片類型

| 類型 | 功能 | 問題範例 |
|------|------|----------|
| 金錢信念 | 顯現潛意識 | 你對錢的感覺是？ |
| 關係互動 | 社會模式 | 你如何談錢？ |
| 目標設定 | 動機根源 | 你真正想要的是什麼？ |
| 恐懼害怕 | 阻礙覺察 | 你害怕什麼發生？ |
| 價值觀 | 優先順序 | 對你來說最重要的是？ |

## 覺察問題範例

- 「小時候父母如何談論錢？」
- 「你有錢時的感受是什麼？」
- 「你最害怕失去什麼？」
- 「別人成功時你的感受？」
- 「你為什麼想賺錢？」

## 價值

1. 顯現潛意識
2. 發現行為模式
3. 覺察情緒觸發
4. 重新設定信念

> 「覺察是改變的開始」

> [!abstract] 核心洞見
> 覺察卡問的不是「你現在怎麼做」，而是「你為什麼會這樣做」。答案往往藏在童年或過去的某個記憶裡。
