# 財務自由實現路徑

> [!abstract] 核心洞見 (Insight)
> 財務自由實現路徑

### 定義與機制 (What & How)
## 核心公式

```
財務自由 = 被動收入 > 總支出
         = 資金 × 投資報酬率 > 生活支出 × (1 + 通貨膨脹率)
         = (收入 × 儲蓄率) × 投資報酬率 > 生活支出 × (1 + 通貨膨脹率)
```

## 三大操控面向

| 面向 | 說明 |
|------|------|
| 資金 | 增加收入來源（主業+副業），提高儲蓄率 |
| 投資報酬率 | 培養機會識別能力、資產配置，覆蓋通膨 |
| 生活支出 | 控制支出水平，保持低物慾 |

## 時間跨度

多數人實現財務自由用**10年**即可達成。

10年跨度會面臨2個人生階段：
- 20-30歲：單身至結婚
- 30-40歲：購屋、養育子女
- 40-50歲：長輩照護

支出變化會影響財務自由進度。

## 關鍵要素

1. **資金**：增加收入管道，無論主動或被動
2. **儲蓄**：儲蓄率高低決定財務自由進度
3. **投資**：投資報酬率需覆蓋通貨膨脹率

## 滾動調整原則

財務自由是需要不斷滾動調整的動態過程：
- 經濟狀況上升，物慾可能上升
- 物價持續上漲
- 人生階段變化
