# K-SELF-010 高效率人生管理系統框架

將個人成長視為一項系統工程，透過六階段筆記系統將內在修復轉化為外在價值實現的完整成長飛輪。

## 核心問題

多數人努力無效的原因不是不夠勤奮，而是系統失衡：沒有穩固的認知基礎，所有目標設定、知識學習和行動執行都建築在流沙之上。

## 六階段成長飛輪

```
認知重塑（信念）→ 願景設定（夢想）→ 知識積累 → 流程優化 → 行動轉化 → 價值實現
     A              B              C           D           E           F
```

### 系統A：認知自我（Beliefs）
清除限制性信念，建立底層價值觀。是所有後續系統的前提。

### 系統B：聚焦夢想（Dreams）
將抽象願景轉化為可執行的工程藍圖，連結長期使命與短期行動。

### 系統C：知識升級（Knowledge Capitalization）
將資訊轉化為可產出的個人資產，實現「知識資本化」而非單純記憶儲存。

**核心循環**：捕捉（Capture）→連接（Connect）→創造（Create）

### 系統D：高效工作（Work）
將知識與目標應用於日常任務，提升單次工作流程的效率與產出品質。

### 系統E：行動突破（Action）
將決策與執行結果反饋回系統上游，透過「決策日誌」與「瓶頸分析」持續優化。

### 系統F：價值創造（Value）
最終驗收層：若前五階段無法變現為金錢無虞與幸福平衡，意味系統需重大調整。

## 各系統的筆記工具

| 系統 | 筆記工具 |
|------|---------|
| 認知 | 認知筆記、信念筆記 |
| 夢想 | 人生終極劇本、夢想九宮格 |
| 知識 | 四色筆記法、思維導圖、費曼技巧 |
| 工作 | 艾森豪矩陣、SOP 模板、會議記錄 |
| 行動 | 決策日誌、習慣追蹤 |
| 價值 | 富裕筆記、BMY 商業模式圖 |

## 順序的重要性

若跳過認知重建（A）直接設定夢想（B），限制性信念會讓目標變得保守或無法執行。若跳過夢想設定（B）直接追求效率（D），個體會變得高效但失去方向。
