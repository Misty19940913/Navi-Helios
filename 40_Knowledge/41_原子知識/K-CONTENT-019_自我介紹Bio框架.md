# 自我介紹文案框架：吸引讀者的Bio公式

## 核心主張

**壞的自我介紹只談自己，好的自我介紹談讀者能得到什麼。**

多數人的Bio與目標讀者無關，讀者看完不知道「這個人能幫我什麼」。

## 無效的Bio做法（常見錯誤）

| 錯誤 | 為何無效 |
|------|----------|
| 只描述日常工作 | 讀者不 care 你上班做什麼 |
| 使用凡人無法理解的術語 | 沒有降低認知負擔 |
| 說「我正在研究某課題」| 讀者要的是解答，不是你的過程 |
| 沒有主題方向 | 無法吸引特定讀者群 |

## 有效Bio的三個必要元素

```
1. 你寫什麼主題
2. 你幫助誰（目標讀者）
3. 你有什麼相關經驗/資歷
```

## 框架

```
我幫助 [目標讀者] 解決 [他們的問題]，
通過/利用 [你的方法/經驗]。

[1-2句相關經歷/成果，建立信任]
```

## 實例

**無效版**：
> 「我是某某，在科技公司上班，喜歡看書和跑步。」

**有效版**：
> 「幫助上班族用下班後的2小時建立第二收入。
> 前工程師，3年內從月薪3萬到被動收入10萬。
> 分享內容：個人理財 × 職涯發展 × 投資心法。」

## 冒牌者情結（Impostor Syndrome）的處理

> 「就算你認為自己不是專家，只要曾解決過某個問題，就具備該領域的專業知識。」

**你不是假裝成專家，而是記錄自己真正走過的路。**

## 為何Bio如此重要

1. **新讀者第一接觸點**：決定3秒內是否追蹤
2. **演算法參考**：平台根據Bio主題推廣給相應讀者
3. **變現前提**：讀者要先相信你能幫他們，才會購買產品/服務
