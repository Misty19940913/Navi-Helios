# ICT 交易概念

## 定義
Inner Circle Trader (ICT) 是一套機構級別的價格行為交易方法，強調理解機構資金的運作方式。

## 核心概念

### 市場結構
- **高點 (High)**：價格波動的明顯轉折高點
- **低點 (Low)**：價格波動的明顯轉折低點
- **結構轉變 (Structure Break)**：價格突破關鍵支撐/阻力後的趋势改变

### 公平價格差距 (FVG / Fair Value Gap)
價格快速波動時產生的價格「真空區」：
- 定義：三根K線中，中間K線的高點低於前後K線的低點（多頭）或低點高於前後K線的高點（空頭）
- 用途：價格回歸FVG時常出現交易機會

### 訂單區 (Order Block)
- 大型機構建倉的區域
- 通常出現在明顯的高低點附近
- 特徵：大型陽燭/陰燭後的盤整區

### 流動性獵取 (Liquidity Sweep)
- 價格尋找流動性池（Stop Loss 聚集區）
- 機構在出貨前的「價格清洗」
- 識別：價格來到之前的高點/低點後迅速反轉

## 交易原則

### 進場時機
1. 識別 FVG 或 Order Block
2. 等待價格回歸到關鍵區域
3. 確認流動性獵取完成
4. 進場設定止損

### 止損設置
- 放在 Order Block 外部
- 不超過前一個結構轉變點

## 與 SMC 整合

ICT 與 Smart Money Concepts (SMC) 高度相關：
- FVG = Order Block 的識別方式之一
- 流動性獵取 = 機構資金流向指標
