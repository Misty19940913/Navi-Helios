# 生命之輪×財富流 (Wheel×WealthFlow)

## 核心定義

生命之輪的四象限與財富流三層世界的對應整合系統。

## 對應關係

| 生命之輪 | 財富流層級 | 財務狀態 |
|------------|------------|----------|
| 身體 | 平流層 | 基礎收入 |
| 關係 | 平流層 | 家庭支出 |
| 心靈 | 逆流層 | 覺察成長 |
| 事業 | 順流層 | 財務自由 |

## 詳細映射

### 身體象限 ↔ 平流層
- 健康管理 → 精力系統
- 運動養生 → 收入基礎
- 飲食控制 → 支出管理

### 關係象限 ↔ 平流層
- 家庭責任 → 結婚生子
- 社交圈 → 人脈投資
- 親密關係 → 財務共同

### 心靈象限 ↔ 逆流層
- 內在探索 → 覺察卡
- 精神成長 → 挑戰經歷
- 意義追尋 → 低谷反思

### 事業象限 ↔ 順流層
- 職業發展 → 財務自由
- 目標實現 → 三種贏家
- 價值創造 → 資產信托

## 整合應用

```
生命之輪評估 → 財務層級診斷
↓                         ↓
象限平衡 → 層級晉升
↓                         ↓
行動計劃 → 財務目標
```

## 均衡目標

- 四象限平衡 → 三層皆通
- 身心健康 → 精力充足
- 關係和諧 → 財務穩定
- 事業順遂 → 進入順流

> [!abstract] 核心洞見
> 生命之輪不平衡，財富流層級也上不去——每個象限的落後都在拖你晉升的後腿。
