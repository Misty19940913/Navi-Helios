# 低成本試錯 (Low-Cost Trial)

## 核心定義

低成本試錯讓玩家在沙盤中練習財務決策，將錯誤留在遊戲中，將經驗帶入現實。

## 價值

| 價值 | 說明 |
|------|------|
| 零風險 | 不損失真實金錢 |
| 可重來 | 多次嘗試 |
| 快回饋 | 即時看到結果 |
| 提取經驗 | 帶入現實 |

## 試錯類型

| 類型 | 遊戲行為 | 現實對應 |
|------|----------|----------|
| 投資嘗試 | 各種投資選項 | 資產配置 |
| 創業體驗 | 企業投資 | 事業發展 |
| 風險測試 | 極限操作 | 風險管理 |
| 抉擇練習 | 不同選擇 | 人生決策 |

## 學習循環

```
1. 選擇 → 做決定
2. 執行 → 行動
3. 結果 → 看 outcome
4. 反思 → 覺察
5. 調整 → 改變
6. repeat → 繼續練習
```

## 遊戲vs現實

| 比較 | 遊戲 | 現實 |
|------|------|------|
| 成本 | 低 | 高 |
| 回饋 | 快 | 慢 |
| 可重來 | 是 | 有限 |
| 教訓 | 免費 | 昂貴 |

## 核心心態

- 開放接受錯誤
- 從錯誤中學習
- 不要怕失敗
- 記住經驗

> 「把錯誤留在沙盤，把智慧帶入生活」

> [!abstract] 核心洞見
> 低成本試錯的核心價值不是「避免犯錯」，而是「用最低代價完成最多次的假設驗證」。
