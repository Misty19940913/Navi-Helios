# 精力系統 (Energy System)

## 核心定義

精力是財富流沙盤中與金錢同等重要的資源，耗盡會導致「猝死」。模擬人生的健康與能量管理。

## 數值範圍

- 初始：5點
- 最大：10點
- 最小：-3點
- ⚠️ 低於0即猝死

## 消耗行為

| 行為 | 精力消耗 |
|------|----------|
| 生孩子 | -2點 |
| 經營副業 | -1點 |
| 逆流事件 | -1~-3點 |
| 失業 | -1點 |

## 恢復行為

| 行為 | 精力恢復 |
|------|----------|
| 結算日補充 | +1~2點 |
| 休息 | +1點 |
| 銀翅膀 | +1點 |

## 遊戲機制

- 每5年敲鐘提醒
- 精力低於0猝死
- 反映健康風險管理

## 與現實對應

| 遊戲 | 現實 |
|------|------|
| 精力5點 | 健康平均值 |
| 精力-3 | 過勞臨界點 |
| 猝死 | 健康崩潰 |

> 「健康是1，其他是0」

> [!abstract] 核心洞見
> 精力系統提醒我們：財富不能建立在健康透支上。現實中沒有「結算日」會自動幫你恢復精力——你需要主動管理。
