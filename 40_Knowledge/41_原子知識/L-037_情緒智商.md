# 情緒智商 EQ (Emotional Intelligence Quotient)

## 核心定義

情緒智商是感知自我與他人心態、建立連結的能力，結合 PQ 形成吸引外部資源的磁場。

## Daniel Goleman 模型

| 領域 | 應用 |
|------|------|
| 自我覺察 | 認識情緒觸發點 |
| 自我管理 | 壓力下情緒控制 |
| 社會覺察 | 理解他人需求 |
| 關係管理 | 建立深度連結 |

## 執行價值

- 防止個體陷入「資源孤立」
- 在面對不利的「行情卡」時，避免非理性拋售
- 維持情緒穩定性

## 與財富流的關係

| 場景 | EQ 作用 |
|----------|---------|
| 企業談判 | 共贏架構 |
| 資源整合 | 群眾募資 |
| 市場波動 | 避免恐慌拋售 |
| 人脈經營 | 信任建立 |

## 提升方法

1. 情緒日誌
2. 暫停練習
3. 同理心練習
4. 教練指導

> 「情商結合玩商形成吸引外部資源的磁場」

> [!abstract] 核心洞見
> EQ 不是「讓你變得更會社交」這麼簡單。在財富遊戲中，高 EQ 意味著：在別人恐慌時你能保持理性，在需要資源時你能召募到人。
