# 人生設計十框架

> [!abstract] 核心洞見 (Insight)
> 人生設計不是具有一系列步驟的線性過程，而是一系列幫助你像設計師一樣思考和行動的心態和思維框架。

### 定義與機制 (What & How)
- **核心定義**：設計師思維框架
- **十個核心框架**：

| 框架 | 核心概念 |
|------|----------|
| 1. 接受旅程 | 從根本上接受你在旅程中的位置 |
| 2. 好奇心 | 忘記尋找熱情；努力去了解自己 |
| 3. 明確重要 | 明確什麼對你來說是重要的 |
| 4. 多元路徑 | 集思廣益，想出多種可能的途徑 |
| 5. 建立網絡 | 透過同情和向他人學習來建立網絡 |
| 6. 設計故事 | 設計你的故事 + 你的個人品牌 |
| 7. 行動驗證 | 在現實世界中測試你的想法和假設 |
| 8. 靈活調整 | 根據所學到的知識調整你的計劃 |
| 9. 未來身份 | 專注於你未來會成為怎樣的人 |
| 10. 相信值得 | 相信你的生活值得設計 |

### 實踐與案例 (Action & Proof)
- **應用**：這些框架不是線性步驟，而是可以根據需要靈活運用的思維工具

### 知識網絡 (Connection)
- **關聯脈絡**：設計思考與設計師心態的理論總結
