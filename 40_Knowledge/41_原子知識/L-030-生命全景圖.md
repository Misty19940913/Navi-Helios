# 生命全景圖

> [!abstract] 核心洞見 (Insight)
> 生命全景圖是一種整合性決策系統，透過視覺化框架將抽象的人生願景與核心價值轉化為可執行的行動策略。如同人生的「Google地圖」——先設定宏大的最終目的地（核心價值與長期願景），自動規劃主要城鎮（生活領域目標）、具體道路（行動計劃），並隨時根據路況（環境變化）調整最佳路線。

### 定義與機制 (What & How)

- **核心定義**：生命全景圖是一種整合性決策系統，將多個生活領域（職業、財務、健康、關係等）的願景與行動，透過視覺化框架統一呈現與管理。
- **運作機制**：以 3-5 項核心價值觀為中心錨點，進行階層化分解——長期願景 → 中期目標 → 短期行動，並透過定期回顧與調整機制保持動態適應。

### 五個關鍵變數

| 變數 | 說明 |
|------|------|
| 核心價值驅動 | 以 3-5 項核心價值觀為中心點，作為決策錨點 |
| 多維度整合 | 整合職業、財務、健康、關係等多個生活領域 |
| 階層化分解 | 將長期願景分解為中期目標和短期行動 |
| 視覺表徵優勢 | 利用圓形圖、樹狀圖、時間軸等視覺化形式 |
| 動態調節機制 | 內建定期回顧與調整流程 |

### 實踐與案例 (Action & Proof)

**行動六步驟**：

1. **確立核心價值**：明確 3-5 項最根本的核心價值觀
2. **識別生活領域**：使用生命之輪等工具評估各領域滿意度
3. **設定長期目標**：進行願景冥想、多路徑設計（如奧德賽計劃）
4. **分解行動計劃**：將宏大目標轉化為符合 SMART 原則的短中期行動
5. **視覺化呈現**：選擇適合的圖表形式整合所有要素
6. **動態調節**：建立定期回顧與調整機制

### 常見問題處理

- **價值觀衝突**：以核心價值為優先，次要價值做權衡取捨
- **目標過於抽象**：透過階層化分解轉化為具體可衡量的行動
- **執行脫節**：建立定期回顧機制，及時調整偏離的行動
- **多領域失衡**：使用生命之輪進行全面評估，識別弱勢領域
