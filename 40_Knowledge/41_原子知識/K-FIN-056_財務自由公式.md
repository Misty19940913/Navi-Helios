# 財務自由公式

> [!abstract] 核心洞見 (Insight)
> 財務自由公式

### 定義與機制 (What & How)
## 核心公式

```
財務自由 = 被動收入 > 總支出
         = 資金 × 投資報酬率 > 生活支出 × (1 + 通貨膨脹率)
         = (收入 × 儲蓄率) × 投資報酬率 > 生活支出 × (1 + 通貨膨脹率)
```

## 四大變數

| 變數 | 說明 |
|------|------|
| 收入 | 主動收入（薪資）和被動收入（投資收益） |
| 儲蓄率 | (收入 - 支出) / 收入，目標至少20% |
| 投資報酬率 | 資金的增值速度，需覆蓋通貨膨脹 |
| 生活支出 | 維持生活所需的成本 |

## 三個可控面向

1. **資金**：增加收入來源、提高儲蓄率
2. **投資報酬率**：培養投資能力、做好資產配置
3. **生活支出**：控制支出水平

## 時間因素

實現財務自由多數人需要**10年**，因為：
- 人生階段變化（結婚、購屋、育兒、照護）
- 經濟起伏影響支出
- 需要滾動調整

## 通貨膨脹考量

投資報酬率需要：
- 覆蓋通貨膨脹率
- 產生足夠的支出費用
- 部分利息回流至本金
