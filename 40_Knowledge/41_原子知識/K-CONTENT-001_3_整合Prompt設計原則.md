# 整合 Prompt 設計原則

> [!abstract] 核心洞見 (Insight)
> 整合 Prompt 透過「角色合一、结构清晰、風格一致」三大原則，將部落格文章與學習指南合併為一體化輸出，實現一次輸入同時產出兩種內容，大幅提升內容產出效率與風格一致性。

### 定義與機制 (What & How)
- **核心定義**：整合 Prompt 是將兩種或多種內容類型（部落格文章 + 學習指南）合併為單一 Prompt 的設計方式，讓 AI 在一次回應中輸出所有需要的內容模組。
- **運作機制**：
  1. **角色合一**：避免重複設定角色、任務流程，一次說明完整需求
  2. **結構清晰**：先文章後測驗，分段輸出，明確標記各部分區隔
  3. **風格一致**：簡潔、易讀、有深度，三種內容類型共享同一語氣與用詞習慣

### 實踐與案例 (Action & Proof)
- **真實案例**：為線上課程平台設計「SEO 完全指南」整合 Prompt，單次輸入產出 3000 字部落格文章 + 10 題簡答題 + 5 題論述題 + 詞彙表，整體產出時間從 4 小時縮短至 45 分鐘。
- **行動指南**：
  - 在 Prompt 開頭明確定義「產出內容類型與順序」
  - 使用分隔標記（如「---」或「###」）區分不同內容區塊
  - 分別指定每種內容的格式要求，避免混合混亂
  - 統一指定「風格要求」，如「對話式但具洞見、段落簡短」
