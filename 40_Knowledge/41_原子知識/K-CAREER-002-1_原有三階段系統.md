# 原有三階段系統

> [!abstract] 核心洞見 (Insight)
> 人生規劃的原有三階段系統：原則確立、五年戰略、高效日常

### 定義與機制 (What & How)
## 一、第一階段：確立核心原則

### 內容
| 項目 | 說明 |
|------|------|
| 世界觀 | 對世界的理解與信念 |
| 價值觀 | 什麼是真正重要的 |
| 人生觀 | 想要成為什麼樣的人 |
| 底限 | 絕不妥協的原則（如：真誠、健康第一）|

### 目的
建立不會動搖的北極星

---

## 二、第二階段：定五年人生戰略

### 內容
| 項目 | 說明 |
|------|------|
| 人生藍圖 | 5 年後想要的生活 |
| 長期目標 | 具體的階段性目標 |
| 里程碑 | 關鍵檢查點 |

### 目的
將願景轉化為可執行的路徑

---

## 三、第三階段：建立高效日常

### 內容
| 項目 | 說明 |
|------|------|
| 良好習慣 | 每日/每週例行 |
| 任務規劃 | Todo、清單管理 |
| 復盤 | 定期檢視與調整 |

### 目的
將戰略落實到日常行動

---

## 四、三階段關係

```
原則確立 → 五年戰略 → 高效日常
    ↑              ↓
    └──── 反馈 loop ────┘
```
