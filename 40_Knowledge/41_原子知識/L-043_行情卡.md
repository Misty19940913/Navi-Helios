# 行情卡 (Market Card)

## 核心定義

行情卡是財富流沙盤中的市場資訊卡，用於判斷投資機會的可行性。

## 使用時機

1. 停在機會格
2. 評估投資選項
3. 決定是否執行

## 卡片內容

| 資訊 | 說明 |
|------|------|
| 市場代碼 | 股票代號/產品名 |
| 價格 | 當前市場價格 |
| 趨勢 | 上漲/下跌/持平 |
| 風險等級 | 高/中/低 |
| 預期報酬 | % |

## 投資選項

### 大機會
| 類型 | 說明 |
|------|------|
| 房地產 | 不動產投資 |
| 企業 | 創業或入股 |

### 小機會
| 類型 | 說明 |
|------|------|
| 股票 | 金融商品 |
| 基金 | 投資工具 |
| 副業 | 兼職創業 |

## 判斷邏輯

```
1. 市場趨勢 → 判斷方向
2. 價格 vs 價值 → 評估合理性
3. 風險承受 → 決定仓位
4. 資金配置 → 執行
```

> [!abstract] 核心洞見
> 行情卡告訴你「市場現在怎麼看」，但它不能告訴你「你應該怎麼看」。在現實中，大多數人看到行情下跌就恐慌，看到上漲就追高——而真正的投資者有自己的判斷框架。
