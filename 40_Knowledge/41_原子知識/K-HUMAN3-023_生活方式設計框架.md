# 生活方式設計框架 (Lifestyle Design Framework)

## 核心定義

最有效的發展方式不是設計一個「理想目標」然後強迫執行，而是識別並解決阻礙自然整合的「主約束因素 (Constraint)」。這是一種「問題優先」的智慧進化論。

## 最小有效發展 (MED)

不追求瞬間的全面革命，而是尋找那個能釋放最多資源（時間/精力）的最小槓桿點。

## 四步 SOP

| 步驟 | 內容 |
|------|------|
| 1. 識別首要限制 | 找出「解決它就能解鎖多個象限」的問題 |
| 2. 應用 MED | 對該限制因素實施最小改變 |
| 3. 資源循環 | 將解決第一個問題後釋出的資源，重新投入下一個限制因素 |
| 4. 級聯機會觀察 | 觀察解決問題後的連鎖反應 |

## 行動指南

解決「能量分配」這個主要限制——透過 MED 建立自動化的筆記流程，就是解決「認知過載」的實踐。

> [!abstract] 核心洞見
> 生活方式設計是指透過系統性解決現實問題，讓多維度的整合狀態自然湧現的過程。
