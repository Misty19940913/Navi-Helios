# 目標定向行為 (Teleological Behavior)

## 核心定義

所有人類行為都是目標導向的。當我們無法達成正面目標時，通常是因為潛意識中正在追求一個隱藏的「目標」，例如為了安全感、保護身分認同而選擇拖延或失敗。

## 行為動機分析

| 行為 | 潛意識目標 |
|------|-----------|
| 拖延 | 保護自己免受評價的壓力，維持不敗的幻覺 |
| 留在死胡同工作 | 追求安全性與可預測性，規避風險帶來的恐懼 |
| 憤怒/攻擊 | 修補受威脅的身分認同，奪回控制感 |

## 身分認同的解構周期

```
確立目標 → 建立知覺過濾器 → 獲取反饋 → 習得行為 → 固化身分 → 防禦身分
```

## 行為改變：第一原理

- **錯誤路徑**：試圖改變行動來達到目標（需要大量意志力，容易失敗）
- **正確路徑**：改變你是誰（身分認同）。當身份轉變後，不符合新身份的行為會讓你感到「噁心」或「排斥」

> [!abstract] 核心洞見
> 如果你遲遲不動，不要問自己為什麼「沒毅力」，而要問：「我維持目前的現狀，是在達成什麼樣的潛意識目標？」
