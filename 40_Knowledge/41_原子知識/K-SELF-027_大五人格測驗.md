# 大五人格測驗

> [!abstract] 核心洞見 (Insight)
> 大五人格測驗（Big Five Inventory）測量五大人格維度，比 MBTI 更有研究基礎，預測力更強。

### 定義與機制 (What & How)
- **核心定義**：五維度人格評估模型
- **五大維度**：

| 維度 | 說明 |
|------|------|
| 開放性 (Openness) | 創造力、好奇心 |
| 盡責性 (Conscientiousness) | 組織、責任感 |
| 外向性 (Extraversion) | 社交、活力 |
| 親和性 (Agreeableness) | 合作、信任 |
| 神經質 (Neuroticism) | 情緒穩定度 |

### 實踐與案例 (Action & Proof)
- **優勢**：
  - 比 MBTI 更具研究基礎
  - 預測力更強
  - 更能描述人格的複雜性
- **在人生設計中的應用**：
  - 深層自我了解
  - 職業適配性評估
  - 團隊組合成員分析
