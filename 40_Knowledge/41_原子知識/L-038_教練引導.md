# 教練引導 (Coach Guidance)

## 核心定義

教練引導是財富流沙盤的核心，透過專業帶領幫助玩家覺察潛意識盲點。教練不是給答案，而是引導你發現自己的模式。

## 教練職責

| 職責 | 說明 |
|------|------|
| 帶領推演 | 主持遊戲流程 |
| 執行紀律 | 維護遊戲規則 |
| 觀察覺察 | 發現玩家行為模式 |
| 盤後點評 | 深度反思引導 |

## 核心技能

### 觀察能力
- 發現行為模式
- 覺察情緒反應
- 識別信念顯現
- 捕捉關鍵時刻

### 提問技術
- 開放式問題
- 反思性問題
- 覺察性問題
- 行動性問題

## 帶領流程

```
1. 遊戲開始 → 說明規則
2. 進行中 → 觀察記錄
3. 事件發生 → 適時提問
4. 遊戲結束 → 深度覆盤
```

## 教練心態

- 中立陪伴
- 不代為決策
- 引導覺察
- 支持成長

> [!abstract] 核心洞見
> 一個好的教練不會告訴你「應該怎麼做」，而是問「你看到了什麼」。覺察是改變的起點，教練只是點燈的人。
