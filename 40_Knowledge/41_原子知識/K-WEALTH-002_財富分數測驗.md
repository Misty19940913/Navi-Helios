# K-WEALTH-002 財富分數測驗

對五大資產進行 1-10 分的快速自評，找到最需要優先修復的資產類別。

## 測驗格式

| 資產類別 | 自評分數（1-10） | 備註 |
|---------|----------------|------|
| 時間財富 | | |
| 社會財富 | | |
| 心理財富 | | |
| 身體財富 | | |
| 金錢財富 | | |

## 解讀原則

- **任一項目 ≤ 4 分**：該領域存在顯著失衡，優先處理
- **任一項目 ≥ 8 分**：該領域狀態良好，維持即可
- **最高與最低分差距 ≥ 4 分**：差距過大，整體財富被短板拖累

## 使用頻率

建議每季度進行一次，重新檢視財富平衡的變化趨勢。
