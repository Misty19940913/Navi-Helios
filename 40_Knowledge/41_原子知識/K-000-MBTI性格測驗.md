# MBTI性格測驗

> [!abstract] 核心洞見 (Insight)
> MBTI（Myers-Briggs Type Indicator）是一種了解性格類型與偏好的心理測驗工具，幫助認識自己的傾向。

### 定義與機制 (What & How)
- **核心定義**：性格類型評估工具
- **四個維度**：

| 維度 | 類型 |
|------|------|
| 能量方向 | 外向(E) / 內向(I) |
| 資訊獲取 | 感覺(S) / 直覺(N) |
| 決策方式 | 思考(T) / 情感(F) |
| 生活風格 | 判斷(J) / 知覺(P) |

### 實踐與案例 (Action & Proof)
- **在人生設計中的應用**：
  - 職業方向探索
  - 團隊合作理解
  - 溝通風格認識
- **限制**：
  - 過度簡化複雜人格
  - 測驗結果可能受當下狀態影響
  - 不應作為職業選擇的唯一依據
- **建議**：
  > 心理測驗可以作為自我探索的起點，但不應成為唯一依據。結合時光日誌、人生指南針等工具使用。
