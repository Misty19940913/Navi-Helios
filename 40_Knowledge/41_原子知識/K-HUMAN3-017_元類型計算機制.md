# 元類型計算機制

## 核心定義

元類型（Metatype）是四大象限優勢組合而成的整體形象，代表整體發展狀態。透過象限評估得知你在每個象限的等級（1-3），然後計算複合形態。

## 象限評估

| 象限 | 評估重點 |
|------|---------|
| Mind（心智） | 認知能力、情緒管理、決策質量 |
| Body（身體） | 健康、精力、體能 |
| Spirit（精神） | 意義感、價值觀、人際連結 |
| Vocation（職業） | 財務主權、時間主權、職業滿意度 |

## 常見複合形態

| 類型 | 特徵 |
|------|------|
| 行政官 | 高職業、高心智、低身體 |
| 戰士僧侶 | 高身體、高精神、低職業 |
| 泰坦 | 身體+職業雙高 |
| 探索者 | 高精神、高心智 |

> [!abstract] 核心洞見
> 元類型不是固定的標籤，而是動態的發展狀態快照。隨著每個象限的進化，你的複合形態也會持續改變。
