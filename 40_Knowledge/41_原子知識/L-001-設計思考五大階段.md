# 設計思考五大階段

> [!abstract] 核心洞見 (Insight)
> 設計思考（Design Thinking）是一個以人為本、反覆迭代的問題解決流程，是人生設計的核心方法論。

### 定義與機制 (What & How)
- **核心定義**：以人為本的創意解題流程
- **五大階段**：同理 → 定義 → 發想 → 原型 → 測試

| 階段 | 英文 | 核心問題 |
|------|------|----------|
| 同理 | Empathize | 我理解自己/他人的需求嗎？ |
| 定義 | Define | 我真正要解決的問題是什麼？ |
| 發想 | Ideate | 有哪些可能的解決方案？ |
| 原型 | Prototype | 如何用最小成本測試這個想法？ |
| 測試 | Test | 這個方案真的有效嗎？ |

### 實踐與案例 (Action & Proof)
- **在人生設計中的應用**：
  - 同理：自我盤點、現況覺察
  - 定義：釐清人生問題（如「我想過什麼樣的生活？」）
  - 發想：奧德賽計劃、三版本五年規劃
  - 原型：原型對話、原型體驗
  - 測試：從行動中學習、持續迭代
- **核心精神**：
  - 以人為本：從人的需求出發
  - 迭代而非完美：快速原型、反覆測試
  - 跨界整合：融合理性分析與創意直覺
