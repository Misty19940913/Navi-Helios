# 平流層 (Plain Layer)

## 核心定義

平流層代表大多數人的日常迴圈，如同從小被教導好好讀書、找好工作結婚生子的循環。使用1顆骰子，模擬穩定的日常生活。

## 生活特徵

- 穩定的日常工作
- 按部就班的人生
- 追求安全感
- 依賴主動收入

## 核心行動

1. 工作賺錢
2. 投資理財
3. 建立被動收入
4. 控制支出

## 常見格位

| 格位 | 事件 |
|------|------|
| 薪水 | 領取月現金流 |
| 機會 | 抽取投資機會 |
| 結婚 | 抽取相親卡 |
| 生孩子 | 支出精力與金錢 |
| 慈善 | 捐贈獲得銀翅膀 |

## 通過條件

```
被動收入 > 總支出 → 進入逆流層（有機會）
```

## 脫離平流層的關鍵

- 增加被動收入來源
- 控制不必要的支出
- 學習投資理財
- 建立正確金錢觀

> [!abstract] 核心洞見
> 平流層不是敵人，而是大多數人積累初始資本的必經階段。關鍵是在平流層時就開始建立被動收入系統，而不是等到「有錢了再投資」。
