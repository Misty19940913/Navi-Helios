# 三層世界觀 (Three-Layer Worldview)

## 核心定義

三層世界觀是財富流沙盤的空間架構，反映人生的三種財務狀態與體驗。

## 空間層級

| 層級 | 象徵意義 | 骰子數量 | 核心特徵 |
|------|----------|----------|----------|
| 平流層 | 複製多數人的日常迴圈 | 1顆 | 累積被動收入 |
| 逆流層 | 人生低谷與強制沉澱 | 1顆 | 覺察卡+恩典脫困 |
| 順流層 | 財務自由與夢想實現 | 2顆 | 三種贏家目標 |

## 各層詳細說明

### 平流層
- 生活型態：讀書、工作、結婚、生子
- 目標：累積被動收入 > 總支出
- 通過條件：被動收入突破總支出

### 逆流層
- 常見事件：失業、離婚、生病
- 通過工具：覺察卡反思 + 恩典脫困
- 心理意涵：重新審視金錢信念

### 順流層
- 生活狀態：財富自由、夢想實現
- 目標：三種贏家（夢想家/創富家/慈善家）
- 核心心態：賺錢如同呼吸般簡單

## 層級轉換邏輯

```
平流層 → 被動收入 > 總支出 → 觸發進入逆流層
逆流層 → 覺察並脫困 → 進入順流層（財務自由）

或

平流層 → 直接進入順流層（少數幸運路徑）
```

## 與生命之輪對應

| 三層世界觀 | 生命之輪象限 |
|------------|--------------|
| 平流層 | 身體/關係 |
| 逆流層 | 事業/心靈 |
| 順流層 | 全部平衡 |

## 達到順流層的關鍵

1. 被動收入 > 總支出
2. 財務知識與技能
3. 風險管理能力
4. 覺察與行動力

> [!abstract] 核心洞見
> 三層世界觀不只是遊戲模型，它是一套診斷當前財務位置的工具。多数人困在平流層，是因為沒有完成「被動收入 > 總支出」的臨界點突破。
