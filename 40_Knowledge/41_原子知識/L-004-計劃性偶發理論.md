# 計劃性偶發理論

> [!abstract] 核心洞見 (Insight)
> 計劃性偶發理論（Planned Happenstance Theory）主張「幸運」不是等來的，而是創造出來的。

### 定義與機制 (What & How)
- **核心定義**：主動創造幸運的理論
- **提出者**：John Krumboltz
- **核心概念**：幸運不是偶然，而是可以透過行動製造

| 技能 | 說明 |
|------|------|
| 好奇心 | 主動探索新領域，而非被動等待 |
| 堅持性 | 即使遭遇失敗仍繼續嘗試 |
| 彈性 | 將阻礙轉化為機會 |
| 樂觀 | 相信每次經驗都有潛在價值 |
| 風險承擔 | 願意嘗試新事物 |

### 實踐與案例 (Action & Proof)
- **與人生設計的關聯**：
  - 原型對話：主動創造機會
  - 原型體驗：將「運氣」轉化為「機會」
  - 行動導向：不斷嘗試、從中學習
- **實踐意涵**：
  1. 主動出擊：不要等待機會，要創造機會
  2. 將拒絕視為資訊：繼續嘗試而非放棄
  3. 開放接受意外：有時最棒的機會來自意外
