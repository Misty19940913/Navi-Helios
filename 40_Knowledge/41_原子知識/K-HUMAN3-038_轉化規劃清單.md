# 轉化規劃清單 (Transformation Roadmap)

## 核心週期架構

系統性轉化不能靠意志力硬推，而是需要分階段的架構化路徑。透過 30/90/180 天的週期設計，將原本「不穩定」的實驗轉化為「自動化」的生活方式，實現位階的永久躍遷。

| 階段 | 天數 | 名稱 | 目標 |
|------|------|------|------|
| T+30 | 問題識別 | Problem Recognition | 透過負面願景鎖定首要限制因素，建立每日 15-30 分鐘的最小可行實踐 |
| T+90 | 方案落實 | Implementation | 進入「著迷期」，磨練特定技能，利用強大象限支撐弱勢象限 |
| T+180 | 生活整合 | Integration | 讓新行為成為「預設狀態」，實現生活原型從失衡向整合者的動態轉移 |

## 一天重新設計生命 Protocol

這套 Protocol 可用於年度自我重設，需要一整天完成。所需工具：紙、筆、開放的心態。

### 三階段結構

成功翻轉身份的人通常經歷三個階段：
1. **失調 (Dissonance)** — 感到不屬於現在的生活，對缺乏進展足夠厭倦
2. **不確定 (Uncertainty)** — 不知道下一步，陷入實驗或迷失
3. **發現 (Discovery)** — 發現真正想追求的東西，6個月內完成6年的進展

---

### Part 1 — 早晨：心理挖掘 + Anti-Vision + Vision MVP

**目標**：建立新的認知框架或感知鏡頭

這像是創造一個新的外殼，離開舊外殼，慢慢成長為它。一開始不會覺得合適——那是好事。

**Step 1 — Anti-Vision（拒絕的地獄）**

用 15-30 分鐘（參考：一個 YouTube 影片的長度）誠實回答：

1. **你學會忍受的平淡不滿是什麼？**
2. **你重複抱怨但從不改變的三件事是什麼？**
3. **每個抱怨的反面：你實際上想要什麼？**
4. **你不敢向尊敬的人承認的真相是什麼？**

然後具象化地獄：
1. **五年後的普通週二**：在哪醒來？身體感覺？9-6點做什麼？晚上10點感覺？
2. **十年後**：錯過什麼？機會之門關閉？誰放棄了你？
3. **生命盡頭**：代價是什麼？從未感受、嘗試或成為什麼？
4. **你在避免成為誰？**
5. **你必須放棄什麼身份？**（「我是那種人……」）社交代價多大？
6. **最讓你尷尬的理由**：什麼讓你顯得軟弱/害怕/懶惰？

**Step 2 — Vision MVP（最小可行願景）**

忘掉實用性。如果可以彈指之間在三年後活在不同的生活：

1. **描述那個理想週二**：與 Step 1 同樣的細節層次
2. **身份陳述**：什麼信念讓那個生活感覺自然而非強迫？「我是那種人……」
3. **本週行動**：如果你已經是那個人，你會做的一件事是什麼？

---

### Part 2 — 白天：中斷自動駕駛（6個時間戳 + 3個通勤提示）

**目標**：打破讓你停滯的無意識模式

在手機設定提醒，問題要包含在提醒裡，這樣你立刻就能開始思考。

**時間戳問題（設定在行進/散步/躺著的時段）：**

| 時間 | 問題 |
|------|------|
| 11:00 | 我現在透過正在做的事在逃避什麼？ |
| 13:30 | 如果有人拍下過去兩小時，他們會推斷我想從生活中得到什麼？ |
| 15:15 | 我是在走向我討厭的生活還是我想要的生活？ |
| 17:00 | 我在假裝什麼不重要，但實際上最重要？ |
| 19:30 | 今天我做的哪些事是為了保護身份而不是真正渴望？ |
| 21:00 | 今天什麼時候我最活著？什麼時候我最死？ |

**通勤/散步問題：**

- 如果我不再需要別人把我看成[步驟2寫下的身份]，什麼會改變？
- 我在哪裡用生機換安全？
- 我想成為的人的最小版本是什麼，我明天就可以成為？

---

### Part 3 — 晚間：整合洞見

**目標**：將洞察轉化為方向

1. **今天的洞察**：什麼感覺最真實，關於你為什麼一直卡住？
2. **真正的敵人**：清楚地命名。不是環境，不是別人。是什麼內部模式或信念一直在操控？
3. **Anti-Vision 壓縮句**：一句話捕捉你拒絕讓生活變成什麼。讀的時候要有感覺。
4. **Vision MVP 句**：一句話捕捉你在建築什麼。

**設定目標（作為觀點而非終點）：**

- **一年鏡頭**：一年後，怎麼樣才知道你打破了舊模式？一件具體的事。
- **一個月鏡頭**：一個月後需要什麼，才能讓一年鏡頭保持可能？
- **每日鏡頭**：明天你能時間封鎖的 2-3 個行動是什麼？

> 目標是不可靠的——它們會改變，讓你感到束縛。目標是觀點。是讓你進入正確心態去執行能讓你遠離不想的生活的行動的鏡頭。不要担心某種終點線。

## 動態調整

每 30 天進行一次適應性評估，因為舊問題解決後，新問題（更高階的挑戰）必然會浮現。

## 行動指南

1. **立即行動**：寫下 24 小時內可以完成的一個「微型解決方案」
2. **問責機制**：加入或建立支持性的社群，防止在不確定性階段退縮
