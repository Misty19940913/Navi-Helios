# 第三至四年：入場年與轉移年

> [!abstract] 核心洞見 (Insight)
> 澳洲移民第三階段（第3年）與第四階段（第4年）：入場年與轉移年的任務

### 定義與機制 (What & How)
## 一、第 3 年：入境年

### 核心目標
驗證能否在澳洲不墜階層

### 三大任務
| 任務 | 說明 |
|------|------|
| 取得澳洲入場資格 | 技術移民/學簽/商業簽 |
| 短期實地生活 3-6 個月 | 工作/通勤/租屋/社交 |
| 測試雙軌收入 | 台灣線上 + 澳洲本地 |

### 驗證重點
- 生活成本是否可負擔
- 當地工作機會
- 社會適應程度

---

## 二、第 4 年：轉移年

### 核心目標
澳洲成為主場

### 三大任務
| 任務 | 說明 |
|------|------|
| 穩定收入移到澳洲 | 正職/合作/本地客戶 |
| 身份/簽證布局 | PR 或可續簽方案 |
| 制度性資產配置 | Super/房地產/稅務 |

### 資產配置建議
| 項目 | 說明 |
|------|------|
| Super | 退休基金 |
| 房產 | 自住或投資 |
| 稅務 | 合理節稅規劃 |
