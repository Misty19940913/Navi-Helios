# 知識數量是假的

你的 vault 有 3,000 條筆記。恭喜。**（1 鉤子）** 但有 200 條孤立知識、50 個檔案的 YAML 不全——「3,000」這個數字是安慰劑，不是實力。**（3 論述）** 系統顯示問題，你才會修正；顯示擁有什麼，你只會更焦慮。**（1 落點）**
