#### **一、睡眠週期監控模板**
```markdown
## 😴 {{DATE}} 睡眠報告  
**總睡眠時間**：<% tp.file.cursor(1) %>  
**完整週期數**：<% tp.file.cursor(2) %>  
**睡眠效率**：<% tp.file.cursor(3) %>%  
**深睡比例**：<% tp.file.cursor(4) %>%  
**REM比例**：<% tp.file.cursor(5) %>%  
```

##### **科學化指標計算**
```dataviewjs
const 睡眠數據 = dv.current();
const 理想週期 = Math.round(睡眠數據.總睡眠時間 / 1.5);
const 睡眠效率 = (睡眠數據.完整週期數 / 理想週期 * 100).toFixed(1);
dv.span(`**睡眠質量評分**：${睡眠效率}%`);
```

##### **動態建議算法**
```dataviewjs
const 作息記錄 = dv.pages('"能量日誌"')
  .where(p => p.睡眠效率 > 80);
const 最佳入睡 = 作息記錄.map(p => p.開始時間).mode();
dv.span(`**歷史最佳入睡時間**：${最佳入睡}`);
```
