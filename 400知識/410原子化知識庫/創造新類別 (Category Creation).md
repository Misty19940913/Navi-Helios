# 創造新類別 (Category Creation)
> [!abstract] 核心洞見 (Insight)
> {{這是一個什麼概念？用一句話說明它能解決什麼核心痛點或帶來什麼槓桿。}}

### 定義與機制 (What & How)
> [!abstract] 來源：[[232-131通才者的優勢]]


透過合成獨特的興趣與技能，創造一個獨一無二的利基市場，從而徹底消除競爭。

1. **唯一性策略**：你不必做得「更好」，因為你是「唯一」。
2. **利基縮小與價值擴大**：將受眾鎖定在特定交集（如：專為健身創作者設計的心理學），雖然受眾變少，但解決問題的精準度與溢價能力大幅提高。
3. **個人品牌即類別**：當你的名字代表了一種獨特的解決方案時，你就成功創造了新類別。

### 實踐與案例 (Action & Proof)
- 嘗試用一句話定義你的業務：我是專門為 [ A 群體 ] 解決 [ B 問題 ] 的 [ C 專家 ]。
- 確保 A, B, C 的交集具有足夠的獨特性。
