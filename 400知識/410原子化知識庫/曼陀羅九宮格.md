# 曼陀羅九宮格
> [!abstract] 核心洞見 (Insight)
> 曼陀羅九宮格是一種高效的創意擴展與整合工具，它透過視覺化的矩陣結構，幫助我們有條理地發散、歸納與深化思考，提升問題解決和創新效率。

### 定義與機制 (What & How)
- **官方描述**：曼陀羅九宮格是一種融合東方哲學智慧的矩陣式思考工具，其靈感源自藏傳佛教的曼陀羅圖騰與中國古代的河圖洛書。它採用 3x3 的格子結構，將核心主題置於中央，向八個方向展開相關子主題，每個子主題又可作為新的中心進行遞迴展開，形成結構化的思考網絡。
- **我的理解**：曼陀羅九宮格就像是你的「創意發動機」和「思考收納盒」。當你面對一個複雜問題或想發想新點子時，你把核心放中間，然後周圍八格快速填入相關的想法。當一個想法很吸引人時，你就把它拉到下一個九宮格的中央，繼續發散八個更細節的想法。這樣一來，你的思緒就像被按了「放大鍵」一樣，既有廣度又有深度，還能保持清晰的結構。

- **運作原理**：此模型運作的核心是透過「中心聚焦」與「放射性擴展」的結合，模擬大腦的聯想機制，但同時提供結構化的框架。它將發散性思考與收斂性思考整合，透過視覺化的矩陣結構，建立主題間的連接與關聯，促進思考的系統性與全面性。遞迴展開的特性，使得思考可以不斷深化，從而發現更底層的洞察或更全面的解決方案。
- **關鍵變數**：
    1.  **中心聚焦原則**: 核心問題或主題置於中央，確保思考始終圍繞關鍵目標展開。
    2.  **擴展平衡性**: 八個方向均衡發展，避免思考偏向單一維度或方向。
    3.  **層級遞迴性**: 支持多層次展開，每個子主題可作為新的中心再次展開九宮格，實現思考的廣度與深度。
    4.  **整合聯結性**: 強調各子主題間的關聯和整合，而非孤立思考，促進全面解決方案的形成。
    5.  **視覺結構化**: 通過格子結構，將抽象思考具象化，增強記憶與理解。

### 實踐與案例 (Action & Proof)
- **具體案例**：產品研發應用：一家科技公司使用曼陀羅九宮格開發新產品。中心格填入「智能健康手環」，周圍八格分別展開為「目標用戶」、「核心功能」、「設計風格」、「材料選擇」、「電池續航」、「數據分析」、「行銷策略」和「競爭優勢」。隨後，團隊將「核心功能」置於中心，再次展開九宮格，細化為「心率監測」、「睡眠追蹤」等具體功能。通過三層九宮格展開，團隊全面覆蓋了產品開發的各個維度，最終開發出集功能、外觀、用戶體驗於一體的成功產品。
- **行動指南**：
    1.  **確立核心主題**：繪製一個 3x3 的九宮格，在中心格寫下你想要解決的核心問題或主題。
    2.  **發散子主題**：圍繞中心，在周圍八個格子寫下與核心相關的八個子主題、子問題或思考方向。
    3.  **遞迴深化**：選擇一個最有潛力的子主題，將其視為新的中心，繪製新的九宮格進行下一層的細節展開。
    4.  **重複與連結**：重複步驟 3，直到思考達到所需的深度或廣度。分析所有格子內容，尋找連接點和整合方向，形成完整的解決方案。
