# H3.0-退行機制 (Regression Mechanics)

> [!ABSTRACT] 核心定義
> 退行是指個體在壓力、創傷或重大環境動盪下，暫時或持續回歸早期（較低層級）的發展模式。這不是一種「失敗」，而是大腦自我保護的「安全模式」重啟，旨在將精力集中於最基礎的生存防禦。

### 1. 退行的三大類型

| 類型 | 特徵 | 影響期間 |
| :--- | :--- | :--- |
| **急性退行** | 由特定突發事件觸發。知識存在但無法調用，行為轉向應激。 | 數日至數月 |
| **慢性退行** | 隨著時間緩慢退化。涉及多個象限與身分認同混亂。 | 長期、半永久 |
| **級聯退行** | 單一象限崩潰引發連鎖崩壞，進入生存模式主導。 | 迅速爆發 |

### 2. 層級特有的退行觸發點

*   **3.0 → 2.0 (整合者轉向個體者)**:
    *   成功導致自滿、與教導者角色脫節、文化/社群排斥、意義結構坍塌。
*   **2.0 → 1.0 (個體者轉向從眾者)**:
    *   壓倒性的挫敗、威權（父母/社會）形象回歸、極度資源匱乏、遭受嚴重的身分攻擊。
*   **文明/元危機觸發**:
    *   生態哀慟、文明崩裂產生的焦慮、資訊超載、認知作戰導致的社群破碎。

### 3. 退行的神經科學背景
退行本質上是 **「杏仁核劫持 (Amygdala Hijack)」** 的長期化。當個體感覺生存受威脅，前額葉皮質（高層級思考區）的功能會被抑制，意識會退回熟悉且自動化的早期「規則系統」或「避險行為」。

---
> [!IMPORTANT] 關鍵認知
> 退行是螺旋發展的一部分。識別退行而不責備自己，是恢復功能的第一步。當你在退行時，不要試圖去挑戰複雜的新項目，那只會加重焦慮。現在是你該啟動 **[[H3.0-退行恢復協議]]** 的時候。
