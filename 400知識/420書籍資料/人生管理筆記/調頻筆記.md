情緒是一個送信人，傳達內心的聲音。
我們很容易根據自己對情緒感受，進行評判。但事實上，情緒的出現都是帶著潛意識的訊息來與我們溝通。
如果你願意接納與理解，情緒自然而然就會離去，
負面情緒有時候是在提醒我們：內心還有傷口，請求釋放傷痛、重獲自由。
越強的情緒，往往包含越重要的訊息，如果不接受、不解讀，他就會反覆出現，以此來提醒我們看見。

「改變提問，改變人生」一書中提及，「學習者心態」、「批判者心態」兩種心境。
學習者心態：聚焦於如何正確解決問題、克服困難
批判者心態：往往只會指責、抱怨、批評、推卸責任，或是往自己身上貼負面標籤。

美國社會心理學家利昂．費斯廷格(Loen Festinger)：「生活中的10%事由發生在你身上的事情組成的，而另外的90%則由你對所發生的事情如何反應決定。」

情緒爆發時，製作一份紀錄，寫出事件、觀察情緒、嘗試轉念、紀錄結果。
用途：調整和轉化情緒
 * 核心目標：
 * 價值轉化：
 * 實踐：
 * 習慣養成：
書寫模板：

| 日期  | 事件  | 情緒問題 | 轉念思考 | 結果  |
| --- | --- | ---- | ---- | --- |
|     |     |      |      |     |

書寫步驟：
1. 紀錄：寫下事件本身，不指責自我，不批判當下情緒
2. 感受：寫下情緒，感受其中的不愉快，發覺背後問題，分析是否有潛藏的限制性信念。
			去思考情緒來自哪裡?是否經查發生?
			情緒起伏的強度往往與以下三要素有關：
			　1.過去積累的創傷沒有得到療育，與信念相關。
			　2.心裡承受能力
			　3.當嚇人事物的刺激程度
3. 轉念：換一個積極視角看待，想想可以透過此事件學到什麼
			先不執著對與錯，而是問自己「事件是否有積極的一面?我還能採取什麼行動來改善?我可以從中學習到什麼?
4. 結果：你會發現很多情緒是不必要的，事情沒有那麼糟。
