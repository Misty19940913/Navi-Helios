類[[生命歷程探索]]

---
製作一份自己的大事紀要，寫出人生中的重大事件，再對每個事件去判斷，如果替換當時反應及作法會有什麼不同。
用途：經驗解構與知識遷移
 * 核心目標：梳理生命歷程，累積核心的知識遷移能力 
 * 價值轉化：將分散的經驗和知識轉化為自身競爭力 
 * 實踐：解構經驗、提取通用知識和解決方案模式 
 * 習慣養成：定期回顧過去的經驗，累積成為高效能人士 
書寫模板：

| 日期  | 事件  | 感受、情緒與信念 | 成功或失敗的模式 | 現在給予過去的建議 |
| --- | --- | -------- | -------- | --------- |
|     |     |          |          |           |

書寫步驟：
1. 回顧：回憶並寫下事件的細節
2. 思考：註記當時事件帶來的感受，尋找隱藏在創傷背後的信念
3. 總結：失敗或成功的原因；透過爬梳過去，汲取經驗
4. 重寫：將關注點放在「改變」，以全新視角審視過去的問題。放棄受害者或反面人物的角色，以積極的態度重寫。
>[!]過去害怕擔憂的情況，是否真的有害?還是自己的想像或偏見?現在有沒有可能嘗試
>若過去的方式走不通，是否嘗試逆向操作?
