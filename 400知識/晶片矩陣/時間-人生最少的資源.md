## 時間不多了
是否曾經想過如何度過這一生？
假設你活了80歲，你是否知道你花了多少時間各種事情上，包含睡覺、工作、讀書、玩樂？
下面的表格或許會給你意想不到的答案，

| 事務  | 每日時間(時) | 起始歲數 | 終點歲數 | 持續年份 | 總耗費年份    |
| --- | ------- | ---- | ---- | ---- | -------- |
| 睡覺  | 8       | 0    | 80   | 80   | 26.66667 |
| 讀書  | 8       | 6    | 22   | 16   | 5.333333 |
| 工作  | 8       | 22   | 60   | 38   | 12.66667 |
| 飲食  | 2.5     | 0    | 80   | 80   | 8.333333 |
| 社交  | 2       | 0    | 80   | 80   | 6.666667 |
| 通勤  | 1.5     | 0    | 80   | 80   | 5        |
| 娛樂  | 2       | 0    | 80   | 80   | 6.666667 |
| 總結  |         |      |      |      | 71.33333 |
| 剩餘  |         |      |      |      | 8.666667 |
扣掉一些常態事物人生剩下8.6年的自由時間。
這個時間對所有人是公平的。

## 時間使用方式
「時間」有三種使用方式
「消費」是維持生存所需的
「投資」是讓自己幸福的
浪費是無法帶來幸福的
是不是感覺與金錢觀很像?
沒錯，但最大的不同在於時間無法結餘。

如何有效利用時間在這樣的分析之下一目了然，「消費」因為是生存所需，因此幾乎不變，僅剩「投資」、「浪費」，傻子都知道接下來該如何做。
減少「浪費」、增加「 投資」
