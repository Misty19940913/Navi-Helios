# 330-XXX 交易風險與離職協議 (Vocation Governance)

此文件根據 **Navi 3.0 轉型策略** 中對 **高風險加速器** 的治理要求而建立，旨在確保在職業象限使用外匯保證金交易時，能夠執行 **計算風險 (Calculated Risk)**，並防止在壓力下再次陷入心態崩壞。

## A. 交易最大損失容忍度 (Max Loss Tolerance)

**目的：** 設定明確的**終止條件**，確保交易行為不會演變為 **基質消耗** 或 **心態崩壞**，嚴格遵守 **Level 2.5+ 加速器** 執行的先決條件。

| 項目 | 數值 | 協議內容 |
| :--- | :--- | :--- |
| **當前交易本金** | 150,000 新台幣 | 作為當前加速器的啟動資金。 |
| **最大損失容忍度** | 30% | 這是允許的最大虧損比例。 |
| **明確止損金額** | **45,000 新台幣** | 當交易帳戶總虧損達到 **4.5 萬新台幣** 時，**必須立即停止所有交易行為**，並啟動 **整合實踐 (Integration practices)**，對交易策略和心態進行徹底覆盤。 |

## B. 備用金達標離職數字 (Exit Strategy)

**目的：** 定義從 **1.0 級保障 (刑事警察)** 轉向 **超級個體/自由工作者** 的明確 **出場策略**。此數字旨在提供 **2 年** 的生活緩衝，以應對 **加速器** 失敗或第二事業初期不穩定的風險。

| 項目 | 數值 | 協議內容 |
| :--- | :--- | :--- |
| **備用金達標離職數字** | **2,400,000 新台幣** | 當 **第二事業** 及 **投資收入** 累積的備用金達到此數字時，即滿足離職的 **財務先決條件**。 |
| **離職觸發條件** | 備用金達標 **且** 第二事業收入足以覆蓋生活支出。 | 在滿足雙重條件後，應立即執行 **極端生活改變 (Extreme Life Changes)** 加速器，以最大化 **職業象限** 的發展。 |
