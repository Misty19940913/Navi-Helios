# 070: 應用案例

學習他人如何應用人生設計原則，可以為我們帶來深刻的啟發。這裡彙整了一些虛構但具代表性的案例，展示了不同背景的人如何透過設計思維，主動創造更符合自己價值觀的生活。

## 案例一：從行銷專員到 UX 研究員的轉型之路

**主角：** 小陳，30歲，在一家消費品公司擔任行銷專員五年。

- **困境**：小陳對日益重複的行銷活動感到厭倦，覺得工作缺乏深度，無法真正理解消費者。他對科技業感到好奇，但不知道自己能做什麼。
- **行動與探索**：
    1.  **[[時光日誌]]**：他發現自己在做消費者訪談和數據分析時，能量和參與度最高。
    2.  **[[030_多元未來發想|奧德賽計畫]]**：
        -   計畫一：成為資深行銷經理。
        -   計畫二：轉職到科技公司做「使用者經驗(UX)研究員」。
        -   計畫三：開一家小型咖啡館。
    3.  **[[040_原型設計與實踐|原型設計]]**：
        -   **對話**：他透過 LinkedIn 聯繫了三位在不同公司的 UX 研究員，了解他們的工作日常、挑戰與所需技能。
        -   **體驗**：他利用週末時間參加了一個線上的 UX 設計入門工作坊，並主動為一個朋友的獨立開發 App 做了免費的使用者訪談專案。
- **結果**：原型體驗讓他確認了對 UX 研究的熱情。他報名了一個為期三個月的密集課程，並利用自己過去的行銷背景，成功地在一家電子商務公司找到了初階 UX 研究員的職位。

## 案例二：尋找工作與生活平衡的律師

**主角：** 麗莎，42歲，一家知名律師事務所的合夥人。

- **困境**：麗莎事業非常成功，但代價是極長的工作時間、巨大的壓力以及與家人疏離的關係。她的[[生命儀表板]]在「健康」和「愛」兩個領域都亮起了紅燈。
- **行動與探索**：
    1.  **[[020_自我認知與定位|人生指南針]]**：她重新反思自己的生活觀，意識到「家庭關係」和「個人平靜」對她而言，其重要性已超過事業上的成就。
    2.  **[[030_多元未來發想|奧德賽計畫]]**：
        -   計畫一：維持現狀，但更嚴格地劃定工作界線。
        -   計畫二：離開事務所，成為一名獨立的法律顧問，專注於特定領域，並自己掌控工作節奏。
        -   計畫三：移居鄉村，成為一名法律援助律師，服務弱勢群體。
    3.  **[[040_原型設計與實踐|原型設計]]**：
        -   **對話**：她與一位已經成為獨立顧問的前同事，以及一位在非營利組織服務的律師進行了深入交談。
        -   **體驗**：她開始嘗試每週「關閉」手機和電腦一個晚上，完全陪伴家人，測試自己能否真正放下工作。她還利用年假，在一個法律援助中心做了兩週的志工。
- **結果**：志工經驗讓她找回了法律的初心，但她也發現自己還無法完全放棄商業世界的挑戰。最終，她選擇了「計畫二」，辭去合夥人職位，創辦了自己的法律顧問工作室。雖然初期收入減少，但她重新獲得了對時間的掌控權，並顯著改善了家庭關係與個人健康。

## 案例三：整合框架工作坊設計 (BMY + DYL)

這是一個應用人生設計框架來創造產品與服務的案例，特別是針對您作為知識工作者和顧問的潛在需求。

### 核心理念與目標
- **理念**：整合《一個人的獲利模式》(BMY) 的商業分析工具與《做自己的生命設計師》(DYL) 的設計思考方法。
- **目標**：引導學員（或客戶）系統性地：
    1.  描繪並分析現況與理想職涯的九大要素（BMY）。
    2.  應用五種設計師心態（DYL）來解決職涯挑戰。
    3.  透過奧德賽計畫構思三種未來人生路徑。

### 工作坊結構與活動設計

| 模組 | 目的 | 核心活動（工具） | 關鍵產出 |
| :--- | :--- | :--- | :--- |
| **模組一：自我探索與現況盤點** | 釐清自我與現況商業模式 | 生命儀表板、生命羅盤、現況個人商業模式圖、生命歷程圖 | **人生指南針**、**現況 BMY 藍圖** |
| **模組二：願景發想與多重可能性** | 打破思維定勢，探索多元路徑 | 奧德賽計畫、價值主張重塑（腦力激盪）、心智圖 | **三種未來人生腳本** |
| **模組三：原型打造與小步快跑** | 將想法轉為可測試的「原型」| 原型設計與測試（對話/體驗）、BMY 市場測試、好時光日誌分析 | **具體行動計畫**、**市場驗證假設** |
| **模組四：驗證、修正與持續優化** | 從測試結果學習，進行結構化調整 | BMY 痛點診斷與重繪、失敗分類與學習、建立支持網絡 | **優化後的 BMY 藍圖**、**行動承諾** |

### 引導與實踐重點
- **引導者角色**：扮演中立角色，促使團隊成員智慧互相激盪，幫助團隊整合不同思維。
- **關鍵技巧**：嚴格掌握時間控制、根據現場參與者動態靈活調整、運用白板/便利貼進行視覺化呈現。
- **設計師心態**：在工作坊中不斷提醒學員運用「偏向行動」、「重擬問題」和「通力合作」的心態，將失敗視為學習機會。
