# 人生設計總論2.0

## 一、人生設計的理論基礎
人生沒有預設的路徑，需要透過我們的選擇來逐步進行，包含主動和被動的選擇。因此套用一個系統性的方法論，來幫助人們在充滿不確定性的現代社會中，創造有意義且可持續的生活方式。

### 人生需要設計的四大核心原因
1. **人生有限**：時間一直是最有限資源，效率是人生最大的難關。
2. **選擇自由過載**：現代社會提供前所未有的選擇自由，選擇過載導致決策癱瘓，缺乏設計容易陷入隨波逐流
3. **不確定性管理**：設計思維提供應對變動的框架，幫助在變動中保持方向感，建立適應性思維模式
4. **整合性發展** ：將價值觀、能力和熱情整合，創造一致的生活方式，避免內外衝突帶來的不滿足感
