# 人生設計的方法與工具

## 奧德賽計畫設計流程
- 準備階段：
  - 完成工作觀與生活觀的書寫與整合
  - 收集自我評估數據：技能、優勢、價值觀
  - 確定「人生指南針」的方向
  - 準備創意思考的心態與環境
- 計畫構思：
  - 計畫1：現有路徑的延伸與深化
  - 計畫2：若計畫1不可行的替代方案
  - 計畫3：完全不同的道路（若金錢不是問題）
  - 為每個計畫創建簡短標題與描述
- 計畫詳細設計：
  - 五年時間軸：關鍵里程碑與事件
  - 提出每個計畫試圖回答的2-3個問題
  - 描述如何在該計畫中賺錢與生活
  - 列出計畫的優點與缺點
  - 繪製五年展望圖景
- 計畫評估：
  - 資源評估：所需時間、金錢、技能、人脈
  - 喜愛度評分(1-10)：該計畫的吸引力
  - 信心度評分(1-10)：實現的可能性
  - 一致性評分(1-10)：與工作觀和生活觀的連貫性
- 行動與反思：
  - 確定立即可採取的小步驟來測試計畫
  - 撰寫從未來自己給現在的感謝信
  - 設計代表每個計畫的個人符號
  - 反思三個計畫帶來的洞見

## 原型製作的實際方法
- 資訊訪談(Information Interview)：
  - 目的：收集關於特定職業、行業或生活方式的第一手資料
  - 步驟：
    1. 確定訪談目標和問題
    2. 尋找合適的受訪者
    3. 提出清晰、開放性的問題
    4. 傾聽並記錄見解
    5. 表達感謝並反思學習
  - 關鍵問題示例：
    - 你如何進入這個領域？
    - 你日常工作是什麼樣的？
    - 你最喜歡和最不喜歡這工作的什麼？
    - 有什麼建議給想進入這領域的人？
- 影子跟隨(Job Shadowing)：
  - 目的：直接觀察某職業或生活方式的真實情況
  - 步驟：
    1. 找到願意被跟隨的專業人士
    2. 明確設定時間範圍(通常半天到一天)
    3. 準備觀察重點和問題
    4. 在過程中保持開放和觀察力
    5. 事後反思與記錄學習
- 體驗原型(Experience Prototype)：
  - 目的：短期親身體驗某種生活或工作方式
  - 類型：
    - 志願服務
    - 實習或臨時工作
    - 參加工作坊或短期課程
    - 生活方式實驗(如搬遷一週)
  - 關鍵原則：
    - 設定明確的學習目標
    - 限定時間範圍
    - 保持細心觀察自己的反應
    - 收集反饋並調整下一步
- 平行原型(Parallel Prototypes)：
  - 目的：同時測試多個想法，比較結果
  - 實施方法：
    - 選擇2-3個不同方向的小實驗
    - 為每個實驗設定相同的時間框架
    - 使用統一標準記錄反應和結果
    - 分析比較不同原型的體驗

## 創意發想與決策技巧
- 腦力激盪(Brainstorming)技巧：
  - 數量優先原則：追求想法數量而非質量
  - "是的，而且..."(Yes, and...)原則：建立在他人想法上
  - 避免評判：先產生想法，後評估
  - 鼓勵瘋狂想法：打破常規思維
  - 使用視覺輔助：心智圖、便利貼等
- 心智圖(Mind Mapping)方法：
  - 以核心問題為中心
  - 分支擴展相關領域和可能性
  - 使用顏色和圖像增強連接
  - 識別跨領域的機會和連結
  - 發現非線性的潛在路徑
- 決策技巧：
  - "人生不能被優化"原則：接受"足夠好"的決策
  - 反向決策法：排除最不可行的選項
  - 10-10-10規則：評估決策在10分鐘、10個月和10年後的影響
  - "默認為行動"策略：在不確定時選擇行動
  - 情感追蹤：記錄決策過程中的情感變化
- 困境突破方法：
  - "重力問題"識別：辨識卡住我們的根本問題
  - "重新框架"困境：從"我如何..."到"有哪些方法..."
  - "錨定點"設置：在不確定中找到確定的錨點
  - "固定本體，可變條件"法：保持核心價值，改變實現方式
  - 5個"為什麼"技巧：深入挖掘根本原因
