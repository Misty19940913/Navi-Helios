# 232-000 主權人生：Master MOC 2.0 (超級個體生命週期)

此文件定義了超級個體從**心智哲學**到**商業實踐**，再到**高效執行**的完整生命週期與戰術流程，旨在透過系統性地解決 **主約束因素 (Primary Constraint)**，實現 **Level 3.0 的整合型生活方式 (Integrated Lifestyle)**。。

## 階段 I：內核校準與主約束因素診斷 (Root Cause Diagnosis)
核心：確立個體的**真實驅動源**，建立內在指南針，精確定位當前發展等級和**主約束因素（Primary Constraint）**，避免將**願望、焦慮、社會期待**誤當成核心價值觀。
