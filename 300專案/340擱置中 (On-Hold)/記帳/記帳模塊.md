### **記帳模塊與自動分類腳本完整操作指南**  
---

#### **Ⅰ. 記帳模板配置**  

##### **1. 建立基礎模板**  
- **路徑**：`模板文件夾/交易記錄.md`  
- **內容**：  
  ```markdown  
  ---  
  type: 交易記錄  
  date: {{DATE}}  
  amount: {{AMOUNT}}  
  category: {{CATEGORY}}  
  payee: {{PAYEE}}  
  account: {{ACCOUNT}}  
  tags:  
    - #消費/{{AUTO_TAG}}  
  ---  
  ```  

##### **2. 設置QuickAdd快速輸入**  
1. **安裝QuickAdd插件**：  
   - 在Obsidian插件市場搜尋安裝 **QuickAdd**  
2. **新增模板指令**：  
   ```yaml  
   名稱: 新增交易  
   類型: Template  
   模板路徑: /模板文件夾/交易記錄.md  
   熱鍵: Ctrl+Shift+T  
   ```  
3. **啟用變量輸入**：  
   ```javascript  
   // 在QuickAdd設定中啟用「Prompt for Values」  
   variables:  
     DATE: "{{date}}"  
     AMOUNT: "{{value}}"  
     PAYEE: "{{value}}"  
     ACCOUNT: "{{value}}"  
   ```  

##### **3. 使用流程**  
1. 按下 `Ctrl+Shift+T` 觸發指令  
2. 依序輸入：  
   ```  
   金額: -120 (負數為支出)  
   商家: 星巴克  
   帳戶: 信用卡A  
   ```  
3. 自動生成筆記：  
   ```markdown  
   ---  
   type: 交易記錄  
   date: 2023-09-01  
   amount: -120  
   category: 餐飲  
   payee: 星巴克  
   account: 信用卡A  
   tags:  
     - #消費/餐飲  
   ---  
   ```

#### **Ⅱ. 自動分類腳本詳解**  

##### **1. 腳本核心邏輯**  
- **關鍵詞匹配規則**：  
  ```javascript  
  const categories = {  
    餐飲: ["餐廳", "咖啡", "外賣"],  
    交通: ["油費", "地鐵", "uber"]  
  };  
  ```  
- **優先級機制**：  
  ```javascript  
  // 若同時匹配多個關鍵詞，取第一個  
  Object.entries(categories).forEach(([key, values]) => {  
    if (values.some(v => tp.file.title.includes(v))) {  
      autoTag = key;  
      return; // 中斷循環  
    }  
  });  
  ```  

##### **2. 擴充分類規則**  
1. **編輯腳本**：  
   ```javascript  
   // 在Templater腳本中新增分類  
   娛樂: ["電影", "KTV", "演唱會"],  
   教育: ["書籍", "課程", "學費"]  
   ```  
2. **動態學習功能**（需AI整合）：  
   ```javascript  
   // 調用GPT-4分析未分類交易  
   const gptCategory = await tp.ai.generate(`分類"${tp.file.title}"`);  
   autoTag = gptCategory || "其他";  
   ```  

##### **3. 手動修正分類**  
- **直接編輯標籤**：  
  ```markdown  
  tags:  
    - #消費/交通  
    - #誤分類需修正  
  ```  
- **觸發重新分類**：  
  執行命令 `/重新分類` → 重新掃描交易描述

#### **Ⅲ. 數據可視化操作**  

##### **1. 建立儀表板筆記**  
- **路徑**：`財務裝甲/現金流儀表板.md`  
- **內容**：  
  ```dataviewjs  
  const pages = dv.pages('"財務裝甲/交易記錄"')  
    .where(p => p.amount);  
  const income = pages.where(p => p.amount > 0).sum(p => p.amount);  
  const expense = pages.where(p => p.amount < 0).sum(p => p.amount);  
    
  dv.header(3, `💰 本月淨現金流: $${income + expense}`);  
  dv.table(["類別", "支出", "收入"],  
    Object.entries(pages.groupBy(p => p.category))  
    .map(([k,v]) => [  
      k,   
      `$${v.where(p => p.amount < 0).sum(p => p.amount)}`,  
      `$${v.where(p => p.amount > 0).sum(p => p.amount)}`  
    ])  
  );  
  ```  

##### **2. 生成熱力圖**  
```dataviewjs  
const heatmapData = pages  
  .groupBy(p => p.date.toFormat("yyyy-MM"))  
  .map(([month, entries]) => ({  
    month,  
    expense: entries.where(p => p.amount < 0).sum(p => p.amount)  
  }));  
    
window.renderHeatmap(heatmapData); // 需安裝Obsidian-Charts  
```

#### **Ⅳ. 高級整合技巧**  

##### **1. 銀行API自動同步**  
1. **使用Zapier橋接**：  
   ```  
   銀行郵件通知 → Zapier解析 → 寫入Google Sheets → Obsidian同步  
   ```  
2. **設定同步頻率**：  
   ```  
   /sync-bank 設定每小時自動同步  
   ```  

##### **2. 語音記帳快捷指令**  
1. **手機設定**：  
   ```  
   捷徑名稱: 「記帳到Obsidian」  
   動作:   
     - 語音輸入內容  
     - 呼叫Obsidian URI: obsidian://new?content={{內容}}  
   ```  
2. **格式範例**：  
   ```  
   "星巴克大杯拿鐵 35元 信用卡A" → 自動解析為交易記錄  
   ```  

##### **3. 照片OCR整合**  
1. **配置Google Vision API**：  
   ```javascript  
   const text = await visionClient.textDetection(tp.file.path);  
   const amount = text.match(/NT\$?(\d+)/)?.[1];  
   ```  
2. **安全協議**：  
   ```  
   啟用端到端加密 → 所有圖片先經加密再傳輸  
   ```
