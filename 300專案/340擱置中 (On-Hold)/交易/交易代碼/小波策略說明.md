# 小波策略交易系統說明

## 策略概述

小波策略是一個基於趨勢轉折點識別的自動化交易系統，利用小波指標識別市場的轉折點，並在趨勢轉換時進行交易。

## 核心交易邏輯

### 多單進場條件
1. **向下趨勢線結束**：小波指標識別出一個下跌趨勢
2. **趨勢轉折**：價格從下跌轉為上漲，形成新的上漲趨勢線
3. **突破確認**：當前價格超過前一個向下趨勢線的最高點
4. **進場執行**：滿足以上條件時自動開多單

### 空單進場條件
1. **向上趨勢線結束**：小波指標識別出一個上漲趨勢
2. **趨勢轉折**：價格從上漲轉為下跌，形成新的下跌趨勢線
3. **跌破確認**：當前價格跌破前一個向上趨勢線的最低點
4. **進場執行**：滿足以上條件時自動開空單

### 止損設定
- **多單止損**：設在前一個上漲趨勢線的最低點
- **空單止損**：設在前一個下跌趨勢線的最高點

## 參數設定

### 交易參數
- `LotSize`：預設交易手數（0.1）
- `MagicNumber`：魔術數字，用於識別策略開的倉位（12345）
- `Slippage`：允許的滑點（3點）

### 風險管理
- `MaxRiskPercent`：單筆交易最大風險百分比（2.0%）
- `StopLossMultiplier`：止損倍數調整（1.0）

### 小波指標參數
- `ShowSWING`：是否顯示趨勢線（true）
- `WidthSWING`：趨勢線寬度（1）
- `UPLineStyle`：上漲線條樣式（實線）
- `DOWNLineStyle`：下跌線條樣式（實線）
- `UPLineColor`：上漲線條顏色（紅色）
- `DOWNLineColor`：下跌線條顏色（綠色）

## 風險管理機制

### 資金管理
- 根據帳戶餘額和風險百分比動態計算倉位大小
- 確保單筆交易風險不超過設定的最大風險百分比
- 自動調整手數以符合經紀商的最小/最大手數限制

### 倉位管理
- 使用魔術數字區分不同策略的倉位
- 防止重複開倉
- 提供倉位管理框架，可擴展移動止損等功能

## 使用指南

### 安裝步驟
1. 將 `小波策略.mq5` 複製到 MetaTrader 5 的 Experts 資料夾
2. 重新啟動 MetaTrader 5
3. 在導航器中找到 "小波策略" EA
4. 拖拽到圖表上

### 參數調整建議
1. **風險控制**：建議將 `MaxRiskPercent` 設為 1-2%
2. **手數設定**：根據帳戶大小調整 `LotSize`
3. **滑點設定**：根據市場波動性調整 `Slippage`

### 回測建議
- 建議在至少1年的歷史數據上進行回測
- 測試不同市場條件下的表現
- 關注最大回撤和勝率指標

## 策略優勢

1. **趨勢跟隨**：在趨勢轉折點進場，捕捉趨勢初期
2. **風險可控**：明確的止損點位和風險管理
3. **自動化執行**：減少人為情緒干擾
4. **視覺化**：趨勢線清晰顯示，便於分析

## 注意事項

1. **市場適應性**：策略在不同市場條件下表現可能不同
2. **參數優化**：建議根據具體交易品種進行參數優化
3. **實盤測試**：建議先在模擬帳戶中充分測試
4. **風險警告**：外匯交易存在風險，請謹慎投資

## 技術實現

### 核心函數
- `UpdateSwingPoints()`：更新轉折點數據
- `CheckTradeSignals()`：檢查交易信號
- `OpenLongPosition()`：開多單
- `OpenShortPosition()`：開空單
- `CalculateLotSize()`：計算倉位大小

### 數據結構
```cpp
struct SwingPoint
{
    datetime time;    // 時間
    double   price;   // 價格
    int      type;    // 類型（1=上漲, 2=下跌）
};
```
