# canvas-visualizer 技能

此技能將「原子化清單」中的動態關係轉化為畫布上的視覺流動。

## 核心操作流程

### 1. 筆記實體化
- 讀取清單中的原子概念，於 `410原子化知識庫/` 建立/更新筆記。
- 正文包含清單中的「核心說明」。

### 2. 畫布佈署 (Text 節點)
- 建立 `text` 類型畫布節點。
- **內容格式**：
    ```text
    ## [[筆記名稱]]
    [核心說明文字]
    ```

### 3. 動態互動連線 (敘事標籤)
- **單一實例原則 (全域去重)**：
    - 嚴格檢索 `410原子化知識庫` 與畫布既有 `id`。
    - **標題校準**：若清單中的標題與現有知識庫檔案名稱相似度高度重合（例如僅多出「一人公司」等修飾詞），**必須強制合併**至既有節點，不得建立新筆記。
- **敘事標籤處理**：
    - 讀取清單中的「AI 互動預測」段落。
    - 將詳細的互動描述（如：能量流轉、因果支撐）填入 `edge` 的 `label` 中。
    - **視覺校準**：由於標籤文字可能較長，確保連線路徑有足夠空間。

## 物件屬性細節
- `nodes` -> `type: "text"`
- `edges` -> `label`: "[清單中的詳細互動敘事]"
