# msitarzewski/agency-agents: 觸手可及的完整 AI 代理商 - 從前端奇才到 Reddit 社群忍者，從創意注入者到現實查核員。每個代理人都是具有個性、流程和成熟交付成果的專業專家。

> **觸手可及的完整 AI 代理商** - 從前端奇才到 Reddit 社群忍者，從創意注入者到現實查核員。每個代理人都是具有個性、流程和證明過交付成果的專業專家。

---

源於 Reddit 貼文並經過數月的迭代，**The Agency** 是一個不斷增長的精雕細琢 AI 代理人個性集合。每個代理人皆具備：

- **🎯 專業化**：其領域的深厚專業知識（非通用提示詞模板）
- **🧠 個性驅動**：獨特的聲音、溝通風格和方法
- **📋 以交付為導向**：真實的代碼、流程和可衡量的結果
- **✅ 生產就緒**：經過實戰測試的工作流和成功指標

**想像成**：組建你的夢幻團隊，除了他們是永不睡眠、永不抱怨且始終交付成果的 AI 專家。

---

```bash
# 將代理人複製到你的 Claude Code 目錄
cp -r agency-agents/* ~/.claude/agents/

# 現在可以在你的 Claude Code 會話中激活任何代理人：
# "Hey Claude, activate Frontend Developer mode and help me build a React component"
```

每個代理人文件包含：

- 身份與個性特徵
- 核心任務與工作流
- 帶有代碼範例的技術交付物
- 成功指標與溝通風格

瀏覽下方的代理人並複製/改編你需要的！

## 🎯 為什麼選擇這些代理人？

每個代理人的設計包含：
