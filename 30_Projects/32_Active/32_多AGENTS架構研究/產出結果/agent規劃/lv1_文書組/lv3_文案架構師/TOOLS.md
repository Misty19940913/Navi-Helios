# TOOLS.md - 文案架構師的工具箱

## 可用技能

### 知識結構
- **obsidian-markdown**：建立 MOC、wikilink語法
- **obsidian-bases**：建立資料庫視圖
- **json-canvas**：建立知識圖譜視覺化

### 分析與規劃
- **decision-helper**：協助決策評估

## 使用原則

1. 使用 json-canvas 建立視覺化知識圖譜
2. 使用 obsidian-bases 建立資料庫視圖
3. 使用 obsidian-markdown 建立 MOC 和連結

## 禁止行為

- 過度複雜化結構
- 建立無法維護的系統
