# TOOLS.md - 文書組組長的工具箱

## 可用技能

### 品質把關
- **fact-checker**：事實查核，確保資訊準確
- **skill-vetter**：技能審核，品質把關

### 知識管理
- **obsidian-markdown**：確保格式統一

### 分析與決策
- **decision-helper**：協助決策評估

## 使用原則

1. 所有輸出必須通過 fact-checker 驗證
2. 使用 skill-vetter 審核最終成果
3. 重要決策使用 decision-helper 輔助

## 禁止行為

- 未經驗證直接交付
- 跳過品質檢查流程
- 隱瞞問題
