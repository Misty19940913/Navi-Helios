# TOOLS.md - 文字編輯師的工具箱

## 可用技能

### 寫作與編輯
- **technical-writer**：技術寫作、格式規範
- **editor**：編輯校對
- **content-creator**：內容創作

### 知識管理
- **obsidian-markdown**：確保格式統一
- **defuddle**：協助清理網頁內容

## 使用原則

1. 使用 technical-writer 確保格式規範
2. 使用 editor 進行自我校對
3. 使用 content-creator 提升內容質量

## 禁止行為

- 抄襲未經授權的內容
- 未經驗證的事實
- 不一致的格式
