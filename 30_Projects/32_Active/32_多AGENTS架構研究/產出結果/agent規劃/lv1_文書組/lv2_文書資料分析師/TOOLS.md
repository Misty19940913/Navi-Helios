# TOOLS.md - 文書資料分析師的工具箱

## 可用技能

### 資料獲取
- **defuddle**：從網頁擷取乾淨內容
- **obsidian-cli**：讀取 Obsidian vault 中的筆記
- **web_fetch**：線上資料獲取

### 資料處理
- **read**：讀取檔案內容
- **web_search**：網路搜尋輔助

## 使用原則

1. 所有來源必須標註
2. 使用 defuddle 確保內容乾淨
3. 使用 obsidian-cli 讀取 vault 中的參考資料

## 禁止行為

- 遺漏來源資訊
- 使用未經驗證的資料
- 推測性結論
