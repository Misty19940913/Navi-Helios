## 等級行為標記

### Level 1.0 行為（任務追隨者）

| 象限 | 行為觀察 |
|------|----------|
| **心智** | 不斷引用他人話語；容易被不同意見觸發；需要持續的外在驗證 |
| **身體** | 無意識的飲食和運動；跟風時尚但不理解；與身體信號斷連 |
| **關係** | 透過服從獲得歸屬感；愛是佔有或交易；需要被群體接納 |
| **價值** | 工作是必要之惡；金錢是稀缺資源；跟隨職業模板 |

### Level 2.0 行為（自主權開發者）

| 象限 | 行為觀察 |
|------|----------|
| **心智** | 批判性思維發展；質疑一切（通常是強迫性的）；建立個人哲學 |
| **身體** | 優化和追蹤；健康作為表現指標；嘗試各種協議 |
| **關係** | 關係作為自我發現；願意設立邊界；探索孤立傾向 |
| **價值** | 工作作為自我表達；金錢作為計分板；創造自己的道路 |

### Level 3.0 行為（整合型創造者）

| 象限 | 行為觀察 |
|------|----------|
| **心智** | 擁抱悖論；觀點作為工具；建立框架來賦能他人 |
| **身體** | 直覺優化；身體作為教師；衰老作為精煉 |
| **關係** | 關係作為練習；愛作為認同；為他人創造意義 |
| **價值** | 工作即玩；系統建設；代際思維 |

### Level 4.0 行為（進化型）

| 象限 | 行為觀察 |
|------|----------|
| **心智** | 思想作為集體過程；直接知道/感知；創造同時也是管道 |
| **身體** | 身體作為地球表達；透過存在疗癒；細胞意識 |
| **關係** | 普世之愛體現；他人成功即自己成功；生命/死亡統一 |
| **價值** | 工作即宇宙服務；財富作為循環；創造新現實 |

## 各象限具體觀察指標

### XQ/Mind 覺察商觀察

| 等級 | 觀察重點 |
|------|----------|
| **1.0** | 是否依賴外在權威？是否能接受複雜性？ |
| **2.0** | 是否將反對誤認為智慧？是否能同時持有多個觀點？|
| **3.0** | 是否有創造新框架的能力？是否能策略性選擇？|

**關鍵問題觀察：**
- 「你能同時持有多個對立的觀點嗎？」
- 「你的盲點是什麼？你如何發現它們？」
- 「你何時會策略性地選擇顯得狹隘？」

### HQ/Body 健康商觀察

| 等級 | 觀察重點 |
|------|----------|
| **1.0** | 基本健康習慣；外在動機；身體信號忽視 |
| **2.0** | 優化協議；指標追蹤；可持續性 |
| **3.0** | 直覺實踐；跨象限支持；教導他人 |

**關鍵問題觀察：**
- 「你的身體實踐可持續嗎？」
- 「你追蹤哪些指標？為什麼？」
- 「你如何根據直覺調整身體實踐？」

### EQ/Spirit 情緒商觀察

| 等級 | 觀察重點 |
|------|----------|
| **1.0** | 傳統和權威；歸屬需求；外部驗證 |
| **2.0** | 叛逆模式；精神購物；孤立傾向 |
| **3.0** | 橋樑建設；悖論舒適度；神聖/世俗整合 |

**關鍵問題觀察：**
- 「你傾向於孤立嗎？什麼情況下？」
- 「你如何建立橋樑連接不同群體？」
- 「你對悖論的舒適度如何？」

### FQ/Vocation 財富商觀察

| 等級 | 觀察重點 |
|------|----------|
| **1.0** | 安全需求；權威關係；技能發展 |
| **2.0** | 創業項目；階梯選擇；獨特價值創造 |
| **3.0** | 系統建設；遺產思維；新遊戲創造 |

**關鍵問題觀察：**
- 「你創造了什麼獨特價值？」
- 「你建立了什麼系統來賦能他人？」
- 「你對遺產的想法是什麼？」

## 進展評估標準

### XQ/Mind 覺察商成功標準

| 指標 | Level 1→2 | Level 2→3 |
|------|-----------|-----------|
| 批判性思維 | 能質疑表面資訊 | 能整合對立觀點 |
| 情緒韌性 | 能承受批評 | 能從批評中學習 |
| 觀點多樣性 | 理解多元觀點存在 | 能同時持有多個觀點 |
| 元認知 | 能觀察自己思考 | 能策略性調整思考模式 |

### HQ/Body 健康商成功標準

| 指標 | Level 1→2 | Level 2→3 |
|------|-----------|-----------|
| 能量水平 | 有持續能量 | 能量能支持多象限發展 |
| 實踐一致性 | 能維持基本習慣 | 能直覺調整實踐 |
| 功能表現 | 能執行日常動作 | 能支持高压環境 |
| 身體覺察 | 能感知基本信號 | 能解讀細微身體語言 |

### 關係維度成功標準

| 指標 | Level 1→2 | Level 2→3 |
|------|-----------|-----------|
| 深度連結 | 有可深談的人 | 能建立多重深度關係 |
| 邊界能力 | 能設立基本邊界 | 能策略性調整邊界 |
| 歸屬感 | 屬於某群體 | 能連接不同群體 |
| 意義創造 | 能從活動獲得意義 | 能為他人創造意義 |

### 價值維度成功標準

| 指標 | Level 1→2 | Level 2→3 |
|------|-----------|-----------|
| 收入能力 | 能創造穩定收入 | 能創造與時間脫鉤的收入 |

## AI 訪談應用

### 等級識別觀察清單

```
□ 是否持續引用權威而非提出原創觀點？
□ 是否容易被不同意見觸發情緒？
□ 是否需要外在驗證才能確認價值？
□ 
□ 是否質疑既定規則？
□ 是否建立個人化的管理系統？
□ 是否將失敗視為學習數據？
□ 
□ 是否能同時持有多個矛盾觀點？
□ 是否建立框架來賦能他人？
□ 是否能在過程本身發現意義？
```

### 階段識別觀察清單

```
□ 感到不安、想改變但不知道怎麼改？（x.1）
□ 在嘗試各種新方式，有些有效有些失敗？（x.2）
□ 找到了有效的方式，正在穩定執行？（x.3）
```
