# 046_好時光・壞時光日誌

> Stage 1 覺察階段的核心追蹤工具
> 產出時間：2026-04-27
> 整合：設計人生 好時光日誌 × Arena 六商系統 × X.1 失調識別

---

## 融合說明

本工具將設計人生的好時光/壞時光日誌整合至 Arena 六商系統，用於 Stage 1 覺察階段。

核心用途：透過規律記錄，識別哪些活動消耗/滋養你的六商能量，發現 x.1 失調信號（精力持續低迷、能量傾斜），為六商現況評估提供第一手數據。

---

## 用途

1. **覺察能量流向** — 客觀看見時間與精力花費的模式
2. **識別 x.1 失調信號** — 哪些商數長期被忽略或過度消耗
3. **為現況評估提供素材** — Stage 1 的自我描述不是憑空而來
4. **發現「偽裝成功」** — 某件事看起來有成果，但消耗了哪些商數

━━━━━━━━━━━━━━━━━━━━━━━━━

14:00-15:30 健身                  ★★★★☆       HQ
19:00-20:00 與孩子玩遊戲          ★★★★★       EQ、PQ
━━━━━━━━━━━━━━━━━━━━━━━━━

【壞時光】讓我感到精力損耗的活動
━━━━━━━━━━━━━━━━━━━━━━━━━
時間段      活動                  感覺（1-5）| 消耗的商數
09:00-10:00 處理重複性行政        ★☆☆☆☆       EQ（低 EQ）
14:00-16:00 沒有目標的會議        ★☆☆☆☆       XQ、PQ
21:00-22:00 睡前滑手機             ★★☆☆☆       HQ、XQ
━━━━━━━━━━━━━━━━━━━━━━━━━

### 感覺評分說明（1-5）

```
★★★★★ = 完全沉浸，精力充沛，事後感到「充電」
★★★★☆ = 良好狀態，事後比事前更有精力
★★★☆☆ = 普通，沒有特別感覺
★★☆☆☆ = 有點疲憊，事後需要恢復
★☆☆☆☆ = 嚴重損耗，事後感到「被掏空」
```

## 每週分析（每週日，15 分鐘）

### 模式識別

```
本週好時光模式：
┌─────────────────────────────────────────────┐
│ 1. FQ/XQ 驅動的活動（創造性工作）最多        │
│ 2. 午後 HQ 活動 明顯提升午後 EQ             │
│ 3. EQ 滋養主要來自深度一對一對話            │
└─────────────────────────────────────────────┘

本週壞時光模式：
┌─────────────────────────────────────────────┐
│ 1. 上午處理行政 = 下午精力全面崩潰           │
│ 2. 超過 2 小時的的被動會議 = XQ 損耗        │
│ 3. 睡前使用螢幕 = HQ 修復受阻               │
└─────────────────────────────────────────────┘
```

### 六商能量平衡診斷

```
每週結束，計算各商數的「好時光」vs「壞時光」比例：

商數  │ 好時光次數 │ 壞時光次數 │ 比率  │ 診斷
------|-----------|-----------|-------|------
FQ    │           │           │       │ 需要確认：是被動消耗還是主動投入
HQ    │           │           │       │ 身體修復時間是否足夠
EQ    │           │           │       │ 關係投資是否失衡
PQ    │           │           │       │ 玩樂時間是否完全被剝奪
AQ    │           │           │       │ 是否在持續高壓無恢復
XQ    │           │           │       │ 有多少時間在「自動駕駛」
```

## 與六商 x.1 失調的對應

本工具直接識別 030_六商階段定義 中描述的 x.1 失調信號：

```
【x.1 失調的客觀指標】來自好時光/壞時光日誌

FQ 1.1 失調信號：
□ 記帳/理財幾乎從未出現在好時光中
□ 「處理財務」總是出現在壞時光（逃避）
□ 沒有任何與創收相關的好時光體驗

HQ 1.1 失調信號：
□ 運動/睡眠/飲食幾乎從未出現在好時光
□ 疲憊、精力低迷是常態（不是特殊情況）
□ 沒有任何身體修復的好時光

EQ 1.1 失調信號：
□ 幾乎沒有深度對話出現在好時光中
□ 關係總是被「需要處理」而非「滋養體驗」
□ 情緒起伏完全由外部事件決定

PQ 1.1 失調信號：
□ 工作/責任幾乎佔據了所有清醒時間
□ 玩樂時間 = 0 或幾乎為零
□ 「玩」總是伴隨罪惡感

AQ 1.1 失調信號：
□ 逆境/壓力是時間記錄的唯一主題
□ 沒有任何「恢復後變強」的好時光體驗
□ 失敗/挫折後沒有完整恢復的記錄

XQ 1.1 失調信號：
□ 「自動駕駛」是常態
□ 幾乎沒有反思/覺察出現在好時光中
□ 所有事情都是條件反射
```

## 常見問題

**Q：忘記記錄怎麼辦？**
A：不要補寫——補寫會扭曲數據。直接從當下重新開始，記住「偶爾漏記」本身也是數據（說明某件事讓你完全忘了時間）。

**Q：好時光和壞時光可以同時出現在同一個活動中嗎？**
A：可以。例如「演講」可能同時有 FQ 好時光（收入/曝光）和 EQ 壞時光（前期焦慮）。紀錄兩者，這是重要的「複雜性」數據。

**Q：什麼都感覺普通怎麼辦？**
A：★★★☆☆ 也是數據。如果一個活動連續兩週都是★★★☆☆，說明它沒有在滋養也沒有在消耗——可能是「舒適區」的信號。
