# 042_行為標記

> Arena 的行為標記系統 — 源自 H3.0
> 產出時間：2026-04-01

---

## 核心定義

行為標記是判斷個人意識發展位階的客觀「診斷標籤」。透過觀察日常反覆出現的行為模式，而非口頭宣稱的理念，能真實區分處於 1.0、2.0 還是 3.0。

---

## 三等級行為標記

### Level 1.0 — 任務隨行者

| 行為標記 | 描述 |
|----------|------|
| 引用缺乏原創 | 不斷引用他人的話，缺乏原始洞見 |
| 易被觸發 | 容易被不同意見引發情緒 |
| 尋求外界認可 | 尋求外界驗證，恐懼不被認同 |
| 盲目跟隨趨勢 | 在不確定性面前感到恐懼 |
| 二元思維 | 非黑即白的思考模式 |

### Level 2.0 — 自主權開發者

| 行為標記 | 描述 |
|----------|------|
| 質疑一切 | 質疑既定規則，追求最佳化 |
| 刻意實驗 | 建立個人化的管理系統 |
| 自主規劃 | 願意承擔選擇後的責任 |
| 接受失敗 | 將失敗視為學習數據 |
| 追求差異化 | 努力與眾不同 |

### Level 3.0 — 整合型創作者

| 行為標記 | 描述 |
|----------|------|
| 擁抱悖論 | 不急於寻求唯一解答，能同時持有多個矛盾觀點 |
| 建立框架 | 建立框架來賦能他人 |
| 整合對立 | 能將對立的事物整合（如：紀律=自由） |
| 意義來自過程 | 在發展過程本身發現意義 |
| 策略性狹隘 | 有意識地選擇何時顯得狹隘 |

---

## AI 觀察重點

| 等級 | 重點觀察 |
|------|----------|
| 1.0 | 是否依賴外部權威？是否能接受複雜性？ |
| 2.0 | 是否將反對誤認為智慧？是否能同時持有多個觀點？ |
| 3.0 | 是否有創造新框架的能力？是否能策略性選擇？ |

---

## 與虛假轉化的區分

| 區分維度 | 真實 3.0 | 虛假 3.0 |
|----------|----------|----------|
| 行為一致性 | 壓力下仍保持整合 | 壓力下退行至早期模式 |
| 知識 vs 實踐 | 理論與實踐一致 | 理論華麗但實踐空洞 |
| 對他人的態度 | 賦能他人 | 可能操控或遠離 |
