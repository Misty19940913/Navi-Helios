## Step 1: 六商現況評估

### 六商核心問題

| 商數 | 核心問題 | 快速自問 |
|------|----------|----------|
| **FQ（財富智商）** | 我如何創造與交付價值？ | 我的工作/創造有意義嗎？感到滿足還是空洞？ |
| **HQ（健康智商）** | 我的能量狀態如何？ | 我的身體感覺怎麼樣？精力充沛還是疲憊？ |
| **EQ（情緒智商）** | 我的情緒管理與關係如何？ | 我能覺察並管理情緒嗎？與他人的連結深度如何？ |
| **PQ（玩商）** | 我如何體驗樂趣與創造力？ | 我在生活中有玩心嗎？會因為單純的快樂而行動嗎？ |
| **AQ（逆境智商）** | 我如何面對挑戰與挫折？ | 遇到逆境時，我的恢復力如何？能從失敗中學習嗎？ |
| **XQ（覺察商）** | 我的認知與成長如何？ | 我能覺察自己的情緒和思維模式嗎？有成長感嗎？ |

## Step 2: 等級識別

### 等級定義速查

| 等級 | 名稱 | 一句話特徵 |
|------|------|------------|
| **1.0** | 順從者 | 服從外部規則，等待指令 |
| **2.0** | 個人主義者 | 追求自我目標，差異化 |
| **3.0** | 整合者 | 多視角整合，創造新遊戲 |

### 各商數等級特徵

#### FQ 等級

| 等級 | 行為描述 |
|------|----------|
| **1.0** | 價值由外部（薪資、地位、認可）定義；不清楚獨特價值 |
| **2.0** | 發展獨特價值主張；追求差異化；工作與意義感掛鉤 |
| **3.0** | 價值創造成為為他人賦能的工具；關注創造的傳承與系統化 |

#### HQ 等級

| 等級 | 行為描述 |
|------|----------|
| **1.0** | 身體被視為工具；飲食/運動由外界指令決定；忽略身體警訊 |
| **2.0** | 主動管理健康；建立自己的健身/飲食系統；追求體態差異化 |
| **3.0** | 身體是內在智慧的訊息來源；能覺察並回應身體細微訊號 |

#### EQ 等級

| 等級 | 行為描述 |
|------|----------|
| **1.0** | 關係由社會角色決定；為維持和諧放棄自我；孤獨感歸咎外部 |
| **2.0** | 選擇性建立關係；追求深度連結；願意設立邊界 |
| **3.0** | 能同時是群體一部分又是獨立個體；創造性地建設關係 |

#### PQ 等級

| 等級 | 行為描述 |
|------|----------|
| **1.0** | 玩樂被視為浪費時間；休閒等同於放空或強迫性娛樂 |
| **2.0** | 刻意安排玩樂時間；能從玩中學習與創造 |
| **3.0** | 玩心融入生活與工作；用玩樂驅動創新與連結 |

#### AQ 等級

| 等級 | 行為描述 |
|------|----------|
| **1.0** | 逆境中被擊敗；歸咎外部環境；無法從失敗中學習 |
| **2.0** | 逆境中恢復；找到自己的 Coping 機制；尋找逆境意義 |
| **3.0** | 逆境中成長；用自身經歷賦能他人；主動尋找突破性挑戰 |

#### XQ 等級

| 等級 | 行為描述 |
|------|----------|
| **1.0** | 接受既定信念；認知等同於權威；不習慣質疑 |
| **2.0** | 建構自己的認知框架；批判性分析資訊；追求深度見解 |
| **3.0** | 同時持有多個矛盾觀點；根據情境靈活調用不同心智模型 |

### 等級識別問題

| 商數 | 識別問題 |
|------|----------|
| FQ | 「如果今天沒有報酬，你還會做現在的工作嗎？」 |
| HQ | 「你運動/飲食是因為自己想這樣做，還是因為別人說你應該？」 |
| EQ | 「你維持關係是因為期待，還是因為真正的連結？」 |
| PQ | 「你會因為單純好玩而做一件事嗎？還是做任何事都要有目的？」 |
| AQ | 「當事情不如預期時，你通常如何回應？」 |
| XQ | 「當你的想法與權威衝突時，你會堅持自己還是服從？」 |

## Step 3: 階段定位

### 階段定義速查

| 階段 | 名稱 | 一句話特徵 |
|------|------|------------|
| **x.1** | 失調 | 舊模式失效，感到不安 |
| **x.2** | 不確定 | 踏入未知，實驗新方法 |
| **x.3** | 發現 | 找到有效資源，穩定整合 |

### 各商數階段特徵

#### x.1 失調期（各商數通用）

| 訊號 | 描述 |
|------|------|
| 身心症狀 | 焦躁不安、無聊、輕度沮喪 |
| 行為表現 | 拖延、對現狀不滿 |
| 認知狀態 | 知道要改變但不清楚具體方向 |
| 典型問題 | 「我只知道現在的方式不管用了」 |

#### x.2 不確定期（各商數通用）

| 訊號 | 描述 |
|------|------|
| 身心症狀 | 脆弱但成長導向；實驗興奮感 |
| 行為表現 | 嘗試新方法；接受失敗作為學習 |
| 認知狀態 | 開放探索；收集資訊和經驗 |
| 典型問題 | 「我在嘗試各種方式，不知道哪個有效」 |

#### x.3 發現期（各商數通用）

| 訊號 | 描述 |
|------|------|
| 身心症狀 | 清晰的行動方向；享受進步 |
| 行為表現 | 穩定的進步；整合新能力 |
| 認知狀態 | 知道自己擅長什麼；為下一階段做準備 |
| 典型問題 | 「我找到了有效的方式，正在鞏固」 |

### 階段定位問題

「過去一個月，你在這個商數上是：」
- 「感到不安、想改變但不知道怎麼改」→ x.1
- 「在嘗試各種新方式，有些有效有些失敗」→ x.2
- 「找到了有效的方式，正在穩定執行」→ x.3

## Step 4: 生活方式原型識別

### 六種原型速查

| 原型 | 特徵 | 失衡訊號 |
|------|------|----------|
| 工作狂 | FQ 主導，其他商數被忽略 | 過勞、burnout |
| 探索者 | XQ/EQ 過度，FQ/HQ 虛弱 | 空有想法無法落地 |
| 優化者 | HQ/XQ 過度，EQ/FQ 空洞 | 高效但空虛 |
| 漂泊者 | 沒有商數深入發展 | 一事無成感 |
| 專家 | 單一商數極度發展，其他落後 | 偏科嚴重 |
| 整合者 | 所有商數相互支持 | （這是目標狀態） |

### 原型識別問題

「以下敘述哪個最符合你？」
- 「我總是把工作放在第一位，即使犧牲健康和關係」→ 工作狂
- 「我一直在學習和探索，但很少把想法付諸行動」→ 探索者
- 「我優化了身體和頭腦，但感到孤獨和沒有意義」→ 優化者
- 「我什麼都想做，但都做不深，感覺在原地打轉」→ 漂泊者
- 「我在某個領域很專業，但其他方面很空白」→ 專家
- 「我的各商數相互支持，生活節奏自然流動」→ 整合者

## Step 5: 錯誤轉變訊號檢測

### 六商錯誤轉變指標

| 商數 | 錯誤轉變訊號 | 真實成長對照 |
|------|-------------|--------------|
| FQ | 教學缺乏實質成果 | 價值創造有可見的輸出 |
| HQ | 完美的健身自拍，但功能性很差 | 實際能力提升，能量充沛 |
| EQ | 精神上迴避真實情感 | 能深度連結，被理解 |
| PQ | 把玩樂當成 productivity 工具 | 單純為了快樂而玩 |
| AQ | 用逆境故事建立個人品牌 | 真正從逆境中成長 |
| XQ | 引用概念，但無法實際應用 | 能用不同框架解決實際問題 |

### 檢測問題

「過去一個月，你是否有以下情況？」
- 「看起來很忙但沒有實質成果」→ FQ 錯誤轉變
- 「健身是為了發朋友圈，不是為了自己的能量」→ HQ 錯誤轉變
- 「參加很多社交活動但沒有真正的連結」→ EQ 錯誤轉變
- 「把玩樂當成另一種工作或證明」→ PQ 錯誤轉變
- 「用逆境故事來建立形象而非真正成長」→ AQ 錯誤轉變
- 「讀了很多書但生活沒有任何改變」→ XQ 錯誤轉變

### 常見傳導模式

## 多商數同時 x.1 整合策略

### 問題定義

當六商評估結果顯示 3+ 個商數同時處於 x.1 階段，這不是「同時處理多個問題」——這是系統性失調的訊號。

**為什麼這會發生？**
- 單一創傷/危機事件同時打擊多個商數
- 長期低 EQ 破壞了所有關係，最終影響所有商數
- FQ 價值觀長期空洞導致全面慢性失調

### 整合策略：核心切入點原則

```
策略 1：找「最大公約數」切入點
============================================
如果多個失調商數共享同一個根因，
只處理那個根因，其他商數會自動緩解。

常見根因模式：
  FQ+HQ 失調 → 財務焦慮 → 解決方案：緊急建立 FQ 安全邊界
  EQ+XQ 失調 → 長期情緒壓抑 → 解決方案：先情緒釋放再覺察
  HQ+EQ 失調 → 長期過勞 → 解決方案：先休息再修復關係
```

### 識別模式

```
【失調商數組合】 → 【最可能根因】 → 【切入優先順序】

FQ+HQ 失調：財務焦慮驅動的身體損耗 → 先 FQ 安全感，HQ 自然修復
HQ+EQ 失調：過勞導致的身心崩潰 → 先 HQ 休息，EQ 自然修復
EQ+XQ 失調：情緒未處理導致的認知失調 → 先 EQ 情緒釋放，XQ 自然修復
FQ+EQ 失調：價值衝突導致的關係緊張 → 先 EQ 修復，FQ 自然修復
PQ+XQ 失調：強迫性努力掩蓋的存在危機 → 先 XQ 覺察，PQ 自然修復
3+ 個商數同時失調：系統性崩潰 → 先建立最小安全系統
```

### 整合行動模板

```
【系統性失調整合計劃】

第一步：建立最小安全系統（48 小時內）
□ 睡眠：至少連續 6 小時不被打斷的睡眠
□ 飲食：沒有財務壓力的基本飲食
□ 安全：確保沒有人身安全威脅

第二步：識別核心切入商數
□ 如果 [X] 商數處理了，[Y] 和 [Z] 商數會自動緩解
□ 切入商數：[      ]
□ 切入原因：[      ]

第三步：單商數優先行動（2 週內）
□ [切入商數] 的最小可行行動：[        ]
□ 目標：從 x.1 移動到 x.2

第四步：監控連鎖反應
□ [Y] 商數是否有緩解跡象？
□ [Z] 商數是否有緩解跡象？
□ 連鎖反應是否如預期？
□ 如果否：重新評估切入假設
```

### 警告信號

```
□ 3+ 商數同時失調且用戶否認問題存在 → 需要專業介入
□ FQ 1.1 + HQ 1.1 同時出現 → 財務危機+健康危機，優先處理人身安全
□ EQ 1.1 + XQ 1.1 同時出現 → 建議先尋求心理專業支持
□ 所有商數 1.1 → 系統完全崩潰，可能需要先離開 Arena 框架
```

## 整合缺口評估：知識/經驗/技能錯位檢測

> 本章改編自 K-HUMAN3-049_多層次跨越.md
> 核心問題：「為什麼我懂了，卻做不到？」

### 為什麼這是 Arena Stage 1 的必檢項目

Arena 用戶常見的困惑：

> 「我在某個商數上自我評估是 2.0，但為什麼生活中還是做不到？」

049 的核心洞見：**知識、經驗、技能可能各自處在不同等級**。這種「錯位」會造成虛假的成長感——你以為你進化了，但行為沒變。

### 三種不一致狀態

| 維度 | 等級 | 說明 | Arena 例子 |
|------|------|------|-----------|
| **知識** | 可達 2.0 | 能理解並談論高層次概念 | 能解釋「什麼是 FQ 2.0」|
| **經驗** | 仍處 1.0-1.5 | 沒有實際長期實踐過 | 從來沒有持續管理過財務 |
| **技能** | 仍處 1.0 | 無法直覺應用，碰到實際問題需從頭推理 | 碰到財務決策時還是焦慮 |

### 快速診斷：三個問題

在每個商數的現況評估後，問自己：

1. **知識**：「這個商數的等級，我**能解釋但做不到**的是什麼？」
2. **經驗**：「這個商數，我**持續實踐超過六個月**的是什麼？」
3. **技能**：「這個商數，我**不需要刻意努力就能做好**的是什麼？」

三個答案如果不在同一等級，代表存在**整合缺口**。

### Arena 常見整合缺口模式

| 商數 | 典型缺口 | 脫離路徑 |
|------|---------|---------|
| **FQ** | 知道如何投資，但從來沒有真正執行過 | 增加刻意練習時間，建立最小投資習慣 |
| **HQ** | 知道如何照顧身體，但情緒性進食/過勞 | 增加反饋循環（記錄睡眠/飲食數據）|
| **EQ** | 知道情緒管理，但壓力下還是爆發 | 增加覺察頻率，用 046 [[046_好時光壞時光日誌]] 追蹤觸發點 |
| **PQ** | 知道要休息，但休息時有罪惡感 | 把玩樂寫進日程，如同工作任務 |
| **AQ** | 知道韌性的重要性，但實際失敗時立刻崩潰 | 增加「失敗實驗」頻率，降低失敗情緒代價 |
| **XQ** | 知道後設認知，但遇到問題還是自動反應 | 增加每日反思寫作頻率 |

### 整合缺口與錯誤轉變的區別

| 區分維度 | 整合缺口 | 錯誤轉變 |
|----------|---------|---------|
| 本質 | 知識>經驗，真實的落後 | 假的成長，包裝出來的形象 |
| 診斷 | 三問法識別（知識/經驗/技能錯位）| [[050_錯誤轉變指標]] 識別（行為與宣稱不符）|
| 處理 | 增加實踐時間和反饋頻率 | 重新校準自我認知 |
| Stage 1 位置 | Step 1（六商現況評估）完成後追加檢測 | Step 5（錯誤轉變訊號檢測）|
