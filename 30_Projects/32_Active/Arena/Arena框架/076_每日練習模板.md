# 076_每日練習模板

> Arena 的每日練習系統 — 融合 HUMAN 3.0
> 產出時間：2026-04-02

---

## 融合說明

本檔案從 HUMAN 3.0 (003) Part V 提取每日練習模板，用於：
- Stage 1 訪談後的行動建議
- Stage 2-3 的30/90/180天策略具體練習
- AI 報告輸出的練習推薦

---

## 寫作練習（所有等級）

> 「開始寫作。自由人的標誌是他們在一生中做很多事情。這需要他們學習如何學習、如何思考、如何謀生。寫作滿足這三個條件。」

### 晨間寫作堆疊（30分鐘）

| 時間 | 內容 | 目的 |
|------|------|------|
| 5分鐘 | 反願景清晰（今天你拒絕什麼） | 聚焦當下 |
| 10分鐘 | 問題識別與解決方案設計 | 問題解決 |
| 10分鐘 | 價值創造（今天你能幫助誰）| 意義感 |
| 5分鐘 | 公開分享（發文、訊息、文章）| 責任感 |

## Level 1→2 過渡練習

### 自主發展堆疊（60分鐘）

| 10分鐘 | 反思/規劃（你的目標）| 即時反饋 |

| 10分鐘 | 明日準備 | 減少焦慮 |
| 10分鐘 | 放鬆練習 | 副交感恢復 |

## Level 3+ 維護練習

### Flow 堆疊（彈性）

- 自然跟隨能量
- 創造多於消費
- 透過存在教導
- 解決有趣的問題
- 建設賦能結構
- 玩無限遊戲

## 數位槓桿路徑（職業發展）

### Phase 1: 建立受眾（Level 2.0）

| 行動 | 頻率 |
|------|------|
| 每日寫作分享你的興趣 | 每日 |
| 記錄你的學習旅程 | 每日 |
| 記錄問題與解決方案 | 每日 |
| 互動他人內容 | 持續 |
| 在售賣前先提供價值 | 持續 |

### Phase 2: 創造產品（Level 2.5）

| 行動 | 說明 |
|------|------|
| 包裝你的解決方案 | 資訊產品起點 |
| 小群體測試 | 封閉測試 |
| 根據反饋迭代 | 快速修正 |
| 規模化有效的 | 專注勝於多樣 |

### Phase 3: 系統與規模（Level 3.0）

| 行動 | 目的 |
|------|------|
| 自動化重複任務 | 釋放時間 |
| 建立經常性收入 | 被動現金流 |
| 創造不倦怠 | 可持續性 |
| 賦能他人成功 | 擴展影響力 |
| 設計再生業務 | 長期價值 |
