## 階段調整系統

### 調整觸發分類

| 觸發類型 | 說明 | 調整幅度 |
|----------|------|----------|
| **例行調整** | 根據 PDCA 結果的小幅調整 | 微調 |
| **方向調整** | 發現策略方向問題 | 中幅調整 |
| **框架調整** | 目標或價值觀需要重新定義 | 大幅調整 |
| **循環觸發** | 觸發回到 Stage 1 重新開始 | 完整循環 |

---

## 觸發條件與調整策略

### 1. 例行調整

**觸發條件：**
- 每週 Checkpoint 發現目標未達成
- 執行中遇到可預期的阻礙
- 能量狀態持續低落

**調整策略：**
```
Step 1: 分析未達成原因
Step 2: 調整下週行動
Step 3: 更新資源分配
Step 4: 記錄學到的教訓
```

### 2. 方向調整

**觸發條件：**
- 連續 2-3 個月目標未達成
- 原型測試結果與預期嚴重不符
- 環境發生重大變化

**調整策略：**
```
Step 1: 回顧階段 2 的假設
Step 2: 分析差距的根本原因
Step 3: 重新設計路徑（回到 Stage 2）
Step 4: 更新行動計劃
```

### 3. 框架調整

**觸發條件：**
- 價值觀或人生觀發生重大轉變
- 外部環境根本性改變
- 發現當前目標與核心價值衝突

**調整策略：**
```
Step 1: 重新進行 Stage 1 現況評估
Step 2: 更新自我認知地圖
Step 3: 重新設定目標（回到 Stage 2）
Step 4: 重新規劃路徑（回到 Stage 3）
```

## 回 Stage 1 觸發判斷標準

### 觸發條件矩陣

| 條件類型 | 具體觸發條件 | 優先級 |
|----------|-------------|--------|
| **商數崩潰** | 任何商數持續 2 週以上在 2/10 以下 | 高 |
| **商數退化** | 任何商數等級下降超過 0.3 | 高 |
| **目標失效** | 原有目標已經完全失去意義 | 高 |
| **價值觀轉變** | 對工作/生活/關係的核心價值觀發生重大轉變 | 中 |
| **外部巨變** | 事業/家庭/健康發生根本性改變 | 高 |
| **成長停滯** | 連續 3 個月無任何商數進步 | 中 |
| **長期倦怠** | 持續 1 個月以上的 burnout 狀態 | 高 |

### 觸發判斷流程

```
Step 1: 檢視商數狀態
        ↓
Step 2: 檢視等級進化
        ↓
Step 3: 檢視目標相關性
        ↓
Step 4: 檢視價值觀
        ↓
Step 5: 做出判斷
```

### 觸發判斷問題清單

| 問題 | 是 | 否 |
|------|----|----|
| 任何商數連續 2 週低於 2/10？ | → 觸發 | → 繼續 |
| 任何商數等級下降超過 0.3？ | → 觸發 | → 繼續 |
| 原有目標已經完全失去意義？ | → 觸發 | → 繼續 |
| 價值觀發生重大轉變？ | → 考慮觸發 | → 繼續 |
| 外部環境發生根本性改變？ | → 觸發 | → 繼續 |
| 連續 3 個月無進步？ | → 考慮觸發 | → 繼續 |
| 持續 1 個月以上倦怠？ | → 觸發 | → 繼續 |

## 調整機制操作流程

### 小調整流程（例行調整）

```
觸發：每週 Checkpoint
    ↓
分析：找出未達成原因
    ↓
調整：修改下週行動
    ↓
執行：繼續 Stage 3
    ↓
記錄：更新 PDCA 日誌
```

### 中調整流程（方向調整）

```
觸發：連續 2-3 月未達成目標
    ↓
回顧：重新審視 Stage 2 的假設
    ↓
分析：找出策略問題
    ↓
重新設計：回到 Stage 2 重新規劃
    ↓
更新：修改 Stage 3 行動計劃
    ↓
執行：繼續執行新計劃
```

### 大調整流程（框架調整/循環觸發）

```
觸發：符合回 Stage 1 條件
    ↓
宣告：明確宣布重新開始
    ↓
Stage 1：重新進行現況評估
    ↓
Stage 2：重新設定目標
    ↓
Stage 3：重新規劃策略
    ↓
Stage 4：重新執行
    ↓
新循環開始
```
