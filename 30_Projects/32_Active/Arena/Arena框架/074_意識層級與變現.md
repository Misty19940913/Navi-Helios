# 074_意識層級與變現

> Arena 的意識層級與變現系統 — 融合商業模式畫布
> 產出時間：2026-04-01

---

## 融合說明

Arena 的意識層級與變現採用商業模式畫布的意識層級框架，用於 Stage 2 目標設定時理解目標客戶的購買準備度。

---

## 核心定義

不同的客戶處於不同的意識層級，對應不同的營銷策略。理解客戶的意識層級，才能設計有效的價值主張和變現策略。

---

## 五層意識階梯

| 層級 | 名稱 | 客戶狀態 | 營銷策略 |
|------|------|----------|----------|
| **1.0** | 無意識 | 不知道問題存在 | 教育，問題覺察 |
| **2.0** | 問題意識 | 知道痛苦但不知道原因 | 痛點闡述 |
| **3.0** | 解決方案意識 | 知道有辦法但不知道選項 | 選項比較 |
| **4.0** | 產品意識 | 知道你的方案但在猶豫 | 證明、案例、社會證明 |
| **5.0** | 最意識 | 完全理解並準備購買 | 行動呼籲、限時優惠 |

---

## 各層級客戶特徵

### 1.0 無意識

| 特徵 | 描述 |
|------|------|
| **狀態** | 不知道自己有問題 |
| **典型語言** | 「我很好啊」「這不是問題」 |
| **痛點挖掘** | 需要教育市場 |

### 2.0 問題意識

| 特徵 | 描述 |
|------|------|
| **狀態** | 知道有問題，但不知道原因是什麼 |
| **典型語言** | 「我最近狀況不太好」「不知道怎麼了」 |
| **痛點挖掘** | 幫助他們找到根本原因 |

### 3.0 解決方案意識

| 特徵 | 描述 |
|------|------|
| **狀態** | 知道有解決方案，但不知道哪個適合 |
| **典型語言** | 「有好多種方法，不知道哪個好」 |
| **痛點挖掘** | 幫助他們比較選項，凸顯差異化 |

### 4.0 產品意識

| 特徵 | 描述 |
|------|------|
| **狀態** | 知道你的產品，但還在猶豫 |
| **典型語言** | 「這個看起來不錯，但...」 |
| **痛點挖掘** | 提供證明、案例、社會證明 |

### 5.0 最意識

| 特徵 | 描述 |
|------|------|
| **狀態** | 完全理解價值，準備購買 |
| **典型語言** | 「多少錢？怎麼報名？」 |
| **痛點挖掘** | 移除最後的購買障礙 |

---

## Arena Stage 2 的應用

### 目標客戶意識評估

在 Stage 2 定義目標用戶時，需要評估：
1. 目標客戶處於哪個意識層級？
2. 他們需要什麼層級的內容和營銷策略？
3. 如何設計從當前層級到購買的漏斗？

### 漏斗設計

```
意識層級
    ↓
1.0（無意識）→ 教育內容
    ↓
2.0（問題意識）→ 痛點內容
    ↓
3.0（解決方案意識）→ 比較內容
    ↓
4.0（產品意識）→ 證明內容
    ↓
5.0（最意識）→ 行動呼籲
```

### 價值主張與意識對應

| 意識層級 | 價值主張重點 |
|----------|-------------|
| 1.0 | 讓他們看到問題的存在 |
| 2.0 | 讓他們了解問題的原因和影響 |
| 3.0 | 讓他們看到你的獨特解決方案 |
| 4.0 | 讓他們相信你的方案有效 |
| 5.0 | 讓他們輕鬆購買 |

## AI 訪談應用

### 識別客戶意識層級的問題

> 「當你想到這個問題時，你現在腦中已經有解決方案的想法了嗎？還是你還在了解這個問題階段？」

### 根據意識層級調整營銷

| 客戶回答 | 識別層級 | AI 建議策略 |
|---------|----------|-------------|
| 「我不知道這是個問題」 | 1.0 | 提供教育的內容 |
| 「我知道有問題，但不知道怎麼辦」 | 2.0 | 提供痛點分析內容 |
| 「我知道有方法，但不知道哪個好」 | 3.0 | 提供比較內容 |
| 「我知道你的方案，但還在想」 | 4.0 | 提供案例和社會證明 |
| 「我準備好了，現在就要」 | 5.0 | 移除最後障礙 |

## 與變現階梯的對應

| 變現階段 | 主要客戶意識 | 產品形態 |
|----------|-------------|----------|

| Stage 2 模式化 | 3.0-4.0 | 標準化課程/產品 |
| Stage 3 被動收入 | 4.0-5.0 | 訂閱/會員制 |
