# 105_資源盤點模板

> 融合來源：商業模式 × 設計人生 × HUMAN 3.0
> 產出時間：2026-04-01

---

## 融合說明

| 來源 | 提供 |
|------|------|
| 商業模式 | 關鍵資源、關鍵合作、成本結構 |
| 設計人生 | 支持者、教練、工具資源 |

## 資源盤點總流程

```
Step 1: 個人資源清點
```

## Step 2: 支持系統識別

### 四種支持者類型

| 類型 | 說明 | 範例 |
|------|------|------|
| **同行者** | 陪你走過高山低谷的人 | 伴侶、密友 |
| **導師/教練** | 提供方向性指引的人 | 人生教練、職業導師 |
| **專家** | 填補特定知識/技能缺口 | 律師、會計師 |
| **社群** | 歸屬感與持續支持 | 學習社群、同業團體 |

## Step 3: 合作夥伴網絡

### 合作夥伴類型

| 類型 | 說明 | 對 Arena 的價值 |
|------|------|----------------|
| **策略合作** | 共同開發市場 | 擴大影響力 |
| **互補型合作** | 填補你的能力缺口 | 完整服務 |
| **平台合作** | 分銷/曝光管道 | 流量 |
| **聯盟合作** | 互相推廣 | 資源置換 |

### 合作夥伴識別

```
【策略合作夥伴】
- 潛在夥伴：
- 互補點：
- 合作方式：

【專家網絡】
- 需要的專家類型：
- 潛在人選：
- 合作模式：
```

### 合作機會評估

| 評估商數 | 問題 |
|----------|------|
| 互補性 | 對方的優勢能否補足我的劣勢？ |
| 價值觀一致性 | 我們的核心價值觀是否吻合？ |
| 利益分配 | 合作的利益如何分配？ |
| 退出機制 | 如果合作不順利，如何優雅退出？ |
