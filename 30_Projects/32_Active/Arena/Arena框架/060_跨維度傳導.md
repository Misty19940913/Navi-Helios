# 060_跨維度傳導

> Arena 的跨維度傳導系統 — 融合 HUMAN 3.0
> 產出時間：2026-04-01
> 更新时间：2026-04-01（採用 H3.0-跨象限效應）

---

## 融合說明

Arena 的跨維度傳導直接採用 HUMAN 3.0 的跨象限效應邏輯，是理解維度間相互影響的核心框架。

---

## 核心定義

四大象限（心智、身體、精神、職業）並非孤立，而是互為燃料與限制。一個象限的突破能為其他象限提供剩餘資本；反之，一個象限的崩潰會吸乾全系統的資源。

---

## 綜效模式（正向螺旋）

### 良性螺旋

```
身體（能量）
    ↓
精神（連結）
    ↓
心智（清晰度）
    ↓
價值（資源）
    ↓
身體（再投資）
```

**循環描述**：當你的身體有能量時，你更有能力建立深度關係；深度關係带來情緒支持；情緒支持使心智更清晰；心智清晰带來更好的價值創造；價值創造再回饋身體能量。

### 卓越迴圈

```
價值（挑戰）
    ↓
心智（學習）
    ↓
身體（實踐）
    ↓
精神（意義）
    ↓
價值（精通）
```

**循環描述**：有意義的工作帶來挑戰；挑戰驅動心智持續學習；學習需要身體實踐；身體實踐带來具身智慧；具身智慧深化精神意義；意義强化對工作的承諾。

---

## 阻礙模式（負向陷阱）

### 貧窮陷阱

```
價值低落 → 生存壓力 → 所有象限被抽乾
    ↑_____________________________↓
```

**循環描述**：價值創造不足導致財務壓力；財務壓力讓你無法休息、投資健康、處理關係；身體/關係問題進一步損害心智清晰度；心智不清晰導致價值創造效率降低。

**解鎖方式**：優先穩定價值維度，打破第一個環節。

### 成功陷阱（忙碌至死）

```
價值過度擴張 → 身體/關係被犧牲 → 過勞崩潰
    ↑_____________________________↓
```

**循環描述**：過度專注於價值創造；犧牲睡眠、運動、關係；身體耗竭導致情緒波動；情緒波動損害人際關係；人際關係空洞減少情緒支持；情緒支持減少後，工作效率降低但更拼命。

**解鎖方式**：建立價值邊界，強制重分配時間到身體/關係。

### 最佳化陷阱（分析癰瘓）

```
心智過度負載 → 身體脫節 → 價值產出具體化失敗
    ↑____________________________↓
```

**循環描述**：過度學習、過度分析、過度優化系統；與身體感覺失去連結；身體不知道什麼是「足夠」；遲遲無法做出價值創造的決定；價值創造失敗讓心智更焦慮、更想學習。

**解鎖方式**：先動身體（運動、身體工作），讓身體告訴心智什麼是足夠。

### 靈性逃避（空談）

```
精神過度擴張 → 身體/價值被忽視 → 現實崩潰
    ↑____________________________↓
```

**循環描述**：過度專注於意義、靈性、關係；忽視身體健康和經濟現實；身體/經濟問題產生焦慮；用更多靈性逃避焦慮；焦慮損害靈性修煉的品質。

**解鎖方式**：將靈性應用於解決身體/經濟問題，特別是服務他人。

---

## 級聯動態

### 突破的級聯

當一個象限發生突破時：

```
心智突破 → 身體跟進 → 價值增强 → 關係擴展
```

**範例**：心智維度從 2.0 進化到 2.5
- 心智更清晰 → 能更覺察身體需求
- 身體需求被滿足 → 能量提升
- 能量提升 → 工作產出更好
- 工作產出更好 → 有更多資源投資關係

### 崩潰的級聯

當一個象限發生崩潰時：

```
關係崩潰 → 心智損耗 → 價值停滯 → 身體耗竭
```

**範例**：關係維度突然崩潰
- 關係崩潰 → 情緒損耗心智資源
- 心智資源減少 → 無法專注工作
- 工作停滯 → 財務壓力增加
- 財務壓力 → 無法投資健康

## Arena 的應用

### 前置條件審核

在 Stage 2 規劃目標時：
1. 識別用戶想要發展的目標維度
2. 計算該維度需要的其他維度等級
3. 如果其他維度落後，先建議發展落後維度

### 解鎖順序原則

| 鎖住的維度 | 優先發展 |
|-------------|----------|
| 身體落後 | 先發展身體，為其他維度提供能量 |
| 關係落後 | 先發展關係，為其他維度提供意義感 |
| 心智落後 | 先發展心智，為其他維度提供清晰度 |
| 價值落後 | 先發展價值，打破貧窮陷阱 |

### AI 訪談應用

當用戶抱怨某個維度的問題時，檢查是否是其他維度的問題在傳導：

| 用戶抱怨 | 可能真正原因 |
|---------|-------------|
| 身體疲憊 | 價值維度過度擴張 |
| 關係困難 | 身體/心智維度問題 |
| 工作缺乏動力 | 意義感（精神維度）匱乏 |
| 心智不清晰 | 身體耗竭或關係問題 |

---

## 與生活設計框架的對應

> 當系統卡住時，檢查哪個象限是「最大限制因素」，而非在原本就強大的象限上過度使力。

**最小有效發展（MED）在跨維度傳導中的應用**：
- 找到那個「解決後能自動改善其他維度」的關鍵維度
- 不要同時發展所有維度
- 先打通瓶頸維度，讓級聯效應自然發生
