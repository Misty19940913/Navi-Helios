## 摘要

收入優化分為職業發展（主動收入）與投資收益（被動收入）兩大方向，核心原則是提升主動收入並轉化為多元被動收入。

## 職業發展

| 策略 | 說明 |
|------|------|
| 技能提升 | 投資於專業技能和證照 |
| 績效表現 | 爭取加薪和升遷機會 |
| 副業發展 | 建立多元收入來源 |
| 網路建立 | 擴展人脈和機會 |

## 投資收益

| 類型 | 說明 |
|------|------|
| 股息收入 | 選擇高股息股票或ETF |
| 債券利息 | 投資於優質債券 |
| 租金收入 | 投資於房地產 |
| 版權收入 | 創作內容變現 |
