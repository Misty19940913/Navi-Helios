## 摘要

財務自由的實現需要三個可控面向：資金（增加收入、提高儲蓄率）、投資報酬率（覆蓋通膨）、生活支出（控制物慾）。

## 核心公式

```
財務自由 = (收入 × 儲蓄率) × 投資報酬率 > 生活支出 × (1 + 通膨率)
```

## 三大操控面向

| 面向 | 說明 |
|------|------|
| 資金 | 增加收入管道，提高儲蓄率 |
| 投資報酬率 | 培養投資能力，覆蓋通貨膨脹 |
| 生活支出 | 控制支出水平，保持低物慾 |

## 時間因素

多數人實現財務自由需要**10年**，期間會面臨結婚、購屋、育兒、照護等人生階段變化，支出變化需要滾動調整。

## 多數人實現財務自由用10年即可達成。
人生在不同的階段，都有不一樣的支出，10年的跨度基本上會面臨2個人生階段。
20歲至30歲，可能面臨單身至結婚；30歲至40歲，可能面臨購屋、養育子女；40歲至50歲，可能面臨長輩照護。
因此這都會影響生活支出的變化，而導致財務自由的進度。

再者，以10年的時間來說，財務狀況的起伏也會影響支出的高低，例如：經濟狀況上升，物慾也會隨之上升，從而導致進度的變化，又或者物價的上漲，因此財務自由是需要不斷滾動調整的。

實現財務自由的兩大核心要素：資金（[[自由數字]]）、報酬率
我們努力工作賺取收入，透過儲蓄來累積資金，再做良好的投資，來取得合理的報酬，以此支付開支。
總結以上，財富自由的公式就變成
```
財務自由 = 被動收入 >總支出
        = 資金 * 投資報酬率 > 生活支出 * （1 + 通貨膨脹率）
        = （收入 * 儲蓄率）* 投資報酬率 >生活支出 * （1 + 通貨膨脹率）
```
在上面這四項中我們可以操作的面向有三個：資金、投資報酬率、生活支出。

---
簡單論述如下
資金：增加各式[[200領域/260財務/261個人金融知識庫/收入管道]]，無論主動或被動，因此涵蓋主業發展、副業培養，並學習資金配置。
- 儲蓄：收入增加後，依靠儲蓄來留下投資用的資金，因此儲蓄率高低也決定財務自由的進度。
投資：培養機會識別能力、資產配置，因此投資不全是去做錢滾錢這件事，也包含投資自己。另外此處的投資報酬率同樣需顧慮通貨膨脹這個因子，來因應部分利息可以回流至本金。簡單來說，投資報酬率需要去覆蓋通貨膨脹率，並產生足夠的支出費用。
