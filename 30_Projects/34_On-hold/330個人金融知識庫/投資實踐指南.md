# 投資實踐指南

本文已蒸馏为核心概念，详见：
- [[K-FIN-040 新手投資路徑]]

## 核心摘要

### 新手投資路徑

**第一階段：基礎建立（0-6個月）**
1. 學習基本投資知識
2. 開立證券帳戶
3. 開始定期定額投資
4. 建立投資記錄

**第二階段：策略優化（6-18個月）**
1. 學習資產配置
2. 開始建立投資組合
3. 實踐再平衡策略
4. 檢視投資表現

**第三階段：進階應用（18個月以上）**
1. 學習進階投資策略
2. 優化投資組合
3. 考慮多元投資
4. 建立長期規劃

### 投資檢查清單

**每月檢查項目**
- [ ] 檢視投資組合表現
- [ ] 確認定期投資執行
- [ ] 檢視市場動態
- [ ] 記錄投資心得

**每季檢查項目**
- [ ] 分析投資組合配置
- [ ] 評估風險水平
- [ ] 考慮再平衡需求
- [ ] 調整投資策略

**每年檢查項目**
- [ ] 檢視年度投資目標
- [ ] 評估整體投資表現
- [ ] 制定下年度投資計劃
- [ ] 優化投資組合

### 核心原則
- **長期持有**：時間是最好的槓桿
- **分散風險**：不要把所有資金集中於單一標的
- **定期檢視**：根據市場變化和人生階段調整策略
