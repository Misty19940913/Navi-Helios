## 摘要

舉債致富分析報表用於追蹤和評估槓桿投資策略的執行成效，核心功能包括：績效追蹤、風險監控、成本分析、決策支持。

## 核心數據欄位

| 類別 | 欄位 |
|------|------|
| 資金結構 | 投資總金額、自有資金、借貸金額、槓桿比率 |
| 收益分析 | 期間收益、累計收益、ROI、ROE |
| 成本分析 | 期間利息支出、累計利息支出、利息覆蓋率 |
| 風險指標 | 最大回撤、波動率、夏普比率、VaR |

## 關鍵公式

- 槓桿比率 = 投資總金額 / 自有資金
- ROI = 淨收益 / 投資總金額
- ROE = 淨收益 / 自有資金
- 利息覆蓋率 = 投資收益 / 利息支出

## 實務案例

- **房地產**：2,000萬投資，600萬自備，1,400萬借貸，利率2.5%，ROE 10.83%
- **股票**：100萬投資，50萬自備，50萬借貸，利率4%，ROE 16%

本報表專為「舉債致富」策略設計，用於追蹤和評估槓桿投資策略的執行成效。透過系統化的數據記錄和分析，幫助投資者清晰了解槓桿投資的績效表現，及時調整投資組合，並更好地評估風險與回報之間的平衡。

### 報表功能
- **績效追蹤**：詳細記錄投資表現和收益情況
- **風險監控**：監控關鍵風險指標和預警信號
- **成本分析**：分析借貸成本和投資回報的關係
- **決策支持**：提供數據支持投資決策和策略調整

## 核心數據欄位

### 基本投資資訊

#### 投資識別
- **投資項目編號**：唯一識別碼
- **投資標的名稱**：具體投資資產名稱
- **投資類別**：房地產、股票、基金等
- **投資起始日期**：開始投資的日期
- **報告期間**：報表涵蓋的時間範圍

#### 資金結構
- **投資總金額**：投入特定資產的總金額
- **自有資金**：個人投入的資金
- **借貸金額**：為該項投資所借入的資金數額
- **槓桿比率**：投資總金額 / 自有資金

### 借貸資訊

#### 貸款條件
- **借貸利率**：借入資金的年利率
- **貸款期限**：借款的總時間長度
- **還款方式**：等額本息、等額本金等
- **還款頻率**：月付、季付、年付等

#### 還款計劃
- **每月還款金額**：固定還款金額
- **本金還款**：每月償還的本金部分
- **利息還款**：每月支付的利息部分
- **剩餘本金**：尚未償還的本金餘額

### 投資表現

#### 價值追蹤
- **期初價值**：報告期開始時的投資價值
- **期末價值**：報告期結束時的投資價值
- **價值變化**：期末價值 - 期初價值
- **價值變化率**：價值變化 / 期初價值

#### 收益分析
- **期間收益**：報告期內產生的收益
- **累計收益**：自投資起始以來的總收益
- **收益來源**：股息、利息、租金、資本利得等
- **年化收益率**：年化投資回報率

### 成本分析

#### 利息支出
- **期間利息支出**：報告期內支付的借款利息
- **累計利息支出**：自借款起始以來的總利息
- **平均利率**：期間平均借貸利率
- **利息覆蓋率**：投資收益 / 利息支出

#### 其他成本
- **手續費**：交易手續費、管理費等
- **稅務成本**：相關稅務支出
- **機會成本**：資金其他用途的潛在收益
- **總成本**：所有相關成本總和

### 績效指標

#### 回報率分析
- **投資回報率（ROI）**：淨收益 / 投資總金額
- **股權回報率（ROE）**：淨收益 / 自有資金
- **槓桿效應**：ROE - ROI
- **風險調整後報酬率**：考慮風險後的實際報酬

#### 風險指標
- **最大回撤**：投資期間的最大損失幅度
- **波動率**：投資價值的波動程度
- **夏普比率**：風險調整後的超額報酬
- **VaR（風險價值）**：在特定置信度下的最大可能損失

### 關鍵公式

#### 槓桿比率
```
=投資總金額/(投資總金額-借貸金額)
```

#### 淨收益
```
=累計收益-累計利息支出
```

#### 投資回報率（ROI）
```
=淨收益/投資總金額
```

#### 股權回報率（ROE）
```
=淨收益/(投資總金額-借貸金額)
```

#### 利息覆蓋率
```
=投資收益/利息支出
```

#### 年化收益率
```
=((期末價值/期初價值)^(12/月數))-1
```

#### 槓桿效應分析圖
- **類型**：散點圖
- **X軸**：槓桿比率
- **Y軸**：ROE
- **用途**：分析槓桿對回報率的影響

#### 風險回報散點圖
- **類型**：散點圖
- **X軸**：風險（波動率）
- **Y軸**：回報率
- **用途**：評估風險與回報的關係

## 報表分析要點

### 績效分析

#### 絕對績效
- 投資總回報是否為正
- 是否達到預期目標
- 與市場基準的比較

#### 相對績效
- 與無槓桿投資的比較
- 與其他投資策略的比較
- 與風險調整後基準的比較

### 風險分析

#### 槓桿風險
- 槓桿比率是否在合理範圍
- 利息負擔是否可承受
- 流動性是否充足

#### 市場風險
- 投資標的的波動性
- 市場環境的變化
- 相關性風險

### 成本效益分析

#### 借貸成本
- 借貸利率是否合理
- 利息支出占收益的比例
- 是否有更便宜的融資方式

#### 機會成本
- 資金其他用途的收益
- 時間成本的考量
- 風險調整後的淨收益

### 房地產投資案例

- 借貸金額：50萬元

- 投資期間：1年
