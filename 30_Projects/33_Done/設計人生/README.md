# 設計人生

> 將設計思考應用於個人發展，在充滿不確定性的現代社會中，創造有意義且可持續的生活方式。

---

## 這是什麼？

一套完整的人生設計教材，以史丹佛大學 d.school 的《Designing Your Life》為核心理論框架，分為七個章節，從自我認知到原型驗證，幫助你系統性地探索和設計自己的人生。

---

## 快速開始

1. **先讀** [[設計人生MOC]] — 了解整體結構與學習路徑
2. **從** [[010_理論基礎]] **開始** — 建立設計師思維
3. **按章節推進** — 每章約 15-30 分鐘
4. **實際練習** — 每章都有可執行的練習模板
