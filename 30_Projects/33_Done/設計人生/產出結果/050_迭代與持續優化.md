# 050_迭代與持續優化

> 人生設計並非一個線性過程，終點不是一個完美的計畫，而是一個持續演進、不斷優化的循環。在透過原型設計收集到真實世界的數據後，我們進入了選擇、學習與迭代的階段。
> 
> **版本**：1.1.0 | **編輯**：Navi | **更新**：2026-04-06

---

## 一、如何做出更好的選擇？

原型實驗的結果，往往不是一個明確的「是」或「否」，而是帶來了更豐富的資訊與感受。當你面對多個看似都不錯的選項時，如何做出選擇？

### 1.1 拋開「最佳選擇」的迷思

人生設計的選擇，不是為了找到那個唯一「正確」或「最佳」的答案。許多時候，好的選擇有好幾個。

> 目標是做出一個「好」的選擇，然後全心投入，並從中學習。

### 1.2 整合你的決策系統

好的決策，是「頭腦」與「心」的結合，需要整合四種思維方式：

| 思維方式 | 說明 | 應用 |
|----------|------|------|
| 認知 (Cognitive) | 運用理性分析 | 評估數據與事實 |
| 直覺 (Intuitive) | 傾聽你的內在聲音 | 感受哪個選項「對了」 |
| 情感 (Affective) | 體察你的情緒反應 | 留意你對不同選項的情緒 |
| 身體 (Somatic) | 留意身體的信號 | 感到興奮放鬆，還是緊張收縮？ |

### 1.3 決策流程：選擇 → 投入 → 放下

1. **收集與創造 (Gather & Create Options)**：你已經在發想階段做到了
2. **縮小範圍 (Narrow Down the List)**：當選項超過5個時，先刪除那些明顯不合適的
3. **做出選擇 (Choose Discerningly)**：運用上述的[[L-017-整合決策系統]]系統做出選擇
4. **擁抱並前行 (Let Go & Move On)**：
   > 這是最關鍵也最難的一步。一旦做出選擇，就要放下對其他可能性的留戀（「要是當初選了另一個會怎樣？」），全心投入你所選擇的道路

### 1.4 決策檢查清單

- [ ] 我已經收集了足夠的選項嗎？（至少 3 個）
- [ ] 我有用四種思維方式（認知、直覺、情感、身體）評估嗎？
- [ ] 我準備好「放下」其他選項了嗎？
- [ ] 我選擇後會全心投入嗎？

## 二、如何區分「該堅持」與「該放棄」？

這是人生設計中最困難的判斷之一。以下提供幾個判斷框架：

### 2.1 評估信號

| 信號 | 建議 |
|------|------|
| 持續的熱情與能量 | 堅持 |
| 看到進步與成長 | 堅持 |
| 外部回饋正面 | 堅持 |
| 長期能量耗竭 | 重新評估 |
| 價值觀衝突加劇 | 重新評估 |
| 多次嘗試仍原地踏步 | 重新評估 |

### 2.2 時間框架檢驗

問自己：
- 我在這個方向上投入了多少時間？
- 是否有明確的「最後期限」？
- 如果再坚持 3 個月，會有什麼不同？

### 2.3 「電梯測試」

如果你在電梯裡被問「你現在在做什麼？」，你能充满信心地回答嗎？

如果答案讓你感到遲疑或勉強，可能是需要重新考慮的信號。

### 2.4 成本收益分析

| 堅持的成本 | 放棄的成本 |
|------------|------------|
| 時間與精力 | 已經投入的資源 |
| 機會成本 | 他人看法 |
| 情緒損耗 | 不確定性 |

> **提醒**：有時「放棄」比「堅持」更需要勇氣。

### 2.5 堅持/放棄檢查清單

- [ ] 我有持續的熱情與能量嗎？
- [ ] 我看到進步與成長了嗎？
- [ ] 外部回饋正面嗎？
- [ ] 我有長期能量耗竭嗎？
- [ ] 我的價值觀有衝突嗎？
- [ ] 我嘗試了多次仍原地踏步嗎？

## 三、建立你的[[L-023-人生設計團隊]] - 實作版

你不是孤獨的設計師。一個強大的支援網絡能為你提供回饋、鼓勵與多元視角。

### 3.1 導師 (Mentors)

尋找那些在你想探索的領域有經驗、並願意分享智慧的人。

**如何找到導師？**
- 從你的專業網絡開始
- 參加產業活動
- 主動 LinkedIn 聯繫

**導師招募清單：**
- [ ] 列出你想请教的三个人
- [ ] 準備好 30 秒自我介紹
- [ ] 準備好 3-5 個具体问题
- [ ] 設定跟進時間表

### 3.2 夥伴 (Allies)

這是一群與你並肩作戰的同行者，你們可以定期分享進度、互相打氣。

**建議：**
- 2-5人的小組
- 定期聚會討論各自的人生設計挑戰
- 互相提供誠實的回饋

**夥伴建立清單：**
- [ ] 識別 2-3 个志同道合的朋友
- [ ] 提議每月一次的聚會
- [ ] 設定聚會主題（輪流主持）
- [ ] 建立 LINE 群組保持聯繫

### 3.3 社群 (Community)

參與和你志同道合的社群，無論是：
- 專業社團
- 興趣小組
- 校友會

**社群參與清單：**
- [ ] 搜尋相關社群（線上/線下）
- [ ] 選擇 1-2 個社群加入
- [ ] 設定每月參與頻率
- [ ] 主動貢獻（分享經驗、幫助他人）

### 3.4 團隊建立檢查清單

- [ ] 我有至少一位導師嗎？
- [ ] 我有小組夥伴嗎？
- [ ] 我有加入相關社群嗎？

## 四、心理彈性 (Resilience) 的建立方法

人生設計並非一帆風順。建立心理韌性，幫助你在遭遇挫折後恢復並繼續前進。

### 4.1 什麼是心理彈性？

心理彈性是指：
- 從逆境中恢復的能力
- 在壓力下保持彈性思維
- 將失敗視為學習而非終點

### 4.2 建立心理彈性的方法

**方法一：建立「失敗清單」**
> 記錄你從「失敗」的原型或嘗試中學到了什麼。這個練習能幫助你重新定義「失敗」，將其視為學習過程中不可或缺的一部分。

**失敗記錄模板：**
| 日期 | 失敗事件 | 學到的教訓 | 下次改進 |
|------|----------|------------|----------|
|      |          |            |          |

**方法二：練習「重擬」(Reframing)**
- 問自己：「這件事有沒有其他的詮釋角度？」
- 找尋「最糟中的最好」
- 將「我失敗了」轉為「我學到了」

**重擬練習清單：**
- [ ] 這件不好的事，有沒有其他詮釋？
- [ ] 最糟中的最好是什麼？
- [ ] 我從中學到了什麼？

**方法三：建立支持系統**
- 有人可以傾訴
- 有可以求助的資源
- 有可以放鬆的空間

**方法四：保持身體健康**
- 睡眠、運動、飲食
- 身體狀態影響心理韌性

**方法五：刻意練習「接受不確定性」**
- 每天做一個小決定
- 練習在不完全確定的情况行動

### 4.3 心理彈性檢查清單

- [ ] 我有記錄失敗並從中學習嗎？
- [ ] 我有練習重擬負面事件嗎？
- [ ] 我有支持系統可以傾訴嗎？
- [ ] 我有照顧好自己的身體嗎？
- [ ] 我有練習接受不確定性嗎？

## 五、迭代與持續優化框架

### 5.1 失敗分類與學習

將失敗視為學習的原料，而非終點。根據 DYL 原則，將遇到的挫折分類：

| 類型 | 說明 | 處理方式 |
|------|------|----------|
| 簡單錯誤 (Simple Error) | 可立即修正的疏忽 | 立即修正（例如：忘記發送郵件） |
| 弱點 (Weakness) | 需避免觸發情境的個人限制 | 覺察並管理（例如：在壓力下容易暴躁） |
| 成長機會 (Growth Opportunity) | 有明確解決方案的挑戰 | 學習克服（例如：需要學習新技能） |

### 5.2 失敗免疫力 (Failure Immunity)

> 記錄下你從「失敗」的原型或嘗試中學到了什麼。這個練習能幫助你重新定義「失敗」，將其視為學習過程中不可或缺的一部分。

**練習：**
1. 記錄每次失敗的經驗
2. 寫下你從中學到了什麼
3. 識別這個學習如何幫助你未來做出更好的決定

### 5.3 痛點診斷與根本原因分析

當原型測試帶來不滿時：

1. **回顧你的發現**：圈出在原型測試中帶來不滿的區域
2. **重擬問題 (Reframing)**：將「我不滿意薪資」重擬為「我提供的價值與市場需求不匹配」
3. **診斷根本原因**：
   - 是「關鍵資源」與「關鍵活動」不匹配？
   - 是「目標客層」與你的價值觀不符？
   - 是工作型態限制了自我表達？

## 六、實際迭代周期建議

人生設計不是一次性的專案，而是持續進行的過程。以下是建議的迭代周期：

### 6.1 日常覺察

- **頻率**：每天
- **內容**：時光日誌、能量覺察
- **工具**：情緒記錄、簡短反思

**日常檢查清單：**
- [ ] 我今天的能量水平如何？（1-10 分）
- [ ] 今天最讓我充實的活動是什麼？
- [ ] 今天最讓我耗竭的活動是什麼？

### 6.2 每月檢視

- **頻率**：每月
- **內容**：[[L-009-生命儀表板]]快速評估
- **問題**：
  - 這個月我的能量水平如何？
  - 有哪些領域需要關注？
  - 我的原型實驗進展如何？

**每月檢查清單：**
- [ ] 四個領域（健康、工作、遊戲、愛）的分數出來了嗎？
- [ ] 有哪些領域低於 5 分？
- [ ] 我有記錄原型實驗的結果嗎？
- [ ] 下個月我要專注什麼？

### 6.3 季度回顧

- **頻率**：每三個月
- **內容**：[[L-008-人生指南針]]校準、奧德賽計畫檢視
- **問題**：
  - 我的價值觀有變化嗎？
  - 當前道路是否與[[L-008-人生指南針]]一致？
  - 需要調整奧德賽計畫嗎？

**季度檢查清單：**
- [ ] 我的工作觀和生活觀有變化嗎？
- [ ] 當前生活與人生指南針一致嗎？
- [ ] 奧德賽計畫需要調整嗎？
- [ ] 我有設定下一季的目標嗎？

### 6.4 年度規劃

- **頻率**：每年
- **內容**：全面的自我評估、下一年度規劃
- **問題**：
  - 這一年我学到了什麼？
  - 我的生活哪些方面有所改善？
  - 明年我想要專注的是什麼？

**年度檢查清單：**
- [ ] 這一年的成就是什麼？
- [ ] 這一年的挑戰是什麼？
- [ ] 我學到了什麼？
- [ ] 明年我要專注的主題是什麼？

> **提醒**：這些周期是建議，你可以根據自己的生活節奏調整。重點是保持「覺察」與「行動」的平衡。

## 七、持續的人生設計循環

人生設計是一個永不停止的循環。你需要建立一個系統，讓自己定期回到這個框架中，進行評估與調整。

### 7.1 定期檢視人生指南針

每隔半年或一年，重新審視你的工作觀與生活觀：
- 它們是否有所改變？
- 你當前的生活是否與之對齊？

**指南針檢查清單：**
- [ ] 我的工作觀有變化嗎？
- [ ] 我的生活觀有變化嗎？
- [ ] 當前生活與指南針一致嗎？
- [ ] 需要做哪些調整？

### 7.2 更新你的儀表板

每季或每月快速檢查你的四個領域：
- 確保沒有哪個領域長期處於「紅色警戒」狀態
- 及時發現失衡並調整

### 7.3 保持時光日誌的習慣

不需要每天記錄，但當你感到迷惘或需要做出改變時：
- 重新開始記錄一到兩週
- 覺察你的能量變化
- 總能帶來新的洞見

### 7.4 勇敢地再次發想

當外部環境發生重大變化（如產業變革），或你的內在需求改變時（如家庭角色轉變）：
- 不要害怕重新啟動發想與原型設計的過程
- 人生設計是持續的，不是只做一次

## 八、擁抱變化

> 人生設計師擁抱變化，將生活視為一場充滿創造力的冒險。

### 8.1 變化不是失敗

- 計畫永遠趕不上變化
- 調整方向是正常的
- 每次迭代都是學習

### 8.2 持續設計

- 人生是一連串的設計
- 每個階段都是新的開始
- 保持好奇，保持行動

## 九、十個核心思維框架

人生設計不是一個具有一系列步驟的線性過程，而是一系列幫助你像設計師一樣思考和行動的心態和思維框架：

| 框架 | 核心概念 |
|------|----------|
| 1. 接受旅程 | 從根本上接受你在旅程中的位置 |
| 2. 好奇心 | 忘記尋找熱情；努力去了解自己 |
| 3. 明確重要 | 明確什麼對你來說是重要的 |
| 4. 多元路徑 | 集思廣益，想出多種可能的途徑 |
| 5. 建立網絡 | 透過同情和向他人學習來建立網絡 |
| 6. 設計故事 | 設計你的故事 + 你的個人品牌 |
| 7. 行動驗證 | 嘗試一下。在現實世界中測試你的想法和假設 |
| 8. 靈活調整 | 靈活。根據你所學到的知識調整你的計劃 |
| 9. 未來身份 | 專注於你未來會成為怎樣的人，而不是單一的靜態未來職業 |
| 10. 相信值得 | 相信你的生活值得設計 |
