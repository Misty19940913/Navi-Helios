# 030_多元未來發想

> 在完成自我盤點、釐清現況後，我們進入人生設計中極具創造力與樂趣的階段：發想 (Ideate)。這個階段的目標是打破「只有一條正確道路」的線性思維，為自己產生多種可能的人生劇本。
> 
> **版本**：1.1.0 | **編輯**：Navi | **更新**：2026-04-06

---

## 一、發散式思維：拋開限制，追求數量

在發想階段，我們需要刻意地「發散思考」，暫時拋開可行性、資源或他人眼光的限制。

### 1.1 核心原則

- **追求數量，而非品質**：目標是產生大量想法，即使有些看起來很瘋狂
- **不評判**：壓抑內在的批判聲音，所有點子在現階段都是好點子
- **鼓勵狂想**：允許自己夢想那些你從未想過可能實現的道路

### 1.2 設計師的法則

> 當你有多個好點子時，你更有可能從中發展出一個卓越的方案。

研究顯示，只考慮一個方案會讓我們陷入「我該不該做這個？」的僵局；而一次構思多個方案，則能讓我們的大腦轉換到比較與對照的模式，從而做出更靈活、更有創意的決策。

### 1.3 發想檢查清單

- [ ] 允許自己產生瘋狂的想法
- [ ] 不批判任何點子
- [ ] 目標：至少產出 10 個想法
- [ ] 記錄所有想法，不要丟掉任何一個

## 二、核心工具：[[L-011-奧德賽計劃]] (Odyssey Plan)

「[[L-011-奧德賽計劃]]」是人生設計中最核心的發想工具。它引導你具體地規劃出**三種截然不同**的五年版本的人生，讓你真切地「看見」不同的可能性。

### 2.1 為何是三個版本？

| 數量 | 效果 |
|------|------|
| 1個 | 陷入「做還是不做」的二元思維 |
| 2個 | 陷入比較，難以抉擇 |
| 3個 | 觸發比較與對照，激發創意 |

三個版本能讓我們的大腦自由地在不同可能性之間跳轉，發現單一方案中看不到的機會。

### 2.2 三個版本的設計

**計畫一：你現有的道路**
- 如果你繼續現在的道路，它會是什麼樣子？
- 可能是你當前專業或工作的延伸與深化

**計畫二：如果計畫一消失了**
- 想像一下，如果計畫一的選項（例如，你所在的產業）明天就消失了
- 或者被AI取代了，你會做什麼？
- 這是一個備用或轉軸計畫 (Pivot Plan)

**計畫三：你的狂想人生**
- 如果錢和他人眼光都不是問題，你會做什麼或成為什麼樣的人？
- 在這個計畫中，你可以完全釋放你的好奇心與渴望

### 2.3 每個計畫的構成要素（實際操作版）

為確保你的計畫不只是空想，請為**每一個計畫**詳細規劃以下內容：

#### Step 1: 視覺化時間軸
```
Year 1 ─── Year 2 ─── Year 3 ─── Year 4 ─── Year 5
  │          │          │          │          │
里程碑   里程碑    里程碑    里程碑    里程碑
```

**操作問題：**
- 第一年我要達到什麼？
- 第三年的轉折點是什麼？
- 五年後的我是什麼樣子？

#### Step 2: 計畫標題
給這個版本的你一個生動的標題：
- 例如：「鄉村數位遊牧教育家」、「城市永續農業顧問」
- 用一句話描述這個版本的生活

#### Step 3: 關鍵問題
寫下這個計畫主要想探索或回答的 2-3 個核心問題：
- 例如：「我真的能在非營利組織中找到滿足感嗎？」

#### Step 4: 儀表板評估（實際填寫）

| 維度 | 分數 (1-10) | 具體原因 |
|------|-------------|----------|
| **資源 (Resources)** | ___ | 需要什麼？有多少？ |
| **喜愛度 (I Like It)** | ___ | 這個計畫讓我多興奮？ |
| **信心度 (Confidence)** | ___ | 我有多大把握實現？ |
| **一致性 (Coherence)** | ___ | 與我的人生指南針多契合？ |

### 2.4 奧德賽計畫檢查清單

- [ ] 我有 3 個不同版本的五年計畫嗎？
- [ ] 每個計畫都有時間軸嗎？
- [ ] 每個計畫都有生動的標題嗎？
- [ ] 每個計畫都有 2-3 個關鍵問題嗎？
- [ ] 每個計畫都有四維度評估分數嗎？

### 2.5 下載模板

[[[L-011-奧德賽計劃]]模板](https://designingyour.life/resources-designing-your-life/)

## 三、其他發想輔助工具

### 3.1 心智圖 (Mind Mapping)

從一個你感興趣的核心主題出發，向外輻射所有相關的想法、職業、專案或學習路徑。

**練習：**
1. 從你的[[L-010-時光日誌]]中找到的高能量活動作為中心
2. 向外輻射相關的可能性
3. 不要審核，讓想法自由流動

### 3.2 最糟想法 (Worst Possible Idea)

刻意去想「最糟糕」的解決方案是什麼。這個練習可以打破完美主義的枷鎖，並常常能在反轉思考中找到意想不到的創新點。

### 3.3 10-10-10 評估

對於每個選項，回答：
- 10分鐘後，你會怎麼看待這個決定？
- 10個月後呢？
- 10年後呢？

這幫助你從長期視角審視短期決策。

### 3.4 [[L-019-情境模擬練習]]與預測

- 建立不同情境的假設
- 預測可能的發展路徑
- 評估風險與機會
- 制定應對策略

## 四、情境模擬練習 - 實際操作版

[[L-019-情境模擬練習]]是幫助你「看見」未來的一種有力工具，透過創造具體的假設場景，讓抽象的可能性變得生動可感。

### 4.1 練習目的

- 降低對未知的焦慮
- 識別潜在的障礙與機會
- 提前規劃應對策略
- 增強對未來的真實感

### 4.2 練習方法：五年後的一天（步驟版）

**Step 1: 選擇一個奧德賽計畫**
- 從三個計畫中選擇你最感興趣的

**Step 2: 想像五年後的某個週二**
- 你正在過典型的一天

**Step 3: 用現在時態寫下這一天**

| 時間 | 活動 | 感受 |
|------|------|------|
| 6:00-7:00 | 起床後做什麼？ |  |
| 7:00-9:00 | 上班路上/早餐？ |  |
| 9:00-12:00 | 上午工作內容？ |  |
| 12:00-13:00 | 午餐？和誰？ |  |
| 13:00-18:00 | 下午工作內容？ |  |
| 18:00-20:00 | 下班後活動？ |  |
| 20:00-22:00 | 晚間時光？ |  |
| 22:00-23:00 | 睡前做什麼？ |  |

### 4.3 多重情境對比

不要只寫一個情境。為每個[[L-011-奧德賽計劃]]創造一個「五年後的一天」，然後比較：

- [ ] 哪個情境讓你醒來時更期待？
- [ ] 哪個情境讓你感到更自然？
- [ ] 哪個情境的能量最高？
- [ ] 哪個情境讓你想起床？

### 4.4 情境模擬檢查清單

- [ ] 我為每個奧德賽計畫都寫了情境模擬嗎？
- [ ] 我有比較三個情境的差異嗎？
- [ ] 我發現哪個情境對我最有吸引力？

## 五、失敗想像練習 - 實際操作版

「最糟情況」(Worst-Case Scenario) 練習不是為了嚇自己，而是為了：

1. **去除恐懼的神秘感**：很多恐懼在明確化後就不再那麼可怕
2. **發現隱藏的資源**：最糟情況往往揭示我們現有但未覺察的支援
3. **提前準備**：即使最糟情況發生，我們也已有所準備

### 5.1 練習：我的最糟一年

**步驟：**
1. 選擇一個[[L-011-奧德賽計劃]]
2. 盡可能詳細地描述「如果這個計畫完全失敗」的一年會是什麼樣子
3. 具體回答：

| 問題 | 回答 |
|------|------|
| 最糟的情況是什麼？ | |
| 誰會受到影響？ | |
| 我會失去什麼？ | |
| 我還擁有什麼？ | |
| 我會如何處理？ | |
| 一年後的我會說什麼？ | |

### 5.2 從失敗中發現洞見

完成[[L-016-失敗想像練習]]後，往往會發現：

- **「最糟情況」並不如想像中可怕**：很多後果是我們可以承受的
- **我們比想像中更有韌性**：我們會找到生存的方法
- **有些恐懼是過度的**：明確後可以放下
- **需要提前準備的領域**：哪些支持系統需要加強

### 5.3 失敗想像檢查清單

- [ ] 我做過失敗想像練習嗎？
- [ ] 我識別出最糟情況中仍然擁有的資源了嗎？
- [ ] 我有發現哪些恐懼是被過度放大的嗎？

## 六、處理「不敢想」的心理障礙 - 突破行動

很多人無法進入發想狀態，因為內心有種種障礙。以下是常見障礙與突破方法：

### 6.1 常見心理障礙

| 障礙 | 內在訊息 | 根源 |
|------|----------|------|
| 完美主義 | 「如果不能做到最好，不如不想」 | 害怕失敗 |
| 恐懼失敗 | 「如果夢想破滅，我會無法接受」 | 自我價值綁定於結果 |
| 社會期待 | 「別人會怎麼看我？」 | 過度在意他人眼光 |
| 限制信念 | 「這不可能實現」 | 過去經驗的投射 |
| 冒名頂替症候群 | 「我不夠資格過那種生活」 | 低自我效能感 |

### 6.2 突破策略 - 立即行動版

**策略一：暫時放下「如何」**
> 讓自己停留在「想要什麼」，而不是「如何做到」。這兩個問題需要不同的思維模式。

**行動：** 
- [ ] 先寫出 10 個「想要」，不要考慮「如何做到」
- [ ] 告訴自己：「今天是純粹做夢日」

**策略二：給自己「做夢的許可」**
> 明確告訴自己：「在這個練習中，沒有任何想法是錯誤的。瘋狂的想法是被鼓勵的。」

**行動：**
- [ ] 對自己大聲說：「我可以做任何夢」
- [ ] 把這個承諾寫下來

**策略三：從「別人的故事」開始**
> 如果直接想自己的未來太難，先從「如果我羨慕某個人的生活」開始。

**行動：**
- [ ] 找一個你羨慕的人，寫下他們的生活為什麼吸引你
- [ ] 從這裡開始延伸到你可能的版本

**策略四：身體先行**
> 有時身體姿態可以影響心理狀態。站直、雙手展開、抬頭——這個姿勢本身就傳遞「我可以」的訊號。

**行動：**
- [ ] 站起來，擺出「勝利者姿態」2分鐘
- [ ] 帶著這個感覺開始發想

### 6.3 心理障礙突破檢查清單

- [ ] 我識別出我的心理障礙了嗎？
- [ ] 我選擇了一個突破策略並執行了嗎？
- [ ] 我給自己「做夢的許可」了嗎？

## 七、發想結果的應用

### 7.1 從發想到行動

發想階段產生的多元可能性，是修正與重塑你人生設計的絕佳素材：

- **新的可能性**：發現從未考慮過的道路
- **舊觀念的挑戰**：重新審視自己的假設
- **恐懼的發現**：識別阻止你前進的恐懼

### 7.2 過濾與聚焦

當你有了大量想法後：

1. **刪除明顯不合適的**：那些與你的價值觀完全衝突的
2. **保留有潛力的**：即使看起來瘋狂
3. **標記需要驗證的**：需要透過原型測試的

### 7.3 發想後檢查清單

- [ ] 我有足夠多的選項嗎？（至少 3 個）
- [ ] 我刪除了明顯不合適的選項了嗎？
- [ ] 我標記了哪些需要原型驗證嗎？
- [ ] 我的選項足夠多元嗎？（不同領域、不同風險程度）

## 八、多元未來思維的核心價值

| 價值 | 說明 |
|------|------|
| 打破單一路徑 | 挑戰「唯一正確道路」的迷思 |
| 減少決策焦慮 | 創造思考空間 |
| 鼓勵非傳統 | 探索高風險或創新選項 |
| 培養適應力 | 面對不確定性的能力 |
