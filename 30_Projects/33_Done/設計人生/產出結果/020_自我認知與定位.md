# 020_自我認知與定位

> 在設計任何事物之前，你必須先了解你的使用者——在人生設計中，**你，就是最重要的使用者**。自我認知與定位是整個設計過程的基石。

---

## 一、從根本上接受現在的自己

人生設計的第一步是**徹底接受(Radical Acceptance)**。這意味著不帶批判地、清晰地認識到你此刻的狀態、感受與處境，並以同情心對待這段經驗。

> 就像承包商在建造建築物之前會檢查地基一樣，從事人生設計的人必須在設計未來之前評估自己的生活、信仰和假設。

### 1.1 練習：寫下你的旅程

誠實地回答以下問題，釐清你真正在什麼位置：

| 問題 | 內容 |
|------|------|
| 我認為我最感興趣的是追求 | ___________，儘管我對 ___________ 很好奇 |
| 在思考我的未來時，我最大的疑問是 | ___________ |
| 我最擔心的是 | ___________ |
| 我目前專注於嘗試 | ___________ |
| 對我來說不起作用的是 | ___________ |
| 對我的生活有益的新變化是 | ___________ |
| 我有興趣花更多時間了解 | ___________ |

**練習的重點**：誠實回答你**現在**的位置，而不是你母親、顧問或社會期待你應該在的位置。

### 1.2 接受的力量

- 接受不是放棄，而是認清現實
- 只有接受現在，才能真正開始改變
- 批判自己會阻礙創意發想
- 意識到許多人都有類似的恐懼和疑問本身就是一種解脫

## 二、建立你的[[L-008-人生指南針]]

你的「[[L-008-人生指南針]]」由「[[L-008-人生指南針]]」與「[[L-008-人生指南針]]」構成，它能確保你所做的重大決定都與你最深層的[[K-SELF_價值觀探索]]保持一致。

### 2.1 [[L-008-人生指南針]] (Workview)

[[L-008-人生指南針]]是你對「工作」這件事的看法總結。它回答了工作在你生命中的意義與目的。

**反思問題：**
- 為什麼要工作？工作應該帶來什麼？
- 工作的意義是什麼？它與個人、他人、社會的關係是什麼？
- 什麼才算是「好」的工作或「值得」的工作？
- 金錢、成長、經驗、貢獻之間的比重為何？

### 2.2 [[L-008-人生指南針]] (Lifeview)

[[L-008-人生指南針]]是你對「世界如何運作」以及「什麼是美好人生」的看法總結。它是你個人的生命哲學。

**反思問題：**
- 人生的意義或目的是什麼？
- 對我而言，最重要的是什麼？
- 我與他人、世界的關係是什麼？
- 什麼是善？什麼是惡？生命中是否有更高力量的存在？
- 什麼能為我的人生帶來喜悅、滿足與圓滿？

### 2.3 整合與校準

完成上述兩份觀點後，花時間比較它們，並反思：
- 我的[[L-008-人生指南針]]與[[L-008-人生指南針]]在哪裡互補？
- 它們在哪裡有衝突或不一致？
- 這種一致或不一致，如何影響我當前的生活與職涯？

這份整合後的洞見，就是你獨一無二的「[[L-008-人生指南針]]」。

## 三、盤點你的現況：儀表板與日誌

在出發前，你需要一張地圖，知道自己身在何處。以下工具能幫助你視覺化地盤點現況。

### 3.1 [[L-009-生命儀表板]] (Life Dashboard)

快速評估你在四個人生關鍵領域的「油量」是否充足：

| 領域 | 說明 | 評估指標 |
|------|------|----------|
| 健康 | 身體與心理 | 睡眠、運動、飲食、情緒 |
| 工作 | 職業與事業 | 滿意度、收入、成長 |
| 遊戲 | 休閒與創造 | 樂趣、創意、休閒品質 |
| 愛 | 親密與社群 | 關係、親情、友誼 |

**練習：** 為每個領域打分數（1-10），一眼看出哪個領域失衡，需要優先關注。

### 3.2 [[L-010-時光日誌]] (Good Time Journal) / 能量參與度地圖

這是一個更細緻的工具。透過記錄日常活動，並標註你在活動中的：

- **參與度 (Engagement)**：你在這個活動中有多投入？
- **能量水平 (Energy)**：這個活動讓你感到精力充沛還是耗竭？

**你可以發現：**
- **讓你進入心流狀態的活動**：這些是你的能量來源
- **讓你感到耗竭的活動**：這些是你的能量消耗點

> 這個練習的重點不在於立刻移除所有耗能的活動，而是「覺察」它們，並思考如何在未來的生活設計中，增加高能量活動的比例。

### 3.3 [[K-SELF_生命之輪]] (Wheel of Life)

從更多維度評估你對目前生活的滿意度：

1. 財務
2. 事業
3. 健康
4. 家庭
5. 社交
6. 休閒
7. 個人成長
8. 精神/靈性

**練習：** 在輪子上標記每個維度的滿意度（0-10），視覺化你的生活全景。

## 四、從過去的經驗中挖掘寶藏

### 4.1 生命歷程探索 (Lifeline Discovery)

繪製你的人生時間軸，標記出高峰與低谷的事件。深入反思這些事件，你會發現：

- 重複出現的模式
- 讓你感到興奮的元素
- 潛在的天賦與熱情

**練習：**
1. 畫一條時間軸，標記人生中的重要事件
2. 標註每個事件的情感價值（正向/負向）
3. 識別反覆出現的主題

### 4.2 [[K-SELF_價值觀探索]]探索

透過[[K-SELF_價值觀探索]]卡片排序、反思重大決定等方式，釐清指導你人生的核心信念。

> **關鍵洞察**：忘記「尋找熱情」，努力去「了解自己」，你的[[K-SELF_價值觀探索]]就是最好的嚮導。

**常見核心[[K-SELF_價值觀探索]]：**
- 自由、創造力、獨立
- 家庭、健康、財務安全
- 成就感、影響力、服務
- 冒險、平衡、快樂

## 五、技能與優勢識別

### 5.1 盤點你的資源

你是誰？你擁有什麼？

| 類型 | 範例 |
|------|------|
| 才能 | 溝通、分析、創意 |
| 知識 | 專業技術、產業經驗 |
| 人脈 | 同事、導師、朋友 |
| 經驗 | 專案、創業、跨國 |

### 5.2 識別可轉移技能

有些技能可以在不同領域通用：
- 專案管理
- 團隊協作
- 問題解決
- 溝通表達

---

### 5.3 自我理解工具（整合自泰勒中心）

#### 自我反思問題

這些問題非常適合反思，幫助你深入了解自己：

1. 如果外星人從外太空墜落，你最想向他們展示或解釋世界的什麼，是因為你對它如此著迷還是如此排斥？
2. 你收聽的播客的主要主題是什麼？
3. 你可以滿懷熱情地銷售什麼？
4. 你到底了解些什麼？
5. 你在學生時期有哪些經驗賦予你獨特的技能和見解？（實習、學生組織、專案、培訓等）

#### 家庭影響活動

深入了解家庭對你的影響，回想童年時對你影響最大的成年人：

- 這些成年人對於工作有何感受？（熱愛？討厭？）
- 這給他們帶來了自豪感還是恥辱？

然後建立兩列：**擁抱** 和 **釋放**

| 擁抱（我想保留的） | 釋放（不再適用的） |
|-------------------|-------------------|
| 工作應該對社會做出貢獻 | 唯一可接受的工作是醫生或律師 |
| 工作關乎與你一起工作的人 | ... |

> 重要的是：你有權利選擇自己的訊息和[[K-SELF_價值觀探索]]作為指導你前進的指南針。

#### 無畏使命宣言

創建三個版本的使命宣言：

| 版本 | 描述 |
|------|------|
| 首要考慮 | 如果你繼續沿著現在的道路走下去 |
| 樞軸草圖 | 如果這個選項消失了 |
| 不受約束 | 如果金錢和形象不重要 |

## 六、心理測驗工具

心理測驗可以作為自我探索的起點，但不应成為唯一依據。以下是幾種在人生設計中常用的測驗工具：

### 6.1 [[K-SELF_MBTI性格測驗]] (Myers-Briggs Type Indicator)

**用途**：了解你的性格類型與偏好
**維度**：
- 外向(E) / 內向(I)
- 感覺(S) / 直覺(N)
- 思考(T) / 情感(F)
- 判斷(J) / 知覺(P)

**適用情境**：
- 職業方向探索
- 團隊合作理解
- 溝通風格認識

**限制**：
- 過度簡化複雜人格
- 測驗結果可能受當下狀態影響
- 不應作為職業選擇的唯一依据

### 6.2 [[K-SELF_大五人格測驗]]測驗 (Big Five Inventory)

**用途**：測量五大人格維度
**維度**：
- 開放性 (Openness) - 創造力、好奇心
- 尽责性 (Conscientiousness) - 組織、責任感
- 外向性 (Extraversion) - 社交、活力
- 親和性 (Agreeableness) - 合作、信任
- 神經質 (Neuroticism) - 情緒穩定度

**適用情境**：
- 深層自我了解
- 職業適配性評估
- 團隊組合成員分析

**優勢**：比 [[K-SELF_MBTI性格測驗]] 更具研究基礎，預測力更強

### 6.3 [[K-SELF_蓋洛普優勢測驗]]優勢測驗 (Gallup CliftonStrengths)

**用途**：識別你的天賦與優勢
**內容**：34 種才幹主題，分為四個領域：
- 執行力 (Executing)
- 影響力 (Influencing)
- 關係建立 (Relationship Building)
- 策略思維 (Strategic Thinking)

**適用情境**：
- 職業發展方向
- 團隊角色定位
- 個人品牌建立

**限制**：付費測驗，且結果可能隨時間變化

### 6.4 [[K-SELF_價值觀探索]]測驗

**用途**：釐清核心[[K-SELF_價值觀探索]]優先順序

**免費資源**：
- [Values Card Sort](https://www.transhistorian.com/values-card-sort/)
- [Personal Values Assessment](https://www.personalvaluesassessment.com/)

**練習：從 56 張[[K-SELF_價值觀探索]]卡片中選擇最重要的 10 個，再縮減到 5 個**

## 八、自我評估框架

### 8.1 現況分析問題

回答這些問題，幫助你全面了解自己：

1. **我是誰？** 描述你的身份、角色、關係
2. **我在哪？** 描述你當前的生活狀態
3. **我去過哪？** 回顧你的生命歷程
4. **我想去哪？** 探索你的願景
5. **我有什麼？** 盤點你的資源

### 8.2 定期檢視

建議每季或每月快速檢查：
- [[L-009-生命儀表板]]四維度
- [[L-010-時光日誌]]（感到迷惘時）
- [[L-008-人生指南針]]（重大決定前）

## 九、自我身份練習

這個練習幫助你深入了解「我是誰」。

**練習步驟：**
1. 準備十張便利貼
2. 每張頂端寫下「我是誰？」
3. 每張寫下一個不同身份
4. 檢視每個答案並擴充原因
5. 依重要性進行排序
6. 找出角色間的共通性
7. 釐清興趣、專長、能力與個性的交集點

## 十、商業模式與人生設計的區別

| 面向 | 人生設計 | 商業模式 |
|------|----------|----------|
| **核心問題** | 「我是誰？」 | 「我如何創造收入？」 |
| **焦點** | 生活方式與[[K-SELF_價值觀探索]] | 市場與價值交換 |
| **工具** | 奧德賽計畫、儀表板 | 商業模式畫布九宮格 |
| **目標** | 個人滿足感與意義 | 營收與成長 |

### 10.1 個人商業模式九大要素（參考用，非核心內容）

| 要素 | 說明 |
|------|------|
| 關鍵資源 | 有形與無形資產（才能、知識、人脈） |
| 關鍵活動 | 主要執行的任務 |
| 目標客層 | 定義特定客戶群體 |
| 價值主張 | 解決問題、滿足需求 |
| 通路 | 傳遞價值的途徑 |
| 顧客關係 | 建立長期連結 |
| 關鍵合作夥伴 | 提供動力、建議、成長機會 |
| 收益流 | 金錢報酬與軟收益 |
| 成本結構 | 有形成本與軟成本 |

> **注意**：上述商業模式分析工具僅供參考，不屬於「純淨版人生設計」的核心內容。如需商業分析，請參閱獨立的「商業模式」MOC。
