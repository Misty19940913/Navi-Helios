# HUMAN 3.0 MOC (Master Object Cluster)

> 本 MOC 專注於 **HUMAN 3.0 多維賦能框架**知識系統，包括四象限發展、等級進化與階段動態。可與人生設計、商業模式畫布系統互補使用。

---

## 核心定義

**HUMAN 3.0**：一個全面的個人發展框架，旨在實現「多維賦能」——最大限度地發揮所有生活領域的潛力。

**核心理念**：
- 拒絕單一領域專家化
- 追求心智、身體、精神與職業四大象限的同步極大化
- 問題是通往更高複雜度與潛能的必經之路

## HUMAN 3.0 核心架構

### 四象限（Four Quadrants）

```
         個人                    集體
      ┌──────────┬──────────┐
  心智│  內在    │   意義   │精神
      │  現實    │   連結   │
      ├──────────┼──────────┤
  身體│  外在    │   價值   │職業
      │  行為    │   創造   │
      └──────────┴──────────┘
```

### 等級進化（Levels）

```
1.0 順從者 → 2.0 個人主義者 → 3.0 綜合者
  ↓           ↓              ↓
外部權威    內部權威      多視角整合
基於規則    追求成就      策略性選擇
NPC         主角          設計師
```

### 階段動態（Stages）

```
x.1 失調 → x.2 不確定 → x.3 發現
  ↓          ↓          ↓
舊模式失效  進入未知   新模式穩定
痛苦迷失    高風險    清晰巔峰
```
