## 二、評估工具

### 2.1 HUMAN 3.0 評估 Prompt

> Dan Koe 创建的 AI Prompt：帮助规划自己的发展轨迹，找到元类型，并确定人生中下一个需要解决的核心問題。

**使用方式**：
1. 將 prompt 貼入 AI（如 Claude）
2. 進行自適應訪談
3. 識別四個象限中的當前發展
4. 找到元類型和生活方式原型

### 2.2 複合形態計算

**量化公式**：
- 四大象限的發展等級進行加權平均（1-3分制）
- 每個象限內有3個原型，每個原型有對應分數
- 計算平均值以定位整體演化位階
- 識別發展短板

### 2.3 生活方式原型識別

| 原型 | 主約束因素 | 轉化路徑 |
|------|------------|----------|
| 工作狂 | 時間 | 設定界線、審視價值觀 |
| 探索者 | 意義 | 將精神追求落實到現實 |
| 優化者 | 完美主義 | 接受「足夠好」 |
| 運動員 | 身體 | 擴展到其他象限 |
| 漂泊者 | 動力 | 找到核心使命 |
| 專家 | 整合 | 跨領域學習 |

## 三、實踐工具

### 3.1 階段檢測

**如何判斷自己處於哪個階段：**

| 階段 | 信號 |
|------|------|
| 失調 | 持續無聊、對現狀不滿、想要改變但不知道方向 |
| 不確定 | 踏入未知、身份認同瓦解、既興奮又恐懼 |
| 發現 | 新模式穩定、清晰度提升、能力鞏固 |

### 3.2 通道識別

**通道的跡象：**
- 談論某個主題時異常興奮
- 時間過得異常快
- 無法停止研究/創作
- 睡前最後一個想法和醒來第一個想法都是這個主題

### 3.3 等級評估

**自我檢視問題：**

| 等級 | 心智問題 | 身體問題 | 精神問題 | 職業問題 |
|------|----------|----------|----------|----------|
| 1.0 | 我相信只有「一條正確的道路」？ | 我是否忽視健康？ | 我是否依賴傳統權威？ | 我是否只追求穩定？ |
| 2.0 | 我是否相信自己的方式是正確的？ | 我是否追求表現？ | 我是否在尋找個人意義？ | 我是否在建立個人品牌？ |
| 3.0 | 我是否能接納多種視角？ | 我是否真正整合身心？ | 我是否欣賞多元真理？ | 我是否找到使命？ |
