# 商業模式畫布 MOC (Master Object Cluster)

> 本 MOC 專注於**商業模式畫布（Business Model Canvas）**知識系統，包括理論框架、實作方法與應用場景。可與人生設計系統互補使用。

---

## 核心定義

**商業模式畫布 (Business Model Canvas, BMC)**：一個視覺化工具，用於描述、分析與設計商業模式。由 Alexander Osterwalder 提出，包含九個相互關聯的構建塊。

**與人生設計的區別**：
- 商業模式 → 回答「我如何創造並傳遞價值？」
- 人生設計 → 回答「我是誰？」、「我想過什麼樣的生活？」

## 核心工具索引

### 設計工具
- 價值主張畫布 - 對應客戶痛點與收益
- 現狀/未來畫布 - 對比分析工具
- 個人商業畫布 - 一人企業應用

### 變現工具
- 變現階梯 - 從服務到產品的路徑
- 產品矩陣 - 免費→入門→進階→高端
- 意識層級 - 行銷策略調整

## 與其他系統的整合

- **人生設計** → 互補：人生設計回答「我是誰」，商業模式回答「如何創造價值」
- **職涯規劃** → 結合：個人商業畫布應用於職涯發展
- **財務規劃** → 整合：收入來源與成本結構對應財務規劃
