# 025_渠道與客戶關係

> 本章節回答商業模式中兩個關鍵區塊：**渠道通路**（如何接觸客戶）與**客戶關係**（如何與客戶建立並維持關係）。
>
> 這兩個區塊常被忽略，但它們決定了你的價值能否有效傳遞給目標客戶。

---

## 一、渠道通路 (Channels)

如何接觸並服務你的客戶？

Wiki 知識：[[K-BIZ-026_渠道選擇原則]]

### 1.1 渠道類型

|| 類型 | 說明 | 範例 |
|------|------|------|
| **自有渠道** | 官網、社群媒體、電子報 | 免費，長期累積 |
| **合作渠道** | 聯盟行銷、夥伴推薦 | 槓桿別人的受眾 |
| **付費渠道** | 廣告、代言人 | 快速但昂貴 |

### 1.2 一人企業的渠道策略

一人企業時間有限，無法像大公司同時經營所有渠道。策略是：**聚焦、少數、深度。**

- **核心平台（控制中心）**：電子報 — 擁有自主權，不被演算法支配
- **實驗室平台**：Twitter/X — 快速測試內容方向，即時獲得市場信號
- **擴展平台（深度視覺）**：YouTube — 適合深度內容，建立長期信任
- **權威平台**：LinkedIn — 適合 B2B 專業人士與企業決策者

### 1.3 渠道選擇原則

Wiki 知識：[[K-BIZ-026_渠道選擇原則]]

|| 原則 | 說明 |
|------|------|
| **你的受眾在哪裡** | 先確認目標客戶的聚集平台，不要自己想當然 |
| **你能長期生產內容** | 選一個能持續輸出內容的平台 |
| **這個渠道能否變現** | 能直接連結到收入來源嗎 |
| **是否有時間槓桿** | 內容能否一次創作、長期分發 |

> 詳見：[[K-BIZ-026_渠道選擇原則]] — 四原則×常見平台評估表

## 二、客戶關係 (Customer Relationships)

如何與客戶建立、維持關係？

Wiki 知識：[[K-BIZ-004_客戶關係類型]]

### 2.1 關係類型

|| 類型 | 說明 | 適用場景 |
|------|------|------|----------|
| **個人化** | 一對一互動 | 高端教練、顧問 |
| **自動化** | AI/機器人服務 | 大量用戶、標準化問題 |
| **社群** | 建立粉絲/會員社群 | 持續學習、忠誠用戶 |
| **共同創作** | 邀請客戶參與產品開發 | 迭代優化、忠誠度提升 |

### 2.2 一人企業的客戶關係策略

|| 關係類型 | 核心動作 | 適合階段 |
|------|------|------|----------|
| **陌生人** | 內容觸及 | 初期冷啟動 |
| **淺層訪客** | 電子報訂閱 | 獲取名單 |
| **註冊用戶** | 自動化 email 序列 | 培育轉化 |
| **付費客戶** | 個人化服務 | 深度關係 |
| **忠誠客戶** | 共同創作、社群 | 品牌大使 |

### 2.3 客戶關係生命週期

Wiki 知識：[[K-BIZ-024_客戶關係生命週期]]

```
認識 → 興趣 → 考慮 → 購買 → 使用 → 推薦
```

|| 階段 | 客戶狀態 | 你的任務 |
|------|------|------|------|
| **認識** | 第一次聽說你 | 內容曝光 |
| **興趣** | 主動關注 | 價值展示 |
| **考慮** | 比較選項 | 信任建立 |
| **購買** | 決定掏錢 | 轉化完成 |
| **使用** | 體驗產品/服務 | 交付價值 |
| **推薦** | 主動推薦他人 | 口碑擴散 |

> 詳見：[[K-BIZ-024_客戶關係生命週期]] — 六階段×關係類型對照表

## 三、渠道與關係的協同

Wiki 知識：[[K-BIZ-024_客戶關係生命週期]] · [[K-BIZ-026_渠道選擇原則]]

渠道決定了客戶如何認識你，客戶關係決定了認識之後會發生什麼。兩者必須匹配。

|| 渠道 | 對應客戶關係類型 |
|------|------|------|
| 電子報 | 自動化 + 社群 |
| YouTube | 社群 + 共同創作 |
| 一對一教練 | 個人化 |
| 課程平台 | 自動化 |
| 付費廣告 | 自動化（初期）|

> 詳見：[[K-BIZ-024]] — 六階段與關係類型的完整對照；[[K-BIZ-026]] — 四原則×常見平台評估
>
> **核心原則**：渠道引流量，關係留住客戶。兩者脫節，流量來了也留不住。

---

## 四、本章小結

- 渠道選擇不在多，在於「你能在那裡長期輸出內容」
- 電子報是一人企業最重要的自有渠道
- 客戶關係是從陌生人到推薦者的完整生命週期
- 渠道與關係類型必須相互匹配
