# DCA Pyramid Forex Strategy — 研究發現

## 策略設計背景

Boss 的原始策略來自 ETF 長期持有：價格跌 5% 加碼，漲 5% 獲利了結。
在外匯環境中，這個概念被改編為：
- 進場：KD 金叉 + 布林下軌
- 加倉：等差數列（0.01 → 0.10 lots），每次下跌固定點數
- 了結：每層獨立設定目標價，達到即了結

## 關鍵發現

### 1. Spread 成本是生死關鍵
```
1 pip spread：勝率 100%，PF ∞
35 pips spread：勝率 22.4%，PF 0.48，回報變負

每次進場成本：
  0.01 lot × 35 pips = $0.35
  10 層平均成本 = $1.75~$3.50
  1.0 ATR 了結利潤可能被完全吃掉
```

### 2. 獨立層次的問題
```
問題：每層有獨立的進場價和目標價
  Layer 1: 進場 1.87478 → 目標 1.87731
  Layer 10: 進場 1.84294 → 目標 1.84507

兩層目標價差距很大（約 250 pips）
後進場的 Layer 10 需要的波動比 Layer 1 更大

解決方案：
  A. 維持現狀（外匯環境正常表現）
  B. 縮小進場間距（50→30 pips）
  C. 提高 ATR 倍數（讓所有層目標更合理）
```

### 3. 歷史最大浮虧（帳戶合計）
```
$86.02
發生在：GBPCAD
時間：2025-10-29 18:00:00
場上層次：6 層（0.41 lots）
佔本金：0.86%
```

### 4. 最佳 ATR 倍數
```
1.0 ATR 在 1 pip spread 環境下表現良好
35 pips spread 環境下，需要更大的 ATR 倍數
估算：35 pips spread → 至少需要 2.0~3.0 ATR 才能盈利
```

---

## 待測試方向

| 方向 | 預期效果 |
|------|----------|
| ATR × 2.0 + 35 pip spread | 覆蓋更多層的 spread 成本 |
| ATR × 3.0 + 35 pip spread | 更高的單筆利潤，可能補償 spread |
| 縮小間距（30 pips） | 更多進場機會，但更多 spread 成本 |
| 加入空頭 | 增加交易機會，但需重新計算參數 |
| 排除 USDCAD、GBPCAD | 移除表現較差標的 |

---

## 結論

1. **Spread 是關鍵變數**：35 pips spread 讓原本盈利的策略變成虧損
2. **ATR 倍數必須提高**：才能在現實 spread 環境下盈利
3. **獨立層次不是錯誤**：是外匯環境下的正常表現
4. **歷史最大浮虧 $86.02**：在 27 個月的回測中可接受
