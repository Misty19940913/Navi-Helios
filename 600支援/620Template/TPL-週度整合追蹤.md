# 週度整合追蹤儀表板 (TPL)
> **用途**：每週規劃時使用，解決拖延與優先級混亂問題。它不僅追蹤「任務」，更追蹤您的「狀態」與「平衡」。
> **理念**：整合 [[HUMAN3.0整合知識庫]] 的四象限與狀態覺察，並結合 [[000_人生設計總覽]] 的目標導向，將時間管理升級為自我覺察。

### 1. 本週核心校準 (Weekly Core Alignment)
> **目的**：在開始行動前，先與您的大方向和內在狀態對齊。

- **本週主要目標 (DYL)**：`[[連結到您的季度或月度目標筆記]]`
- **本週狀態診斷 (H3.0)**：
    - **能量水平評估**：(低 / 中 / 高)
    - **焦點象限**：本週我感覺哪個象限最需要我的關注？ (心智 / 身體 / 精神 / 職業)

### 2. 承諾清單 (Weekly Commitment List)
> **目的**：將矩陣中的任務細化，並加入 H3.0 的診斷維度，這是解決「拖延症」的關鍵。
![[承諾清單.components]]
---
### 使用說明
1. **週初填寫**：花 15-20 分鐘填寫「核心校準」和「承諾清單」。
2. **分配象限**：為每項承諾標註 H3.0 象限，檢查是否平衡。
3. **定位矩陣**：為任務評估重要與緊急程度，放入 Q1-Q4。優先保障 Q2 的時間。
4. **每日檢視**：每天早上花 5 分鐘決定當日的 1-3 個 MITs。
5. **拖延時紀錄**：當發現自己拖延時，**不要自責**，立刻到「拖延反思」欄，用 H3.0 的語言做個簡單的紀錄 (e.g., "能量不足"、"對目標感到 Dissonance"、"害怕失敗"、"這件事沒有意義")。
