把 [[ACE Folder Framework]] 中的 `x` 資料夾想像成你筆記的工具箱。

這些筆記用來*支援*你的其他筆記。
- `Images` 會自動添加到「Images」資料夾（以及你帶入 vault 的任何其他附件）。
- `Templates` 也會被儲存在「Templates」資料夾中。

你可能也想把這些特殊檔案放在這裡，例如：
- `Canvas` 檔案（這些是 .json 檔案）
- `Excalidraw` 檔案（值得注意的是，它們實際上仍然是標準的 markdown .md 檔案）
