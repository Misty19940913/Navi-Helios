# 提示詞：加入表情符號 (Emojify)

為內容 `{}` 添加相關的表情符號以增強視覺感，規則如下：
1. 在文字的自然斷句處插入表情符號。
2. **禁止**連續放置兩個表情符號。
3. 保持所有原始文字內容不變。
4. 選擇與語境、語氣匹配的表情符號。

**僅回傳處理後的文字。**
