# 提示詞：改寫為推文串 (Twitter Thread)

依據以下規則將內容 `{}` 轉換為 Twitter 推文串：
1. 每則推文必須在 240 個字以內。
2. 第一行必須獨立顯示 `THREAD START`。
3. 使用 `---` 分隔每則推文。
4. 最後一行必須獨立顯示 `THREAD END`。
5. 確保內容引人入勝且清晰易懂。

**僅回傳格式化後的推文串。**
