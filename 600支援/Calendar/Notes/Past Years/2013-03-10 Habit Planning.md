# 習慣規劃

## 習慣線索
1. 地點
2. 時間
3. 情緒狀態
4. 其他人
5. 立即先前的行動

## 早晨
### 迷你運動
*線索*：不用思考，起床後直接去浴室。
*常規*：本體感覺平衡：兩邊各30秒
- 關節熱身：腳踝、膝蓋、臀部、手肘、肩膀、脖子 [替代：走5-10分鐘]
- 25個鳥狗式/防火栓式 / 25個收縮式俯臥撐 / 25個腹部：平板支撐蜘蛛式
- 25個體重深蹲/弓步 / 頭倒立1分鐘
- 想像美好的一天，保持良好站姿
*獎勵*：清晨的成就感、自控力和內啡肽。

### 早餐
*線索*：完成良好站姿的視覺化
*常規*：準備希臘優格、杏仁、肉桂 / 服用維生素、興奮劑
*獎勵*：飢餓的滿足感。健康而聰明的開始。

### 提問：日期定向、目标和日记
*線索*：把食物帶到房間。坐下前全身伸展 = 提問
*常規*：在Evernote中打開新筆記 / 開始提問並回答。其中一些大聲說出來。
*獎勵*：清晰的頭腦、目標感和優先順序與能量的協調。

## 晚間
### 商務：與團隊回顧和計劃
*線索*：接近下午5點或主要工作日結束。
*常規*：回顧我今天完成了什麼。我沒有完成什麼。下一步是什麼。 / 與團隊聯繫，回顧這一天。 / 計劃第二天。
*獎勵*：清晰的頭腦、目標感和優先順序與能量的協調。

### 運動
*線索*：完成工作日的情感感受。穿上運動服。時間大約是下午5點。有時與其他人一起。
*常規*：[[Lucia Stance]] 2-5分鐘
*獎勵*：自控感、信心、內啡肽，以及充滿活力和健康的身體；以及為當天剩餘時間恢復的能量。

## 夜間
### Lucia姿勢
*線索*：完成一天的情感感受。一個人。夜裡。
*常規*：Lucia姿勢 2-5分鐘
*獎勵*：控制感和意志力。

### 回顧和計劃
*線索*：完成Lucia姿勢後。
*常規*：回顧：獲勝時刻 / 日記、沉思 / 規劃第二天 / 提問
*獎勵*：成就感、成長、鞏固記憶的感覺。

### 伸展和瑜伽伸展
*線索*：完成第二天的規劃後。站起來。
*常規*：快速全身 / 肩倒立、搖籃式、輪式、坐姿伸展、頭倒立
*獎勵*：飢餓的滿足感。健康而聰明的開始。

### PMR和輕度視覺化
*線索*：完成頭倒立後。躺下進入攤屍式。
*常規*：PMR身體5分鐘 / 輕度視覺化第二天 / 記錄任何浮現或揮之不去的想法。
*獎勵*：準備和信心的感覺。
