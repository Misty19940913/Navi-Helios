## History of LYT Mode

這個 [Obsidian](https://obsidian.md/) 主題是我第一個所愛：[[Cybertron Theme]] 主題的確定性演進。

[@nickmilo](https://github.com/nickmilo) 和 [@ceciliamay](https://github.com/ceciliamay) 在一個 LYT 工作坊中相遇。同時，Cecilia 正在設計 Obsidian 2021 年度主題大賽的冠軍作品，現在已成傳奇主題 [Primary](https://github.com/ceciliamay/obsidianmd-theme-primary)。在冠軍揭曉之前，Nick 向公眾尋求幫助，希望將 Cybertron 提升到下一個層次。這需要比他自己所能集結的更多 CSS 和設計技巧。幸運的是，Cecilia 舉手了。然後 Nick 委託 Cecilia May 設計 LYT Mode，這就是成果。


## Features

1. 與 Cybertron 相比較進化後的配色方案
    - Markdown 強調的特殊顏色
2. 專為彼此設計的美觀易讀字體：**DM Sans DM Mono**
3. 同時支援**手機和桌面**版本
4. 支援所有 Obsidian 視圖：傳統模式、即時預覽和閱讀模式
5. **自訂圖標**
6. 最小化動畫以實現更專注的工作
7. LYT Mode 語法高亮
8. **更緊湊的側邊欄行高**間距，讓您能夠看到更多筆記、標籤和連結
9. 灰顯的 Markdown 格式（標題除外）
10. 更大的彈出視窗
11. Callouts 的設計支援
12. 支援**簡報**模式
13. 支援**發布**功能

## Supported Plugins

LYT Mode 專注於強調對 markdown 或筆記原生功能以及 Obsidian 所有核心功能和視覺提示的支援。也就是說，LYT Mode 應該可以與大多數插件配合使用，但此列表顯示了受到特別關愛或設計支援的插件。

- Calendar by Liam Cain
- Sliding Panes (Andy's Mode) by death_au
- Kanban by mgmeyers
- Excalidraw by Zsolt Viczian
- Dataview by Michael Brenan
- Hover Editor by nothingislost

## Additional Details

### Alternative Checkboxes aka Icon Bullets

LYT Mode 還為核取方塊提供了額外的語法。這些就像子彈日誌的「圖例」一樣運作。[其他使用替代核取方塊作為快速視覺參考。](https://www.youtube.com/watch?v=8IL0wFUT6XQ)

| Syntax  | Description | 
| ------- | ----------- |
| `- [ ]` | Unchecked   |
| `- [x]` | Checked     |
| `- [>]` | Rescheduled |
| `- [<]` | Scheduled   |
| `- [!]` | Important   |
| `- [-]` | Cancelled   |
| `- [/]` | In Progress |
| `- [?]` | Question    |
| `- [*]` | Star        |
| `- [n]` | Note        |
| `- [l]` | Location    |
| `- [i]` | Information |
| `- [I]` | Idea        |
| `- [S]` | Amount      |
| `- [p]` | Pro         |
| `- [c]` | Con         |
| `- [b]` | Bookmark    |
| `- [f]` | Fire        |
| `- [k]` | Key         |
| `- [w]` | Win         |
| `- [u]` | Up          |
| `- [d]` | Down        |
