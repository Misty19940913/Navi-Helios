> [!Training]- # 努力
> > *你可以做什麼？*
> 
> 若想獲得集中的視圖，請前往 [[Efforts]]。
> 
> 使用這個來保持優先順序並根據需要快速調整你的帶寬。
> 
> > [!Box]+ ### 🔥 進行中
> > ``` dataview
> > TABLE WITHOUT ID
> > file.link as "",
> > rank as "Rank"
> > FROM "Efforts/On"
> > SORT rank desc
> > ```
> 
> > [!Box]+ ### ♻️ 持續中
> > ``` dataview
> > TABLE WITHOUT ID
> > file.link as "",
> > rank as "Rank"
> > FROM "Efforts/Ongoing"
> > SORT rank desc
> > ```
> 
> > [!Box]- ### 〰️ 沉澱中
> > 努力可以很容易地從「進行中」轉移到背景中的「沉澱中」。
> > 
> > ``` dataview
> > TABLE WITHOUT ID
> > file.link as "",
> > rank as "Rank"
> > FROM "Efforts/Simmering"
> > SORT rank desc
> > ```
