## Phase 1：Ideation（第1-2週）

### 第一步：找出你的利基（Niche）

**利基 = 人群 + 問題的交集**

> 「你只透過為願意付錢解決問題的人解決問題來賺錢。」

**三問題過濾法（來自 Taki Moore）**：
1. 我喜歡他們嗎？（Do I like them?）
2. 我能幫助他們嗎？（Can I actually help them?）
3. 他們願意付 \$2,000+ 嗎？（Will they pay?）

**建議：賣昂貴的東西（$2,000+）**
- 便宜東西也能致富，只是更慢
- \$3,000 產品：每月 3 個客戶 = $100K/年
- \$2,000 月費 retainer：4-5 個客戶 = $100K/年

**技能清單（Craft Skills）**：
- 你現在被付錢做什麼？（你的 CV）
- 你解決過什麼個人問題？（減肥、健身、學習...）
- 從既有的技能出發比從零開始快很多

**流程：發散 → 收斂 → 浮現（Diverge → Converge → Emerge）**
- 先列出 15-20 個潛在利基
- 收斂成金、銀、銅三個最佳選項
- 在過程中正確答案會自然浮現

---

### 第二步：創建第一版報價（Offer）

**報價 ≠ 產品**

| | 定義 |
|--|------|
| **產品** | 實際的東西（書的內容、課程影片）|
| **報價** | 產品的包裝（標題、副標、推薦語、包裝設計）|

> 「顧客買的是報價，收到的是產品。」

**多數新創業者犯的錯**：先花 3 年建產品，才發現沒人要。

**正確做法**：先有報價（1天就能寫出標題+副標+電梯簡報），拿去向目標客戶展示，看有沒有人說「太棒了！我要報名」並願意付錢，再開始建造。

## Phase 2：Validation（第1-3個月）

**目標：獲得第一筆銷售**

> 「在客戶真的拿錢給你之前，所有的利基和報價都是理論。」

### 三種發現電話框架

**框架一：市場研究（無壓力）**
```
「我在構思一個幫助[X人群]解決[Y問題]的業務，
可以聊聊15分鐘做市場調查嗎？純研究，不賣東西。」
```
→ 適合：全新利基、無信任基礎、菜鳥階段

**框架二：免費教練課**
```
「我在這個領域有幾步領先，來電聊聊看能不能幫到你，
作為交換給我一點回饋就好。」
```
→ 適合：已有部分專業技能、教練輔導型服務

**框架三：銷售電話**
```
「這是一個申請面試，我想評估你是否適合我們的方案。」
```
→ 適合：對報價有信心、教練/課程/社群產品

### 驗證目標：10-30 通發現電話

- 每次通話後根據回饋調整報價
- **「沒有報價能在市場第一線接觸後存活」**（No offer survives first contact with the market）
- 理論Craft再好，不接觸真實用戶等於零

### 三種獲取通話的方式

1. **現有網路**：發訊息給朋友、舊同學、同事（最快速）
2. **內容創作**：在 LinkedIn/Instagram 發佈解決特定問題的內容，精準族群會自然接觸你
3. **陌生開發**：
   - 線下社群活動（真實人脈）
   - LinkedIn Sales Navigator（連接請求 → 培養 → 成交）
   - 論壇/社群（，作者早期在 Student Room 論壇幫人解答醫學申請問題，三年獲得上千名客戶）

## 關鍵思維框架

### 發散 → 收斂 → 浮現
通用創造力原則，適用於任何領域

### 報價六P（Six Ps of Offer）
1. **Person** — 誰是核心目標客戶？（absolute bullseye）
2. **Problem** — 他們的痛苦是什麼？
3. **Promise** — 10字以內的承諾
4. **Process** — 你如何解決？（Program、Plot、Plan...）
5. **Price** — 價格
6. **Position** — 如何定位

## 重要提醒

- \$100K/年 = 每月 $8,333，並非天文數字
- 教練Abdallall 的個人經驗：第一次從零到 $100K 花了 2 年；現在有技能基礎的話，3 個月可達成
- **內容不是用來成名，是用來建立權威性（Authority）**
- 演算法現在很聰明，只要內容真的好，精準的人會被推送給你
- 最快的事業建立在已有技能上（而不是從零學新技能）
